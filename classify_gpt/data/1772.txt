【什麼是有小孩的美好？】
: 我認為大概有以下幾點：
: 1 .孩子愛你：小孩是愛你的，小孩生出來依賴你、跟你一起吃飯、跟你一起做很多事
，

如果沒有外援， 小孩24小時黏著你， 你沒有任何自己的時間。 連大便都要把小孩帶上
。 吃飯也不用想好好的吃。  晚上睡覺也要起來好幾次。 我已經忘記早上睡超過9點是
多少年前的事情了。 沒錯， 黏成這樣， 已經是恐怖情人了。


: 2.孩子陪你再過一次童年
: 聚餐吃火鍋，聊工作、獎金、買了什麼東西、八卦、車，
你很幸運， 可以跟朋友聚餐聊天

: 小孩吃火鍋，看著升起的蒸汽，
: 說哇，問你這是什麼？伸出手想捕捉，

然後， 手就往火鍋裡面碰了
接著就去廁所沖水

: 看到火鍋裡的丸子浮沉傻笑，一直要你看。

然後， 就拿著筷子湯匙亂打亂撈
全桌都髒
最好的方式就是綁在椅子上。 讓他遠離

: 以前出門的時候找的是好看的衣服，
: 家裡缺的東西，新開的餐廳，
: 大部分的時候都跟消費有關，
: 跟小孩出去，看路邊的瓢蟲可以看3分鐘。這3分鐘雖短，但你可能十幾年都沒這樣看過
大自然了，

抓毛毛蟲，蝸牛，蜘蛛，蟬，壁虎是日常
危險的蜜蜂和蜈蚣或四角蛇， 我阻止了
我寧可他少些大自然， 手會乾淨一點

: 3.孩子填補心中缺的那個空洞
: 工作上，同事談論挖洞給誰跳，
: 誰發了封email又跟誰槓起來，
: 然後談論部門與部門間互相推諉，
: 爭取薪水跟獎金買好車好房，
: 我不是多清高的人，
: 但有時看職場太多負能量耍手段，
: 坦白說也會煩。
: 孩子的來到，溫暖太社會化的心，
: 下班打開門，孩子看到你高興的表情，
: 大聲唱著學校教的歌。
明明哭的比笑的時候多，起床哭， 吃飯哭， 看電視哭， 洗澡哭。 任何事情都可以哭。
 完全沒有道理可以講

: 週末你準備看劇跟睡覺，
從小孩出來， 沒進過電影院

: 孩子說想去玩沙玩水，把你帶到自然裡，
一直往海裡衝， 光把他們拉回來
就沒心情玩了

4.孩子伴著你：
: 你忽然感到孤獨的時候，孩子陪你；
: 你感受悲傷的時候，孩子也陪你，你快樂的時候，孩子也陪你。
沒錯， 24小時黏著你。 想放開也不可能

: 老婆跟朋友出去的時候，
: 我最愛開車帶小朋友出去，一個男人一個小孩，到家好的咖啡店喝咖啡吃蛋糕，
然後去百貨公司逛玩具，
: 聽小孩介紹玩具，回家吹冷氣看卡通，
: 幫兒子洗澡，然後一起睡午覺，
又吃甜食， 又玩玩具， 還可以看卡通。 你小孩一定很愛你

我也想回到1打1的日子
一打2真的太累了

: 【結語】
現在生活只剩下工作和小孩
希望大一點，他們能獨立
把自由還給我
