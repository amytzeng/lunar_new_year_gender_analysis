: 而這個建立，其實不是在於「給予」，
: 而是在於「剝奪」跟「隔絕」。
: 很多窮人小孩的教育失敗，
: 就是在於給得太多。
: 當小孩有電視、有手機、有網路、有電動、有玩具，
: 他選擇很多的時候，
: 自然就會離讀書越來越遠。
: 而我們家族裡，小孩最會念書的家庭，
: 總共生了三個小孩，經濟不算寬裕，
: 但三個小孩都是在沒有補習的情況下，
: 考上了台灣的醫學系，兩個國立，一個私立。
: 我印象很深刻，
: 他們三個小孩都是在3歲左右，
: 就已經可以流暢的閱讀報紙了。
: (當時我大概12-13歲左右)
: 而可以有這樣的成果，
: 並不是靠花錢，也不是靠天分， (科學證據支持，天才是靠早期且正確的培養)
: 是靠他們讓小孩在上學之前，
: 除了書本，不會接觸到任何娛樂。
: 父母想要看電視、玩電腦、玩手機，
: 一律都是8點過後，等小孩上床睡覺，才會開始使用，
: 他們在小孩面前，唯一的娛樂就只會看書而已。
: 所以他們的小孩，從小就把書本當玩具，
: 書本成為他們唯一的娛樂，
: 從父母身上最容易模仿的行為，
: 就算假日出去玩，
: 行程也80%以上都是書店和圖書館。
: 小孩的大腦，大概在5歲前會定型，
: 所以當你能持續5年，只給小孩接觸書本的生活，
: 就已經足夠讓小孩建立良好的閱讀習慣了。
: 後面上小學，雖然會開始被各種娛樂汙染，
: 但他們對於閱讀的習慣與能力，
: 依然會勝過同齡的人非常多。
: 而且父母持續5年的閱讀習慣，
: 多少會強化到自己，
: 對於教育小孩也是一個優勢。
: 別說什麼工作很忙很累，
: 把你看電視和滑手機的時間都拿出來，
: 多數窮人也是有辦法看書的。
: 更別說大多數的人，也就是在座的各位，
: 真的是窮人，有低收入戶資格的，應該是少數吧?
: 你們根本不算窮好嗎?
: 至於你說什麼自學的訣竅，
: 拜託!「學習怎麼學習的書」，根本就是一個分類了，
: 對於有出色閱讀習慣和能力的小孩，
: 根本不是問題好嗎?
: 那上面這些行為，需要有錢才能做到嗎?
: 每個月1000塊，甚至500塊就綽綽有餘了，
: 對只能接觸書本的小孩來說，
: 1、2本書就夠他們「玩」很久了。
: 只是如我所言，知易行難，
: 就算我跟你講了，多數人也不願意去做，
: 只會希望用講起來最簡單，
: 但實際上卻不可行的方式解決，
: 也就是自己變得很有錢，什麼都可以砸錢解決。
: 現在雖然是多元入學，
: 但本質還是在考試，

考試制度已經改了，
108課綱大學科系，申請入學規定：
考試成績50%（最多只能佔50%）+
備審資料50%（最多只能佔50%）

所以基本上家裡開公司的，
申請商學院就非常有利了。

另外，
在備審資料中一定會計算學生的
「自主學習」成績，

一直在讀書讀書讀書的學生
和一位畫畫得獎、烘焙得獎、鋼琴檢定等的學生，
或參與各式物理、數學、生物、科學競賽的學生，
差異就很大了......

我只是想說，這年代的升學，
只靠考試成績，完全 不行。

: 如果你筆試成績都非常的好，
: 然後再從書本學習面試的技巧，
: 加上輔導室免費的實戰模擬，

備審資料就先輸了，
面試技巧哪裡有用......
: 在現在這個環境也不會多吃虧好嗎?
: 甚至講難聽點，
: 很多人以為自己小孩養不好，是輸在錢，
: 但你會沒錢，很多時候是自己的問題，
: 好逸惡勞、眼高手低，
: 只愛幻想簡單輕鬆但不實際的解決辦法。
: 你這個問題不改，又讓小孩耳濡目染的話，
: 就算直接給你很多錢，全部砸在補習和家教上，
: 小孩的成績也不會好多少。
: 反過來說，
: 就算讓那些有錢人變得跟你一樣窮，
: 他們教育出來的小孩，恐怕依然照樣贏過你。
: 因為教育這種事情，最重要的真的不是錢，
: 而是看你在小孩定型的那前5年，
: 能不能堅持下去，扮演一個好的父母讓小孩模仿。
: 有錢或許真的比較簡單，
: 但沒錢也是可以做到，
: 真的別把錢當成你偷懶的藉口。
