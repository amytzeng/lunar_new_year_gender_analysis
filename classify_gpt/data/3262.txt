: 最近幫小孩辦收涎，收涎那天提前接公婆來住處，等著和娘家家人會合一起去收涎的地方
: ，時間快到時接到媽媽打來電話說車子在住處前面路口和機車發生擦撞，結束通話和先生
: 說明狀況後，準備一起趕去現場，在旁的公婆卻像完全活在平行世界一樣，還跟我說孫子
: 怎樣可愛之類的話。到現場，幸好雙方都沒大礙，但是之後還要做筆錄，和店家約的時間
: 快到了，於是就載小孩和公婆先出發。一路上仍然電話聯絡狀況，和開車的先生討論，同
: 車的公婆依舊像是活在平行時空一樣，只顧著和孫子玩。後來到收涎儀式結束了，娘家家
: 人還是沒能趕到，過程中公婆依舊沒問半句話，想說這是小孩和大人的回憶，於是忍在心
: 裏沒發作。晚上睡前忍不住和先生說心裡的想法，他說公婆可能不知道我們在說誰，或是
: 沒注意聽。隔天和先生一起回娘家，出發前又和公婆說明了一次狀況，他們只說了車子撞
: 壞了之類的話就繼續和孫子玩。

  解讀事情可以分成客觀的事件描述，跟主觀的詮釋
  「沒有主動表達關心」是事件描述
  「活在平行時空」跟後面所說「不聞不問」是主觀詮釋


: 之後幾天不斷回想結婚來，懷孕到生產的過程，突然清醒了，發覺自己遲鈍的可怕。懷孕
: 期間大部分的時間公婆幾乎是不聞不問的，頂多是說有需要什麼跟我先生說，總是媽媽不
: 停的交代懷孕該注意的事，替我準備補身體的東西。後期因為有些狀況沒辦法繼續上班，

  「沒有跟你說該做什麼不該做什麼」是事件描述
  「不聞不問」是主觀詮釋

  如果婆婆在妳懷孕期間不斷告訴你：
  「要注意這個注意那個」、
  「這個不能吃那個要多吃」、
  「這件那件事情不能做。」
  「應該要怎樣你怎麼沒做」
  這樣比較好嗎?
  可能要請有被婆婆指導如何養胎的版友分享

  如果婆婆幫你補身，正好沒猜準心意，準備到你不愛的
  又或者觀念不同，準備到醫生說不可以吃的
  這時候會不會有拒絕的壓力?

: 請假在家安胎，公婆說他們不會坐車，公公身體也不好，要麻煩我請媽媽來照顧我。後來
: 婆婆又說她不懂坐月子的事，要麻煩媽媽幫我，原本就不想麻煩家人，已經找好月子中心
: ，當下並沒有太在意。後來確定剖腹日期，公婆突然密集的打電話來關心，當時心裡已經
: 有些疙瘩。剖腹手術需要全身麻醉，情況差可能要摘除子宮，幸運的是後來順利完成手術
: ，只是術後前幾天狀況不太好，晚上痛的睡不著，也許是太疼痛了，隔天他們突然坐車上
: 來，說想看孫子，因為管制的關係看不到，失望的說白跑一趟沒多久就回去了，當時也沒
: 太在意。之後從月子中心回家，開始新手媽媽的生活，婆婆說怕我太累，想要過來幫忙照
: 顧小孩，考慮到兩老年事已高，而且比較希望媽媽來照顧就婉拒了。之後一段時間，她談
: 話時不時會提到無法來顧孫子的事，先生也有些不開心，忍不住跟先生說他們曾說安胎時
: 無法來照顧的事，也許傳到婆婆那之後才沒在提。

  細節太多不想一一列舉

  只是您可能要想一想
  這些你抱怨婆婆沒做的事情
  婆婆如果做了，
  但不符合你心意(可能性很大)
  會不會比不做還差?


: 覺得難過，回顧起來媽媽為我和小孩做了很多事，她甚至主動說回去上班後可以幫忙照顧
: 小孩，因為諸多決定就小孩送托。而婆婆知道我想將小孩送托，第一個反應是不能請媽媽
: 幫忙照顧嗎？媽媽為我們做了很多事，最後卻得到這樣的回報，而我，如果不是先生總是
: 無微不至的照顧小孩和我，真有種錯覺，好像我仍然還沒結婚，總是接受著媽媽各種的付
: 出。

  期待婆婆跟媽媽一樣為自己付出
  是不是有點太理想化?

: 想到之後，真不知該怎麼面對公婆。先生看到他們和孫子玩這麽開心，也想讓他們常常看
(文常恕刪)


  整體而言我只想回應「怎麼面對?」
  我以前也走過跟你一樣想不開的路
  後來大悟：何苦為難自己！

  各種客觀事件，經過不同角度的主觀解讀
  就可以演成完全不一樣的劇本

  婆婆沒有像媽媽一樣主動為你做種種事
  你可以把它想成「婆婆不關心我、婆婆只愛孫、婆婆只愛兒子」
  也可以把它想成「婆婆可能是不想干涉我」、「婆婆可能不知道怎麼跟媳婦互動」

  她到底怎樣想，我們無法確定
  但妳怎樣想，可以左右妳自己的心情

  就算對方真的對不起妳
  (到底是有多對不起妳?????)
  一直想她的不是，也只是增加自己的痛苦罷了

  把事件描述客觀化
  把對對方動機的揣摩美化
  真正該為自己負責的是妳自己
  當我們成年之後，應該從被照顧者轉換成照顧者的角色
  就沒有這麼多抱怨了
