: 目前遇到下面2個問題:
: 1.金錢觀念:
: 因為太太家境較為富裕，所以養成消費無法節制的狀況，婚前已經幫她還過一筆約15W的
: 卡費，

        問題點1:

        絕對不要幫親人還卡債，包括二等親內。銀行最
        喜歡他們這種會使用循環利息，或開始繳滯納金
        後有親友會幫忙還債的客人。

        等他們的債務清償之後，銀行會主動幫忙提高額
        度，或者打電話來推銷「這裡有一筆利率好便宜
        的信貸，有沒有需要呢~」。

        建議的做法：

        不要幫她還任何債務，讓她信用破產，這樣就沒
        有銀行願意借款給她。

        除此之外，要注意她會不會偷變賣家裡值錢的東
        西，甚至去跟地下錢莊借錢，那問題就會變得非
        常複雜。務必要注意她平常是否有異常的開銷。

: 婚後到生小孩前，太太的收入都是自己使用，且鮮少拿出作為家庭使用(但她常常強調有
: 出過2~3W協助我家裡處理事情，因為是真的有從她那裏出錢，我也不想多說甚麼)
: 108年因為工作、收入變動的關係(太太已是全職媽媽每個月由26K變為24K作為吃飯費，其
: 中所有"家庭"開支均由我支出)，

        問題點2:

        就算你一肩扛下家裡所有開銷，也該關心家用/
        飯錢怎麼花的。給予太太不過問用途的零用金
        是一回事，家用飯錢是共同支出的一部份，應
        該要控管。

        人性就是這樣，當有一筆不會被過問明細的可
        支配所得時，就會盡量把它花完。例如午餐明
        明可以用100元解決，就一定要吃到180元，有
        便宜的當季水果可以吃，卻一定要買高檔的草
        莓櫻桃。

        24K的飯錢對於一大兩小來說根本超高，平均
        每個工作天花費一千元，尤其如果學校有供餐
        ，周末假日出遊也都是你在付錢的話。問題應
        該就是出在那些亂買的書籍玩具課程。

        建議的做法：

        直接給她你設定好的預算，並聲明這筆錢只能
        用來吃飯，想要買玩具或書本，就少吃幾餐，
        自己權衡。

        做這件事雖然會引起對方的不滿，但如果沒有
        嚴格控管預算，那你就只能想辦法增加收入，
        通常這個更困難。

        如果她有透支飯錢的習慣，表示她沒有資格管
        錢，那麼以後飯錢就改成每天給一次。不用覺
        得這樣做很絕情，沒有在賺錢養家的人，根本
        不能體會被一文錢逼死的那種感覺。

: 使用信貸借了一筆錢作為家庭應急使用費(目前仍在還款中)，之後109年3月後收入變穩定
: ，
: 到收入穩定後信貸約仍有5W在帳戶中，有跟妻子溝通這是應急用，不要任意使用，但去年
: 年中想要使用這筆錢的時候發現已被太太花光，而且完全沒先講。
: 之後為這件事吵架，要求她出去工作賺錢；妻子回復會節省，預計在年底存回至少一半的
: 費用。(但當然沒有存任何錢回去)
: 今年1月初的時候因為太太說急著要繳卡費，請我先給當月的生活費，我好奇心使然，偷

        問題點3:

        對於沒有自制力與金錢管理能力的人，千萬不可
        以共同持有帳戶，不然會像兩個相連的水塔一樣
        ，最後兩邊的水位會一樣高。對你太太而言，共
        同帳戶等於擁有一張有無限額度的黑卡。

        建議的做法：

        在家裡放置五千元甚至更少的現金作為預備金。
        如果動用預備金必須說明理由。

        你連家庭應急使用費都要動用到信貸，表示你的
        收入並沒有很高，信貸的利息加計各種費用之後
        ，實質利率大概要9%起跳，也就是你辛苦賺了一
        百元的薪水，卻只能花九十元。與其這樣，不如
        一開始就勒緊褲帶，賺多少錢花多少錢。何必借
        一筆信貸放著？為了讓戶頭看起來有現金比較有
        安全感嗎？

: 偷看一下她的信用卡帳單，一看又嚇一跳，一張有7W多的循環卡費，一張有4W多的循環卡
: 費
: 怎麼又開始欠債了，跟她又大吵，她說她最錯誤的決定就是做全職媽媽要跟我拿錢。
: 可我是覺得可以節儉一點過生活，但不要再有卡債了。

        問題點4:

        娶(嫁)一個會動用循環利息的伴侶。循環利息就
        像吸毒一樣，使用過之後就很容易上癮。這些人
        會宣稱只要支付少少利息就可以早買早享受，因
        為這些都是遲早要買的東西。其實根本是在透支
        自己未來的收入，同時擴大了購買慾。

        建議的做法：

        要求對方剪掉所有的信用卡，所有花費都以現金
        支付。但實務上這點幾乎做不到，用這個理由要
        求離婚也不太可能成立。

        所以你能做的就是確保夫妻財產分離，尤其居住
        的房子不可以掛她的名下，否則當卡債違約時，
        銀行第一個就會法拍你們的房子。而你為了不要
        讓房子被法拍，就會咬牙幫她清償貸款，這樣她
        就獲得重生，又可以重新借信貸。

        你岳父要富養女兒是他的選擇，你太太要嫁給你
        這個窮小子也是他的選擇，大家都是成年人，對
        自己的行為要負責。不要怕丟臉，直接跟你太太
        說，你無法也不願負擔她的卡費，請他自己找她
        爸爸幫忙。


: 2.家事/育兒
: 家事:因為太太幫小孩報名很多活動，所以家裡幾乎都沒在打掃，大概就是週末沒活動的
: 時候我用吸塵器打掃這樣；但我對家裡很亂常常會念，她說她不是傭人，照顧孩子已經很
: 累了，叫我不要要求這麼高。
: 可是我覺得把家裡整理得舒服，跟照顧孩子應該不衝突阿(而且老大已經去幼兒園了，
: 不是應該變輕鬆了嗎?)


        問題點5:

        全職照顧兩個幼童確實很累，家事不是她一個人
        的事情。比較愛乾淨的人要負責多做一點家事。
        家裡很乾淨不是人生或者家庭的義務。

        建議的做法：

        你如果上班也很累沒空幫忙太多家事，那就降低
        你的標準。再不然就把孩子送安親班或保母，要
        求太太去上班，然後你們再平分家事，或者外包
        給別人每週來打掃一次。

: 可是我覺得把家裡整理得舒服，跟照顧孩子應該不衝突阿(而且老大已經去幼兒園了，
: 不是應該變輕鬆了嗎?)

        不愛乾淨的人，就算整天無所事事，也不會想去
        打掃，寧願去追劇跟滑手機。如果你太太要你下
        班回家沒事就抄心經，每天還要抄滿10大張紙，
        你覺得有意義嗎？

: 育兒(兩個孩子一個4歲、一個5歲半):幫孩子買超多的書、玩具與報名才藝課程，孩子的
: 東西多到家裡面有一大堆還有沒開來玩過或看過的書或玩具，我希望可以少一點，但她說
: 她都有做功課，要我相信她這些都是孩子需要的東西(但我發現週末的時間大都在外，

        問題點6:

        孩子的成長是夫妻共同的責任。需要買什麼書籍
        、玩具，參加什麼課程，你都有權利共同決定。

        建議的做法：

        下班回家或者週末的時候，你也要完全參與孩子
        的生活，有些父母會覺得教養孩子要靠書本道具
        但其實親自陪伴才是最好的方式。

        為了宣示你的決心，看過的書可以上網拍賣，只
        給予固定的空間例如一個或兩個收納箱裝玩具，
        超過的就丟掉或賣掉。

        同時明確告知，教育費用都不能算在24K家用裡
        ，如果還是透支，以後飯錢就改成每天給。

: 參加課程或是比賽，真的還有時間看跟需要玩這麼多的東西嗎?)

        週六日的時候，你都怎麼陪伴孩子？親自帶他們
        出去，或者坐等太太安排孩子的行程？

: 如果管教孩子到很生氣的狀態，常常會大吼大叫(左右鄰居都會知道的音量，一周至少1~2
: 次)，而且會不當體罰(曾經發生過拿書丟孩子、推孩子去撞桌子，最誇張我看過孩子已經
: 躲在角落了她還踹跟丟塑膠椅子)，這個時候如果我去制止她，她會更暴怒打孩子。
: 不知道是不是應該去做婚姻諮詢，還是......,

        問題7:

        這個最嚴重。如果你太太已經對孩子家暴，你
        還不制止她，以後你小孩長大只會看不起你，
        並永遠擺脫不了母親的情緒勒索，因為你連保
        護他們的勇氣都沒有。

        建議的做法：

        陪著太太去看心理醫生，基本上她已經生病了
        ，有躁鬱症或者憂鬱症的問題，雖然無法根治
        ，但是藉由適當的情緒藥物，降低她的情緒敏
        感度之後，就可以大幅改善她的人際關係，以
        及對孩子的教育方式，不會因為小事就暴怒。

        並聲明如果再次家暴孩子，你會立刻打113舉發
        她，申請保護令。若還是不改善，你會收集證
        據然後提出離婚，她將不會有任何贍養費，還
        要反過來提供孩子的教育經費直到18歲。

        基本上你這些問題我差不多都遇到過，不限發
        生在伴侶身上。不解決這些問題，你的人生機
        本上是黑白的。


        (補充)

        另外，方法雖然告訴你了，但是實際上做起來
        都很困難，要是很簡單，也不會變成今天的局
        面了。

        所以一切的關鍵是你的「決心」，你願意花多
        少代價來解決這些問題。我講的問題不是你太
        太的卡債，而是1.你的家庭經濟問題。2.家庭
        暴力問題。3.孩子寧願選擇媽媽。

        一個上班族的收入，通常難以短時間暴增，即
        使有明顯增加，通常也是犧牲了更多的時間才
        換來，不要工作越換越差就很厲害了。炒股票
        也絕非正道，別看股板好像人人都翻轉了人生
        ，更多的是小賺大賠的韭菜們。

        在無法開源的情況下，節流就是唯一的手段，
        由於原文沒有多描述所以也不善加猜測，但你
        必須把家庭所有的開支包括保險費都列出來，
        根據緩急輕重排出順序。信貸務必優先清償，
        而且照這樣看起來，你自己的信用卡也該先剪
        掉才對。自己做不到的事情，又怎麼能要求別
        人做？

        其次，家暴問題務必解決，但是弄到叫警察來
        把你太太抓走，我猜你也不忍，不然其實你一
        開始就直接揍她就好了。所以還是要從根本的
        問題著手--她的心生病了。

        如果她只是有時候會暴怒，平常還能正常地與
        人互動(不然你的第二胎怎麼生出來的)，那幾
        乎可以斷定是她的情緒有問題，可能是先天的
        ，也可能是因為照顧兩寶累積的壓力造成。

        無論是哪一種，在她沒有發作的時候，溫柔的
        跟他表情你很心疼她這樣生氣，不只傷了孩子
        也傷了她自己。你願意陪她去看身心科(不用
        去看什麼婚姻諮商了，直接對症下藥才有用)
        。現在的人壓力大，身心問題已經是文明病，
        沒有什麼好可恥的。

        安撫情緒的藥物很多種，但是副作用因人而異
        ，即使你太太願意接受建議服藥，剛開始的時
        候難免會因為藥物的副作用而感到強烈不適，
        因而想自行停藥。建議有藥物不適的情況時，
        就立刻回診與醫生討論，試著更換藥物。另外
        ，為了減少副作用的不適，可以調整到睡前的
        時候服用。在治療的過程中，你的支持會是最
        大的關鍵。

        最後，孩子即使被虐待，但因為自知沒有獨立
        謀生能力，所以還是會選擇依賴照顧者。如果
        你小孩被家暴，卻還寧願選擇媽媽，表示你在
        孩子的心目中，根本沒有地位，也就是沒有用
        的爸爸。

        因為原文沒有說明，不知道平常晚上以及周末
        假日，你陪伴孩子的時間有多少。但你應該好
        好思考，為什麼孩子寧願被媽媽毒打，也不想
        選爸爸？這個部分你要自己去找答案。


        往最壞的方向想，如果你太太完全不接受任何
        改變，就是要花那麼多錢，就是要放任自己的
        情緒，就是要打小孩。

        那麼，你準備好要離婚了嗎？你是捨不得離婚
        ，還是自知離婚之後你無法獨立照顧孩子，所
        以只好也跟著孩子接受你太太的情緒勒索？萬
        一你太太覺得嫁給你根本是從天堂掉到地獄，
        換她主動提離婚，你有把握爭取到孩子的撫養
        權嗎？

        如果不想讓問題變成如此不可收拾，先解決一
        切問題的根本--你太太的情緒問題，其他問題
        基本上都會順便解決了。
