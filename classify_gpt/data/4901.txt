誰不是以自己的出發點在做事?

重點是你們真的有想要解決生活上的衝突嗎?

你太太真的是把心中怒氣發洩在周遭，

這點要去個人諮商

: ex:
: 小孩吃飯會慢要提醒
: 但是太太講出口的永遠是"我很忙, 你吃飯這麼慢拖到我出門"

小孩是多小?

國小的應該可以溝通或是看懂時間吧?

放個小鬧鐘給他，在時間內(30-40分鐘)吃完，時間到請他拿去收好，

時間到沒吃完也是收好。

小的要看年紀，3歲以下也可以慢慢訓練。


: 晚餐太太會煮 但是不會以小孩喜歡吃得為主

為什麼要以小孩喜歡吃的為主?

: 卻很愛問"今天這個好吃嗎""什麼都要吃不要挑食"
: "這只有一點點辣, 要嘗試"

這邊聽起來還可以啊?

如果媽媽的廚藝不佳，這樣可能是個問題。


: 小的很愛黏媽媽
: 但是不管大的小的從小她就是不負責陪玩
: 能說本故事書已經是極限

不負責陪玩有兩種，

一種是不想做覺得沒有意義無聊的事情，

一種是不知道應該要做什麼，

可以讀故事書的話，那就一直讀就好了啊。


: 重點黏住了我也拉不走
: 但是她也不打算滿足她的需求

所以他的需求是什麼?

: 最後小的鬧到炸掉再反過來狂罵小的鬧脾氣
: 今天早上再次發生大的賴床被罵 吃早餐又被罵
: 我又被迫一定要起來一起盯著小孩

因為老婆不爽，所以起來罵人，你覺得可以嗎?

不是先安撫所有人之後好好吃個飯嗎?

(至少要安撫小孩吧)

: 很奇怪的是我每次在很嚴肅的講話

很嚴肅的講話是罵人的意思嗎?

: 她就要跟著念, 帶著情緒念
: 重點她永遠不承認她在罵(我跟兒子都認知她是在罵)
: 最後她又冒出一句"我為什麼生出你這種...(停住)"
: 我直接回她"你敢講出口試試看"

我覺得這邊真的要請他去諮商，

但你的回話也只是想壓下去而已，

雖然你老婆先說的很不應該說的話，

可你可以說類似

"請你不要說下去"

"請你冷靜一下"

"你正在說出傷害人的話"



我想我們的情緒高張都只是想要控制場面，

讓世界照著我們心裡面想的方式運轉，

小孩好好吃飯早睡早起自己寫功課，

可這些都是需要家長先花很多力氣時間去引導，

但不管怎樣都要有好的態度才有辦法控制場面，

情緒發散出去是很傷人的。那些責怪根本不是什麼大事情。



: 我真的覺得講話不經大腦的人很恐怖
: 小的愛吃魚皮肉皮 她就講"吃皮會性早熟" <--這句話導致大的不吃肉的皮

這邊覺得還好

: 只要教小孩最後就是爆走講"你就是笨就是白癡"
: <--小孩跟媽媽鬧翻的時候就直接承認自己是笨蛋白癡

但是嚴重在否定小孩的人格時候，

你在旁邊看嗎?  還是跟小孩說你不是笨蛋，不是白痴呢?

我從來不讓別人用這種負面的話否定我的小孩，

我媽有說是因為想激勵她，我就問她"你喜歡別人說你笨來激勵你嗎?"

我媽再也沒有說我女兒很笨這句話


: 再怎麼溝通都是我行我素
: 不合她意一哭二鬧三離婚
: 最麻煩的是連她自己爸媽都很輕描淡寫的忽略她女兒這種行為

他家都這樣是能怎樣，你要期望什麼?

: 畢竟在娘家爆走一次
: 一次在她爸媽來我家暴走
: 再回想剛結婚岳母跟我講了一句"她脾氣不好, 你多擔待"
: 其實回想起來真的是...
: 只能說結婚了小孩生了
: 在還沒鬧到簽下去之前只能請小孩們一起忍耐了

看一下正向教養相關的書籍吧

忍耐什麼? 你沒有罵你小孩嗎?

你可以先不要受到你太太的影響，

然後朝正向教養的模式訓練自己，

調適自己的心情。


我相信你一定為了這個家庭做了很多，

做了家事顧了小孩，

很累也覺得不平衡，

假設你太太心理出了問題，

或許轉念太太生了重病，

會不會讓自己心理比較好過，

好過的目的不是要你協助太太(她要去做諮商，像是生病看醫生)

目的是讓你可以更善待自己的小孩，

被太太波及的小孩，伸出手幫幫自己的小孩。


不管是不是要離婚，都先好好幫助自己的小孩，

然後你太太的原生家庭可以不用再複製下去了。
