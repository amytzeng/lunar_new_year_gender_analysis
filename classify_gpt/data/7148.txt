: 因為在外地讀書，這些事都是從媽媽和弟弟口中得知
: 也是我剛好最近回去後慢慢觀察看到的
: 以下正文
: 一開始媽媽覺得爸爸的行為最近變得跟以往不一樣
: 平常疫情前只會去運動中心打羽球的變得開始在家都會運動
: 平常飯後都會跟媽媽一起吃零食的連吃零食都拒絕了
: 原本以為他下定決心要認真減肥很開心而媽媽也以為是好事
: 但是弟弟前陣子說爸爸問他Line的問題，無意中從爸爸手機中的訊息內容發現跟一個女生聊天
: 所以弟弟先跟媽媽告知這件事並問媽媽這件事她要怎麼處理
: 後來媽媽趁爸爸洗澡看到他的Line跟一位打球的女球友過從甚密
: 而且女方已婚有小孩而且已經聊了兩個月
: 雖說女方對爸爸是沒意思但是最近從爸爸的行為舉止中發現手機不離身
: 而且異常期待跟女方回訊息種種行為都不像是從前的爸爸
: 這部分媽媽是覺得真的要走到離婚也沒關係畢竟我大學也畢業弟弟也成年了
: 媽媽自己也有一份穩定的工作（這一兩年要退休也有終身俸）也有娘家可以住自己
: 但媽媽比較擔心的是爸爸會不會因此被告或是因為這樣被打或什麼的
: 如果對方或她的家人到工作場域宣揚這件事對彼此也不太好
你煩惱的點
1:被告
沒啥問題!
現在的執政團隊以及司法機關
已經把外遇定調為民事問題,
大概是違反合約等級

你可以想像成婚姻只是一個合約
你爸爸只是違反婚姻合約
目前司法實務上,一般妨礙家庭
就是賠個15~30 萬結案
不會有任何前科也沒有其他問題

2:被打
更不用擔心了,除非對方老公是 8+9
就算如此,被打了可以告傷害
對方不想有前科來跟你爸和解
可以要更多錢!

3:到工作場域宣揚
除非你爸是那種有道德要求的工作
ex:老師
否則現在外遇真的稀鬆平常
也就茶餘飯後聊一下
你爸也這麼大歲數了,也不會影響升遷

綜上所述,你擔心的點幾乎都不是問題!
更不用說看起來你爸跟球場阿姨八字
都沒一撇,這可以算超前擔憂了~

: 因為說實話，爸爸現在再怎麼不好以前也是很顧家的
: 平常假日或寒暑假都會一起全家出遊，家事大家也都是分工的很好
: 爸媽平常也沒在吵架也都有不錯的正職
: 但真的不知道年紀也不小了 這次怎麼會這樣
: 所以想問問版友現在該如何開口跟爸爸說這件事情，畢竟是我們先偷看他的手機訊息
: 也想求助各位大大們這件事情該如何解決
: 謝謝
板友有提到幾點重點
1:資產處理
2:媽媽的身心狀況
我覺得都相當具有參考性
這才是你真正需要注意的問題


如果真如你所述
媽媽不怕離婚,
只是不知道怎麼開口

這太簡單了,
媽媽隨便安排一個出遊行程
或者回娘家,大概一兩周時間
不要短於一周!
然後都不要跟爸爸聯絡

爸爸找不到人一定來問你們
一開始你們就裝傻
等到爸爸急了
看誰跟爸爸比較熟的
就淡淡說一句
"媽媽好像知道有個球場阿姨跟你很熟"
就這麼一句話就好
其他就甚麼都說不知道就好

一周後你媽回來,
你爸經過這一周的煎熬
他心裡自然會有決定
屆時怎麼知道這件事已經不重要了
重要的變成是要怎麼走下一步!
