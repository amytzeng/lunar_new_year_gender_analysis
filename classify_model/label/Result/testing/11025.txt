-----------------------------------------(以下正文)

去年準備訂婚(11月)時，我發現我與男友溝通一直存在問題，雙方都在鬼打牆
(ex: 我覺得對方家人老是以禮俗和經驗下指導棋 ，我男友卻老是跟我解釋對方無惡意…
 )

因為訂婚前真的非常焦慮，兩人一起去了婚前諮商，我釐清了一些自身的問題 ，也試圖
在調整自己想法作法
( 以上面例子來說，我後來完全不給他面子，只要他家人直接對我說這些莫名其妙的指導
棋，我直接當場回懟，駁了這些所謂的好意，完全無心理負擔 )

自從諮商後，大概深刻理解了優點的同時是缺點。

過去因為我們是遠距的關係，可能小別勝新婚，每次短期相聚(週末、1  2週) 我當時真
的覺得彼此金錢觀、家庭觀、行為做事也相符，才有了後續婚姻的討論。

真正開始有婚姻的規劃，是對方回到家鄉後半年(我一直在家鄉工作，彼此雙親也是) ，
我知道他家裡人催著他結婚(而我也一直期待著)，直到我終於再次問了對方，對方才開始
著手進行婚姻的規劃

開始有了婚姻的規劃後，我發現跟我想像有了出入…
過去遠距時我們在處理事情就是很單純兩人的想法(因為他大部分沒意見，所以多數是我
打點一切我會處理妥當)
他搬回家裡後，一直都讓我有種錯覺： 我在瘋狂跟他媽媽、他家人 搶奪他的愛和時間分
配、甚至是金錢分配

先說明他背景：男友獨子，上面有兩個已婚的姐姐+他媽媽一個人

目前討論到婚後同居生活，
因為他不放心他媽媽，所以每週平日晚上2  3天會跟他媽媽吃飯( 不包括我) ，週末的時
候，如果他姐姐們回家，他會希望他回自己家過( 據我觀察瞭解，她姊約每兩週回家一次
)

對我來說，我大概已知他與他媽媽密不可分，他也無法反抗他媽媽，所幸的是他媽媽對我
來說還算好應付

但我無法理解，我們才剛結束完婚宴(4月底)準備進入同居蜜月，他又提出他會想回家跟
他姊相聚是什麼鬼
(我已經退而求其次不想與他媽媽爭奪了，為什麼我還要跟他姊他其餘家人分他的時間？
！！！)

而且特別跟我提出？！！！


不是欸，那相對的我是不是也可以提要求，我也要這麼黏原生家庭？？？

題外話，我家人完全尊重我(管不了我)，雖然我住家裡但吃穿用度一切自理(跟我家人分
開煮飯、洗衣生活作息一切分開，宛如同居室友XD)

我對婚姻的想法，重視層度：
婚後兩人小家>>> 雙方父母>>>>>>>>雙方其餘親友

對方的說法是: 依事情而定，沒有辦法排出優先順序，每件事處理方式都不同


唉，其實從之前4月底辦完婚宴後，每週都在吵架， 不論是他處理事情的消極、他家人婚
宴當天的出包(穿睡衣讓我敬茶等等的敷衍態度) …

引爆點是：上週末跟他家人出去，但不告訴我
起因是我有交代他負責的事情他要處理完(他也有答應週末要處理完)，因為我週末工作上
有事也沒跟他相處，事後再跟確認時，他沒處理，原因是他沒空處理( 臨時他家人揪他出
去玩)

我當然傻眼同時覺得所以？！！！ 那你不跟家人出去玩不就解決了？！！

他覺得我干涉了他自由，甚至回說：那是不是以後不跟我說，單純說他有事，就不會吵架
了


唉… 我該放生嗎？


已經不知道第幾次他無法處理好他家人、他、 我 之間的各種問題了

經歷了4次婚前諮商了，依然無解

每一次都沒解決，每一次對方也都逃避
(當然我也是一直忽視我心裡真正想法，想著轉念我心態也要調整，同時期待著時間會解
決…)


原本預期這三個月先同居同時磨合，9月左右登記… 現在我再次意識到我跟對方家庭觀不
合，往後可能更多衝突… 該持續下去嗎…唉！

-----------

先感謝大家的回應，我先回覆金錢關係：
1. 一切婚禮的所有的花費都是AA，如果要怪我禮服選太貴，還找婚佈婚禮小物等等的太
花錢我就認了 ( 沒辦法，價值觀的差異，我就是相對有主見的沒錯。)

2. 過去交往時，也都是以AA為基礎(這一餐他，下一餐我，交通他，住宿我等等的)

另外，我覺得他跟他媽媽吃飯含過夜我可以接受。(每週2  3天)
但是每2到3週的週末，他會想回家跟他家人在他家過夜，這一點對我來說我是真的不平衡
。
因為相對於我家人對他的邀約吃飯/出遊/過夜，他是有壓力的，所以通常會先沉默，我感
受到尷尬後我會直接幫他找理由拒絕掉。
(但也不是全部，只能說我過去我與他家人吃飯出遊頻率>>>他跟我家人吃飯相處頻率)


我沒有特別提到我自己跟我原生家庭的相處細節，但不代表不好喔！所以我自認為已經盡
全力為了對方更好融入我家庭，盡可能處理好我家人這邊的擔憂。


我只能說，謝謝大家指教，我會再想想。
