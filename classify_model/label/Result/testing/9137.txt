你會覺得離婚與否無所謂，會覺得老婆這樣對待自己也沒差，最大的原因就在於你從未體
驗過一段正常的幸福男女關係

就因為沒有過好的經驗，所以已經生活在痛苦中了，自己也不會知道


想必你周遭應該有幸福的夫妻朋友吧，有看過網路其他人分享的幸福婚姻經驗吧

在生活上看到人家在經驗你覺得美好的事物，可是因為在你的生活中沒有，於是你在文上
寫完零零總總的夫妻事件後，在最後再催眠自己不要去計較、不要去比較，一切都無所謂


這個不是你內心真正的聲音，只是一種不想要“不喜歡自己婚姻”的逃避而已

看著其他人幸福，心裡面有著那些羨慕，然後壓抑著不滿足，別過頭去想用各種最短暫、
最簡易的方式讓自己安於現況，這實在不是好的想法，嚴格來說是很迂腐的念頭


千萬別一直用自己的觀念生活著，當你po文其實也代表內心深處希望改變，所以原文留言
中很多鄉民的話真的要用心看

希望你能過得更好，別跟那個“可以過得更好”的機會絕緣了


※ 引述《lunyen (小書)》之銘言：

: 我太太問醫生的問題，充滿焦慮的巨細靡遺，我在旁都不好意思，
: 好不容易結束了，我連忙跟醫生道謝，
: 有一次醫生忍不住地問我一句：先生，請問你太太是不是中小學老師？
: 或者，她不滿意某個醫生的診斷或開的藥，
: 她會找機會衝進另一個診間，問另一個醫生，
: 另一個醫生多半只會說，尊重之前給你開處方醫生的診斷。
: 我之所以說，婚離跟不離，差別其實不大的原因在於，
: 由於工作的緣故，我們平常并不住在一起，算是假日夫妻，
: 四歲的女兒平常晚上是我岳父母在顧，
: 周末長輩回去鄉下老家休息，我們夫妻倆人照顧女兒，
: 我平時住學校宿舍，也不會每天晚上去看女兒，
: 原因是，每次周間晚上我去看女兒，
: 我岳父就躲進房間，關燈玩手機，
: 因爲他覺得彆扭，我走了之後，我岳父才出來。
: 我想，他彆扭我也彆扭，加上我要是忙起來也很忙，我就不每晚過去陪女兒玩了。
: 所以，我覺得就算離婚，我的日子跟現在相比，其實也沒有差很多啊！
: 至於共同的興趣愛好的培養？這個我是徹底放棄了，
: 我也沒什麽特別的愛好，平常工作就是個寫論文做研究上課的，
: 工作之餘，就是運動健身而已。

: 我得到的feedback就是：你日子會不會過得太爽了？
: 人家XXX的老公，生兩個，太太可以什麽都不管，老公帶娃帶得那麽好，你能嗎？

: 她的愛好，大概就是逛街，逛到我都投降，
: 我都主動說，你去逛，我在書店看書等你會合。
: 最後，有位版友說，要我不要那麽軟弱，
: 其實前幾天她噴我，我也說了，請她思考一下如果婚姻結束，是否對彼此都好？
: 這幾天，確實平靜了很多，倒不是完全不講話的冷戰，因爲她還會有求於我的時候，
: 要我洗碗買東西之類的。
