: 我原PO，想說事情快告一段落
: 就不太想回文打擾大家，但好像還是有些版友一直想要搞清楚一些事情
: 所以請讓我再浪費大家一些版面
: 這應該也是我最後一次回文了
: 講在前頭，我之前發文的本意，其實不是想要搞清楚誰犧牲的比較多、誰妥協的比較少
: 或許有些朋友會一直揪著我的文字組成結構
: 抨擊我下意識有這個想法啦怎樣怎樣，這點就讓大家各自詮釋了
: 我只是想要透過大家的回覆，瞭解我是否在這段關係當中有看不到的盲點
: 誠如之前的回文，大家的留言的確對我很有幫助
: 好，回到大部分版友最想搞清楚的點
: 1.生小孩
: 我沒有想生，雖然我喜歡跟小朋友玩耍、互動
: 但自己審慎評估過後，我覺得我無法成為一個稱職的父親
: 再加上考慮到太太的身體狀況
: 所以我跟另一半的對外共識就是不生

你有傳宗接代的壓力，但你個人也選擇不生
那就應該好好告知你父母，不要讓父母有錯誤的期待
說清楚後，這壓力也不會只有在你老婆身上
你若能分擔長輩的壓力
你老婆也不會有那麼大的負面情緒

: 2.不與父母同住
: 這點是一直以來的共識，就不多說什麼
: 3.減少一起回家吃飯的頻率
: 偶爾我爸媽會邀我們假日回去吃飯
: 可能一兩個月一次吧
: 但如果太太有事、或是不想去的時候，我就不會硬要他去
: 或是省事點，乾脆我們兩個就都不出席了

『不想去就不去』，這就是我說的，他只想當『老婆』，而沒有想當『媳婦』
所以他遇到衝突與矛盾，沒有想要去面對或是處理
不過他的壓力一部份可能是你第一點沒做好導致的
我覺得也不能怪他

: 4.宴客
: 我們一開始都是傾向雙方家長吃個飯就好
: 後面溝通到一個程度，由於岳父岳母堅持謝絕文定、聘禮這些傳統儀式
: 所以雙方就有個共識，不然就辦個家宴＋少少邀幾個我跟太太共同好友
: 大抵不超過10桌的等級
: 也因為有共識，所以我們才會一起去看飯店、簽約

我不確定這是『遷就後的結果』還是『共識』
如果不接受這婚就結不成，那我認為這不叫共識
如果對方願意用其他方式去替換他的堅持，去下放他的需求
那我才會把最後的討論結果看作『共識』
但沒有說遷就不對，或是不應該『被遷就』
但是你要知道每一件事的本質
你才會知道如何自我定位、做好心理平衡

: 5.你們之間還有愛嗎？
: 這幾天深聊過後，太太表示還是有
: 但已經被我們之間過去累積的相處問題消磨地差不多了

愛不是用說的，怎麼經營怎麼做才是重點
套用愛的真諦這首歌，愛是永久忍耐又有恩慈
那他應該會更願意去面對他與你爸媽的關係
讓你可以做好你身為『兒子』的角色
而不是不回你老家，讓你爸媽解讀成你們關係不融洽

: 6.寫在最後
: 我承認，在這段關係當中我的確很多部分都沒有處理得很好
: 個性上的缺陷、價值觀的衝突也是如此
: 但最根本的原因就是沒有像原本預期／承諾的一樣
: 幫我太太妥善抵禦來自我家那邊會造成他壓力的部分
: 這部分的確是我太過樂觀
: 所以什麼火山孝子、沙文主義、男權至上、搖擺不定的批評
: 我都虛心接受
: 諮商也已經安排好，至於剩下的部分我就不再多做回應了
: 還是謝謝大家這段時間的關心

網路的匿名性，大家的講話都會比較『直接』
你覺得有意義的接受，沒意義的當耳邊風就好
不用虛心接受，老話一句
你要先釐清事物的本質
把惡意的言論當作建議
那負面情緒憋在心裡就不好了
就跟你一直在遷就你太太一樣

: 最後結果如何，我都會好好收拾心情往人生的不同階段繼續前進
: 打擾大家了

真的不會打擾大家，這邊的人討論美語教育到有點疲乏了
很謝謝你的提問，所以我也很認真的在回文
希望多少能對你有些幫助
