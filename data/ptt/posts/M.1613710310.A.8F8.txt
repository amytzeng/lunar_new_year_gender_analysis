※ 引述《u4fu3 (chi)》之銘言：
: 剛好小時候家裡曾經歷有車到沒車的過程，分享一下我的感覺～
: 我家是的大約從我出生到7-8歲以前有車，在那之後爸爸把車賣了！原因是當時的車子老
: 舊了，爸媽覺得開車的頻率其實並不高（剛剛問我媽 有車時大概一個月出遊1-2次）當時
: 覺得再買一台車不便宜，所以家裡就沒再買車
: 我覺得有車跟沒車，開車出去玩的頻率一定會降低很多很多，從我小時候旅遊的照片可以
: 明顯看得出來XDDD雖然大家都覺得要出去可以租車，但會有以下幾個阻礙
: 1.成本思維
: 有車時，開車出門不會把「車子折舊」當作成本，反而會覺得車子就是要開啊，不開放著
: 會壞掉！
: 但租車出去玩，心裡會把車租當成成本，容易想著這趟出去玩三天兩夜要多花多少錢...
: 有時候想想就覺得不划算

  這是真的，不過隨著自己的情況（居住地 工作）這個思維可以慢慢調整了
  我也曾經覺得租車三天兩夜好貴 但是出去玩 不管有沒有租車
  住宿吃飯都是一定要的 租車三天兩夜加上油錢 算9000的話
  在台北市租車位＋養車一個月絕對超過
  （我住台北市內 車位如果要租24小時的超過八千 只租晚上的話也是要四到五千
養車的油錢 保險費 稅金 保養 算下來一個月也要六千至少）

: 2.便利性
: 租車需要駕駛人騎車或搭車去牽車，旅遊回來也還要駕駛人把大家載回家後再去還車，旅
: 遊時間會收受壓縮！借還車時間也要配合租車公司的營業時間，早上六點想出發還不行，
: 回程有時候也會著急著要還車！簡單來說就是比較麻煩啦

如果在台北市以外的地方我不確定 但是台北市現在有租車是類似共享
（就是大家說的那家啦）家裡不會太遠的公共停車場也許都有車喔 app可以預約操作
缺點是連假要提前約 但我們自己試驗過 租還車加起來不需要十五分鐘
（可先查查附近有沒有服務的停車場）因為公婆住其他地方 接他們來家裡或是
先生要帶他們出門就會定這個 除了過年外 我們來回一天以內記得都是兩千上下
（當然 這種頻率不高 平常回婆家也是坐大眾交通可到）


: 3.彈性
: 租車比較難說走就走，有時候只是想去北海岸喝咖啡、陽明山看夜景、烏來逛老街泡溫泉
: ....這種一日遊半日遊有車時你可以當下想去就去，沒車時...動力直接少一半（更不用
: 說很多人會覺得去北海岸喝咖啡幾小時要租車一天超不划算！但很多地方搭大眾運輸真的
: 就是很麻煩，想到那個交通跟路途，還有帶小孩大包小包就冷一半啊）

這是真的 機動性會比較少 如果比較喜歡說走就走的
沒有車相對會不方便


我認為是養車是一個選擇 就像有人喜歡打球 喜歡裝家庭電影院
喜歡幹嘛幹嘛一樣 算下來一定都是花費
但就看你真心喜歡的是什麼

養車的附加好處是可以說去玩就去玩 萬一家人需要
支援好像也蠻快
但是缺點就是 如果沒有這麼喜歡說走就走 可能就多考慮一下
現在租車也有很多小時計費的 前提就是先看看自己家附近有沒有

婚姻點：我們婚後先生長期有配車 但很少開（當時上下班很方便）
後來工作調動 車子沒了 我們就是用需要時租車來調整
前陣子他說公司想要配車給他 但是停車費應該不會補助全額
他自己是不想要 我也不想 我說現在附近停車費一個月晚間要五千
白天公司車位也不一定有等等等等 即使公司養車我都不覺得划算
（我們兩三個月才會需要一次車子）

還有一點 是我先生並不是很喜歡開車的人
上下班他坐捷運相對舒適
如果為了停車費用開車上下班
（我家附近月租整天的要九千以上）
他的情緒會有不好的影響XD
所以我的成本中還算了情緒與體力成本（有些人真的不愛開車啊）

但我們沒有小孩啦 兩人出門也習慣輕便
這是懶人的小算盤



: 結論：
: 養車真的很貴，要算錢的話肯定算不贏租車！我自認也是蠻精打細算的人，也住在捷運走
: 路就能到的地方，上下班也是騎車比較快（台北超塞！而且停車費超貴！）但我跟先生還
: 是買車了！也沒後悔過，雖然每次繳稅金跟保費時都會抱怨一下好貴XDDD但全家出遊的幸
: 福感是蠻踏實的
: 我到現在還記得小時候全家出去玩的一些事情，雖然是7-8歲以前的事了！也記得我國高
: 中時坐親戚的車出去玩，心裡其實很羨慕他們一家人可以這樣開車出遊，一起在車上聊天
: 吃餅乾XDD（啊我們一家人還因為沒車 只能分散坐在不同親戚的車上...但那時候完全不
: 敢跟爸媽說，怕他們聽了傷心)
: 另外你一個月抓一萬養車應該是抓多了！我們家開3500cc的車，油錢、稅金都算重，但一
: 個月不含車價攤提，平均養車費用也沒到一萬塊啊...建議你可以再精算一下～
