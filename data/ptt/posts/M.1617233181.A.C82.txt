: 對於如何當媽媽賦予很大壓力
: 好多的時候我覺得很難為
: 當家庭主婦的都會希望經濟要獨立
: 自己要可以賺錢 不用拿錢看人臉色過活
: 可是當職業婦女卻常需要為小孩犧牲
: 小孩生病通常會是媽媽要請假帶小孩
: 白天忙著工作，下班忙著張羅晚餐
: 孩子小時候若是學校有回家手工作業
: 要陪孩子想幫忙動手完成
: 學校辦活動也要親自參加
: 做家事 陪小孩 忙了一整天精疲力盡
: 平常累得要命 假日要帶小孩出門玩
: 會很懷疑什麼時後能有屬於自己的時間
: 為工作認真努力 又要為小孩用心良苦
: 至於現在的食品安全問題
: 總是希望自己的小孩吃的健康又營養
: 如果自己煮早餐就得早起
: 如果買外食就又希望能好吃又健康
: 但殊不知 食安問題搞得煩惱
: 不能吃太多糖 不能太鹹太油
: 市面上食品添加劑很多
: 買青菜要擔心農藥問題
: 肉類又怕有生長激素 瘦肉精
: 自己煮的要美味 水果要多汁
: 一邊煮還要一邊注意小孩動向
: 希望他們纖維質要吃夠 吃健康
: 少油少糖少鹽 但偏偏小孩又愛偏食
: 家裡要保持乾淨對好好休息很重要
: 若是雜亂的空間 對心情跟思緒有壓力
: 擔心塵蟎會導致過敏 氣喘
: 吸塵器 除溼機 空氣清淨機
: 市面化學合成清潔劑對身體可能有影響
: 垃圾要分類 時間到要拿去丟
: 東西要整理收納 有些時候還要佈置
: 營造家有溫馨感
: 至於注重外貌協會的
: 又會說女人要美麗 不要當黃臉婆
: 魅力要會打扮 給孩子跟先生都有面子
: 身材要保持苗條 最好看起來很年輕
: 保養得好 看起來有自信
: 還要擔心孩子童年快樂嗎？孩子交那些朋友？要學才藝？要存多少錢當教育基金？要買什

: 要有工作、要漂亮、要給孩子吃的營養、過得開心、住得舒適、快樂長大、注意安全、親
: 子互動關係好、夫妻相處融洽、家庭教育不能少

: 太不盡責了
: 可是問題是 我們也是人阿
: 為什麼社會上給予媽媽的壓力越來越大


其實這或許跟身為媽媽沒有這麼強烈的關係，
或許是本身的焦慮性格、價值觀及從眾的心理。

就飲食來說，何謂過油、過鹹、過甜。
我可以吃個糖醋汁糖超多，
也能吃個沙拉，醬汁以酸為基底，
只用少量的糖去平衡味道。有些菜就是要甜，
有些就是要酸，這是味型。

做深入的研究，不要道聽塗說。
至從大學畢業以後我已經十多年不看電視了，
發現那些困擾我的雜訊都消失了，
現在我若是要了解一個主題，
會花短則數天，長則數月透過不同平台收集資料，
你的健康觀念是來自於電視頻道或是平面媒體?
抑或是身邊社交圈的討論話題?

你或孩子本身有氣喘嗎?
有驗過過敏原嗎?塵蟎機有多少實質效果?
資訊的來源是廠商嗎?
我氣喘30幾年，驗過過敏原，結論超多，
連茶都是，但我喝茶從沒過敏過，
看他的刺激不夠，我不需要擔心，
對我個人而言灰塵、跑步、感冒最容易激發。

化學清潔劑對人體有害，
當清潔油污時，天然的又是否有效?
我老婆最愛拿著刷子刷發霉的浴室磁磚縫，
我說這樣很浪費時間，她說，她現在有時間。
我就拿著瓢白水，直接噴在牆上，開抽風，
30分鐘後回來，用水沖沖牆壁，去沙發上廢，
輕鬆愉快!

我老婆的問題在於，你不能永遠都假設，
自己有足夠的時間及注意力去處理這些問題。
一件生活小事可以，那五十件生活小事呢?

你對育兒、飲食、教育的擔憂，
處理的方式，你有足夠的注意力面對所有問題嗎?
以個人的經驗來說，若從不提升層次深入了解、
解決問題，
那這些事就會不停的在你身邊繞，
不停的消耗你有限的注意力，
永遠都不會被解決。

社會認同亦是，
我結婚四年，沒拍婚紗沒宴客，沒有蜜月。
很多朋友都，難道你老婆不想拍嗎?
一輩子一次而已耶!
我們就看著他們，
心裡想著，我們是不同世界的人。
回頭又看著，結婚10年，拍了婚紗宴了客，
遵循了古禮，蜜了月，買了鑽戒，
生了兩個孩子，現在卻要離婚的姐姐。

到底我現在擔心的，究竟重不重要，
又是否以正確的態度面對。
