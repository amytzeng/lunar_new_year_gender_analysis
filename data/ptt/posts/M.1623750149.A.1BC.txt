: 想請教版上的夫妻
: 婚後兩人的生活什麼才是最重要的？或是你們是如何判斷這個人適合一起進入婚姻的呢
: **該篇也在womentalk內po過，但我覺得描述不好沒有清楚表達自己最想問的，以及有些
: 額外補充其實不重要就修改了
: 會這麼問的原因是跟男友交往6年，也到了討論結婚的時機點，但我自己也一直在內心評
: 估是否適合結婚，以理性的討論，整理幾點如下：
: 1、男友包容度很高，有爭執時能理性/溫柔討論&釐清
: 2、會幫忙分擔家事，雖然生活能力不太好，但願意嘗試
: 3、不用擔心經濟問題，我們倆都可以負擔自己的經濟
: 4、很疼女友，會買小點心或是小禮物哄女友開心
: 但偶爾疑慮時的幾點：
: 1、我是比較重視生活&未來目標的人，雖然不是什麼抱負，但會希望自己在各種地方都能
: 有小小進步或是學到新的東西，但男友覺得工作/經濟穩定就好，下班只想打手遊跟看NBA
:  好好放鬆 ，也不認為要擬定
: 計畫，或是小計畫然後按部就班
: —>我希望另一半對未來有憧憬/目標以及願意去付出，完成每一階段的小目標。
: 譬如我會希望自己在工作上更有成就所以去學一些工作技能，或是希望自己維持好一點的
: 體態學習怎麼重訓跟健康飲食，更長遠的目標可能要思考如何買房或是人生規劃
: 2、我們倆是互補，據男友的說法，他覺得我比較文青，重視氛圍喜歡一些儀式感或是生
: 活體驗的事，譬如我可能覺得兩人一起釀梅酒很浪漫，或是一起去學手沖咖啡能提升生活
: 的體驗也很有趣，但他心裡真實的聲音可能就會覺得麻煩&浪費錢，無法從中獲得真正的
: 開心或理解意義為何？（雖然以前還是會因為讓我開心而願意一起），但畢竟要真的喜歡
: 或是有興趣才是不會有負擔
: —>雖然興趣不一定要一樣，但我覺得每個人都要知道怎麼過自己喜歡的生活，他可以愛
: 打手遊/看NBA，我偶爾也會陪他一起追動畫聽他說打手遊的事情，但有時候我發現我不在
: 家或是回老家，他可以兩天一夜只打手遊/吃飯/整理家務，我就會很困惑如果沒有邀約你
: 做一些其他的事情，是不是生活就是這樣了，大家可以理解當一個人專心作自己有興趣的
: 事情或努力時魅力度會大大提升這事嗎～（或是我應該好好欣賞他認知打手遊&研究NBA的
: 時刻）
: 在進到討論婚姻這件事時，我會不斷思考剛剛提到的，在生活品質/認知上我們的確有些
: 差異，也造成有時候我覺得有種難以言喻的落寞，我覺得每個人都應該要好好體驗生活，
: 但男友可能覺得上班很累了，下班只想在沙發上滑手機看電視結束這一整天
: 所以我很好奇，當進到婚姻，是不是只要另一半愛你/會對你好/願意分擔家事就已經是達
: 標了，
: 我下面的這些對老公的要求其實只是折磨自己折磨他人而已
: 1、對方知道自己的生活小目標是什麼，會朝目標前進 ，無論是工作或是人生
: 2、對生活保有熱忱，嘗試自己有興趣的事情，可以跟對方分享

婚後跟婚前最大的差別在於兩人會長時間的生活在一起
所以妳該注意的，是這個人所有和生活相關的面向跟妳是否契合

最主要的幾個面向有：住哪裡、要不要生小孩、價值觀、生活習慣

前兩項是一翻兩瞪眼，如果兩人無法達成共識，基本上也不用結婚了。
後兩項會決定妳跟這個人長時間相處下來對妳人生品質造成的影響。

很多女人把另一半對自己好不好當成適婚指標，
我只能說有的男人是非常目標導向的生物，
他為了達到目的可以六親不認、可以臥薪嘗膽、甚至胸口碎大石，
一旦目的達成，他就會恢復原來的面貌，
所以這不見得可以作為一個準確的指標。

至於妳說對未來有目標......
我至少可以舉出三個生活有目標卻苦了他們另一半的人夫當例子：
第一個下班後自學各種樂器、喜歡做木工和爬山健行，
他用同樣的標準對待他老婆，但他老婆對這些事都沒有興趣，
我常聽她老婆抱怨每個周末都被搞得累得要死，
不然就是聽他抱怨他老婆是個不積極的沙發馬鈴薯；
第二個學外文、考證照、還參加地方上的義工團體，
這位人夫雖然不會拖太太上山下海，但是他把所有家事都丟給太太，
所以太太只能一人扛起所有家務，還要照顧高堂和幼子；
第三個是王牌業務，為了打入外國人的生活圈，
跑去學品酒、學西式料理、高爾夫、唸EMBA，
他雖然不勉強太太參與，但時常嫌棄太太做菜難吃、言語乏味沒有見識，
他太太已經很努力學習新知，卻始終趕不上先生的腳步。
對了，這三位人夫都很高薪，
所以他們的太太都說過"看在他會賺錢的份上就巴結一點"這種忍氣吞聲的話，
如果妳嚮往另一半有明訂目標規劃未來的特質，
也許妳該想想：妳有她們的覺悟嗎?
