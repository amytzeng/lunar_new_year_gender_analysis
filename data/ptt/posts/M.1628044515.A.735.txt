針對酵母事件來分析

: （事件3）
: 某天午休前我先把要做黑糖饅頭的材料備好，後來臨時想多做起司口味的
: 下樓時公公還在午休，我就先出門去買缺少的材料，回來時......
: 公：妳那些酵母粉有多少?
: 我：4克
: 公：4克太多了，媽媽每次用麵包機也只加一點點
: 我：不會呀，是因為媽媽麵粉用比較少八
: 公：妳的麵粉用多少?
: 我：400克
: 公：太多了 太多了
: 我：不會太多啦，通常酵母粉和麵粉的比例是1:100
: 公：太多了啦，一半就差不多了，太多了
: 我：（安靜不說話）
: 公：太多了太多了，一半就差不多了啦（無限）


公公說這些話完全可以忽視，
因為平常他也沒在做，他不知道酵母的比例，
也不知道婆婆平常做的粉量，
跟沒做過的人討論不必要。也不要放在心上。


重點在婆婆，才是有實際操作經驗的有用討論。
烘焙是科學，講究比例和精準的重要，
並不是像料理很隨意，哪個多加少加沒關係。

原po和婆婆這部分的概念都是有的，
麵粉對酵母的比例100：1  正確，
所以並不是酵母是多貴啦很小氣膩，
而是有沒有正確取量的問題。


: 晚餐後，我婆婆去拿了我做的饅頭吃
: 婆：好吃呀，爸爸說妳酵母粉放很多喔
: 我：還好吧，4克
: 婆：我之前用麵包機做麵包放一點點
: 婆：用280克的麵粉
: 我：那是因為你們麵粉比例比較少，通常酵母和麵粉比例是1:100
:     妳們的酵母比例大概就是2點多
: 婆：對呀，2.3
: 我：我也沒多放，磅秤到4我就停
: 婆：妳用磅秤喔，那不準，通常我都用這個量（拿給我看）
: 婆：上面有刻度，像我每次做就是放一匙，就是2克左右
: 我：所以如果我的酵母太多，那就是那磅秤有問題
: 婆：對呀，不要用那個磅秤量酵母，因為那沒辦法量到0點幾的
: 我：恩恩，那就是磅秤的問題，因為它沒辦法量像酵母粉這麼輕的東西
: 婆：妳下次就用我這個量
: 我：但那也沒辦法精確的量出克數
: 婆：是阿，沒辦法
: 我：恩恩好

如何正確取量到四公克的酵母，
這裡出現錯誤。

酵母是很輕的東西，一般料理電子秤
大多秤到一克，但是重量太小也容易誤差，
像取1克的鹽等到它感應到的時候，
已經到了2克，有時還要敲打一下才會跳數字。

更何況家裡的秤本來就疑似有問題，
它跳到4的時候都不知道是多少了= =

所以，婆婆說不要用秤量是對的…
小份量很容易秤不準。
面對酵母很輕又需要精準使用的難題，
有種東西叫酵母量匙，專門用來計量酵母，
它已經把體積和重量刻度好了，
不會再秤錯啦，而且還 —— 很便宜。
