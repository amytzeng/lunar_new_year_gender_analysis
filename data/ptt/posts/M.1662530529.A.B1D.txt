: 是否同意記者抄文：否
: 在此分享去年買房的經歷，差一點沒買到，心有餘悸，
: 給同樣買房艱辛的網友參考…文有點長，請見諒
: 背景說明：
: 我和老公在南部，公婆在中部，老公姐姐(簡稱大姑)住北部有男友
: 有5人群組(我.老公.公婆.大姑)
: 公婆所得稅級距20%而且薪水差不多，其中一人的薪水等同我們夫妻加總
: 去年結婚沒有聘金嫁妝，婚後各自經濟獨立財務各管、家用平分
: 婚前已協議不與公婆同住
: 婚後先租屋同居：
: 591物件很少，只有兩間可選，租屋約看房時，老公將照片po在群組，
: 婆婆：風水不好，我只是表達意見，不負責也不阻止簽約
: 最後放棄簽約，第二間房婆婆沒意見於是簽約，住進去反而問題多
: 工作環境穩定後準備買房，討論雙方可以拿出多少錢：
: 老公：我爸媽會贊助，如果買大房子，他們老後搬過來，會贊助更多
: 我：婚前就說好不同住，為什麼要買大房子？
: 我：不希望被干涉，就算完全沒贊助我們也可以負擔頭期房貸
: 最後結論：
: 1.公婆各自出100萬，其餘我和老公分攤
: 2.老公堅持買屋前一定要公婆看過
: 3.我堅持不被公婆干涉，如果要干涉，大不了我自己出錢。
: 找房過程：列出具體數據比較有感
: 條件：(1)地點在特定區域(2)室內大於20坪(3)房價800萬，可稍微超過
: (4)屋齡15年內(5)平面車位(6)管理費2500內
: 這樣的房子很搶手，沒看到任何物件
: 一個月過去了，老公覺得再等等，我眼看房價漲感到焦慮
: (找那麼久都沒合適的，即使有，在公婆來看之前就被搶走了吧…)
: 實地看房只看到A屋符合大部分標準，但室內只有18坪
: 老公不滿意，我覺得還能接受，在某天群組：
: 婆婆：這周六要南下看A屋，想避開下週農曆7月
: 我：不好吧~~有颱風，怕開車危險
: 婆婆：我就是要去！你不約房仲，我就自己約房仲
: 於是我約了房仲，看房前一天公婆說有事要取消…
: 兩週後，公婆又說周六來看A屋(之前說農曆7月看房不好…？)
: 公婆南下前一天突然售屋網出現各方面條件吻合，出價858萬的B屋，
: 於是隔天看A屋B屋，看過B屋後每個人大致滿意(頂多一些小缺點)，
: 報價含裝潢及大部分傢俱家電，符合實價登錄，週日下斡旋，公婆沒反對
: 賣家答應週三下午當面談
: ~~週三當天早上~~在群組裡~~
: 公婆：太貴，屋主一轉手賺100多萬，別人都說貴，斡旋從7XX萬喊
: 公婆：改到周末，我們南下談，或者不要簽約，我們周末南下簽約
: 公婆：契約要有審閱期，不然違法，我要看契約書，誰知道賣家有沒有亂寫？
: 公婆：上次看房時間太短不準，要看第二次
: 公婆：房仲只想賺你的錢，騙你說房子你不買也有其他人買
: 大姑：你是不是覺得花我爸媽的錢就可以隨便出價？
: 大姑：注意你的態度！
: 大姑：弟你要小心以後會被家暴！
: (無論是否行情價，公婆永遠嫌貴，大姑覺得是我”拿”公婆的錢)
: 下午跟賣家洽談順利，賣家沒多要價，在簽約前一刻
: 因為之前公婆強調簽約前務必通知他們，於是老公打電話
: 婆婆：再殺價2萬
: 老公拿著手機轉頭對賣家：我媽說再砍2萬
: 賣家臉色瞬間很難看，老公說完又掉頭講電話，留我一個人面對..
: 老公跟公公電話長達1個半小時，電話那頭的公公就是不同意，
: 嫌貴、挑缺點，無論怎麼解釋反駁在他看來都是錯的，
: 我擔心再等下去，賣家會反悔，畢竟還沒簽約
: 一旁的房仲看不過去搶走電話，10分鐘後說”你爸同意了”
: (老公全程講電話，沒有我插嘴的餘地，我原本以為半小時內會好…)
: 現場簽約，房子住到目前都蠻好的，樓上同坪數也在半年後漲2~300萬，
: 事實證明，後來市場上也沒有出現一樣條件的房子。
: 感想：
: 我很感謝房仲擺平公公，大概房仲也看不下去了吧…
: 房子跟我們有緣，不然不會在公婆來的前一天出現
: 事後我跟老公爭吵，我不懂為什麼一定要公婆點頭才簽約？
: 我哪裡得罪大姑？同樣的話由我說，大姑指責我，老公說就沒事
: 被指責時我很難受，公婆站在大姑那邊，我從沒有在他們口中聽到道歉
: (老公有向我道歉，因為他沒有保護好我，以後統一由老公發言，我安靜)
: (我拿群組畫面問周圍朋友，措辭語氣都很正常，也不懂為什麼要攻擊我？)
: 大姑有在看ptt，以下這些話想對大姑說：
: 希望這篇不要放到群組鞭我，跟妳一年也只相處1~2天，
: 如果我有得罪妳的地方，我向妳說聲對不起，可能我就是被妳討厭吧？
: 還是說我不該反駁公婆？就該乖乖配合、當一個服從的媳婦嗎？
: 如果我屈服了，房子沒買到，你們有想過我的感受嗎？
: 房仲說過一句話：有的父母還沒學會放手…
: 我冒著被罵的風險寫這篇文，這種故事情節太多了…相信很多人跟我一樣
: 買房很不容易，我只想要一個舒適的家，可是好難…怎麼這麼難？
: 花自己的錢不能作主，房子名字寫老公(跟老公商量好的)群組就一片安靜…
: 感謝各位看到這裡，買房是婚姻的一大考驗(殺手？)，希望各位買房順利。
兩百萬的事情很多人都說了，反正參考看看，該說的都有人說了

買房子這件事情老人可能不如年輕人懂市場，但買東西經驗和社會經驗絕對屌打年輕人

年輕人在做很多決定有老人家瞻前顧後，長輩的疑慮可以不用當場到說服長輩

但是把建議和顧慮思慮進去作對的決定，往後後悔的機率比較低

而且看起來公婆的社經地位是比小夫妻你們兩個好的!他們的決定比妳們對的機率高

另外不論是房仲或是賣家都是外人，再怎麼談得來這些人都只是你生命中的過客

公婆和大姑才是你更該在意的人，賣家的重點是脫手，房仲的重點是促成買賣

公婆的本意是幫妳們不做後悔的決定!交易上的演戲看看就好，在意該在意的

不要本末倒置了!

另外在買賣的市場，妳太年輕了!不動產的市場沒有這間沒買到就天崩地裂

這個心態要自己想通!心急是吃不了熱豆腐的，由其實買房子這種大筆的交易，更要心平

最後我要提醒 很多事情兒子可以做的不代表媳婦可以做，

如同女兒可以做得不代表女婿就可以做! 媳婦再怎麼跟公婆貼近，和女兒就是不一樣

妳老公可以說的話不代表你可以說，講激烈一點，妳老公可以跟公婆沒大沒小吵一架

過幾天他們可以像沒事一樣坐在客廳看電視抬槓，但妳只要有一點臉色就會被記一輩子

以這個角度再回去好好檢視你的發言

給個建議就是，妳覺得合理但可能和公婆想法不一致的發言先說服妳的老公

說服後老公由老公去發言! 如果你連老公說服不了更遑論說服公婆了。
