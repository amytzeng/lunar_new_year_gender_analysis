: 事件一
: 看電視世界新聞某國經濟狀況不好導致房價大漲
: 租屋價格大漲40%（數字忘了）
: 妻：「好可憐喔連租屋都漲成這樣生活一定很辛苦。」
: 夫：「你不知道他們都住國宅嗎很便宜，而且租遠一點就好了哪會辛苦。」
: 妻：「如果是我們房租漲那麼多倍被迫搬家的確很辛苦啊，通勤很累，他們很可憐耶怎麼
: 沒有同理心。」
: 夫：「那不然你要捐錢給他們嗎？」
: 妻：「我單純看到新聞的有感而發說他們房租暴漲很可憐，為什麼要扯到捐錢？」
: 夫：「你覺得他們可憐就捐款啊我有說錯嗎？」

有沒有國宅我不知道，但他就是為了否定妳，而且講了「租遠一點就好」很矛盾

我會反問他：租遠一點為什麼不辛苦？搬家跟通勤不辛苦嗎？

他可能會愣住不知道怎麼回答，或者發現自己的矛盾點（前提是他有自覺想改進）

但因為妳的「沒有同理心」另開戰場了，所以他也攻擊性回應，這段話題無法繼續

: 事件二
: 最近颱風的關係風切聲很大
: 妻：「有點擔心颱風已經走了風切聲還那麼大，冬天東北季風感覺也會很大聲。」
: 夫：「不會，颱風的風是四面八方吹過來才那麼吵，東北季風只有一個方向不會那麼大聲
: 。」
: 妻：「東北方正對主臥衛浴的窗戶，我就是擔心衛浴那個方向的聲音啊。」
: 夫：「不會，東北季風是有方向的。」
: 妻：「對啊我指的就是衛浴的方向啊。」
: 夫：「好好好你說得對。」

坦白說，應該是妻子沒有明白，他的意思是颱風沒有固定方向是亂吹

但東北季風是一個方向的吹，跟家裡窗戶的方位沒有關係...

不過他發現妳沒搞懂以後，看起來也沒耐心解釋所以就敷衍過去

或者他覺得妳在反駁他：對啊我就是說這樣啊！

其實這段妳用問的會比較好，問他講的跟妳講的差別在哪裡？

兩邊都不要用反駁的方式對話，才能冷靜的發現對話問題點

: 事件三
: 妻：「昨天因為發生了一些事情心情不太好睡不著失眠了。」
: 夫：「我昨天也因為思考股票不知道要買哪支所以也失眠了。
: 妻：「為什麼我跟你說我心情不好，你不是應該要先問怎麼了嗎，而是直接忽略講你自己
: 的事。」
: 夫：「沒有啊因為你跟我說你失眠，我也跟你說我也失眠啊，難道只有你可以講你失眠我
: 不可以講我也失眠嗎？」
: 妻：「你可以主動跟我講啊，但因為是我先講的，先後順序不是應該你先關心我再換我關
: 心你嗎？」

如果丈夫看這篇文的話，想跟丈夫說：

妻子一開始說發生了「一些事情」，就是在丟球等對方回應「什麼事情？」啦

這點比較偏向不關注對方的發言，只聚焦在自己身上

需要被關心這點可能需要更直白的表達

: 事件四
: 夫：「以前這條商店街很多店家現在都倒光了。」
: 妻：「對啊，可能疫情期間成本太高了撐不住了。」
: 夫：「不是，是現在人力資源基本時薪一直調高，但來店客量又不足，店家會精算開店根
: 本賺不到錢，不如關起來不做了。」
: 妻：「我們講的原因不是一樣嗎？」
: 夫：「沒有，不太一樣。」

妳可以問他：我們說的地方哪裡不一樣？

檢討下來會發現，其實他講的跟妳沒衝突，只是想補充其他觀點而已

偏偏開頭要講一個「不是」反駁

只要把話改成：

「對啊，而且現在人力資源基本時薪一直調高...」

就很可以繼續聊下去了啊

: 雙方雖然有共識要改進但好像沒什麼進度

其實我很想知道你們結婚前是怎麼相處的

怎麼可能這麼聊不來...熱戀時期是怎麼度過的XD

看下來丈夫問題比較大

既然你們已經有共識想改變，這樣很棒啊

以後發生這種狀況可以不要逃，來好好檢討分析一下

妻子也先不要表現生氣，耐著性子讓他知道這樣反駁令妳不舒服

想討關心那段也建議直接說：我想要你關心我（事件三）


他就是習慣性反駁，自己也沒意識那種

聽到關鍵字就只想講自己的想法、自己的事情，沒有關注別人在講些什麼

而且很容易陷入爭論模式，只要妳反駁他就會開始嘴硬或是敷衍


題外話，這種人社會上不少

我前同事也是無論說啥都要：不是、沒有、不對、不一定喔

連跟他說：我感覺OOO、我不喜歡OOO、我偏好OOO (喜好話題)這種沒有對錯的事情

都要回我：不一定喔...可能...

我：我說的是我的感覺，哪裡還有不一定？

他：呃......

靠杯難聊，別說是夫妻了連一般人也很難當朋友


回過頭來，我自己想的方法是：

當他先講了否定開頭，後面講的跟妳一模一樣的時候

可以追問他講的跟妳講的哪裡不一樣，去釐清對話哪裡出問題

通常是兩種可能：

1. 你們在講不同的事情（事件二）

2. 其實類似，只是習慣性否定（事件四）


我先生偶爾這樣被我抓出來問：那你為什麼剛剛要說不是？

都會笑說：對欸我剛怎麼會說不是？
