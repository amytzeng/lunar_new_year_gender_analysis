建議你們去做婚姻諮商

這篇看下來
你懷疑自己有人格障礙 也許這樣的觀察不是沒有根據
有沒有想過是亞斯伯格光譜之類的？
（這邊不是做診斷 只是外行人的直觀感受 請找心理醫師）
因為你對一些句子
跟人際互動的解讀的確異於常人
似乎不懂“情境”的意思
就是一個句子在不同場合有不同意思
而且要考慮講話的人“為何要講這句話”
而不是只把一個句子單字 單獨挑出來看 不管脈絡

這邊說“不能理解為何負面情緒會外溢”
有點偏題了 也讓一堆人戰場跑歪
實際上是你不懂為何你的某些反應
會引起你老婆的負面情緒

我可以理解你老婆跟你這樣朝夕相處
可能會很難忍受
因為別人一聽就懂 看個眼神就知道的事情 你不只不懂
還一直想要用你自己的方法分析

但這不一定是一種錯 而是天生的個性
我覺得你應該跟你老婆好好講開 你真的不懂 不是你故意激怒他
如果你的個性就是無法理解這些溝通上
除了字面上以外的意思（情緒，語境，表情）

那也許透過第三者 分析你的個性
讓你老婆理解 你就是這樣的人 進而包容你 會更好

: 是否同意記者抄文：/NO
: 很抱歉再次佔用版面，我真心想解決婚姻問題

: 我不能理解為什麼負面情緒會外溢
: 1.為什麼
: 妻：為什麼吃完東西不收？
: 夫：因為剛接電話，然後...
: 妻：我沒有要問你為什麼
: 夫：.......
: 我真的無法理解或同理
: 什麼時候疑問句其實是祈式句
一般人會直接認為女方在生氣 所以就是在“質問”你為何不去做這件事
質問就是用問句來責怪的意思
你該做一件事兒沒做 他在責怪你為何不做
而你卻不懂為何問句=祈使句 實際上不是祈使句 而是在質問


: 相同的例子還有
: 誰吃完東西沒收？
: 你吃完東西可不可以收一下？
: 桌上東西吃完，是誰要收？
: 早期我會很認真回答，然後就是爆點開始
這就是問題
你太把句子用字面上的意思解讀 不看語境
也許還讀不懂老婆的表情

: 中期我會默默收拾，然後得到一堆埋怨
: 現在我會主動先說「我等一下馬上處理」
: 然後得到一個白眼
等一下馬上處理 他可能覺得你在拖 所以生氣


: 2.選邊站
: 妻：我跟你講，我公司同事很機車
: 夫：大概因為他上個月業績很差吧
: 妻：你幹嘛幫他講話
: 夫：我沒有
: 妻：你有
: 我以為這是個「為什麼機車」、「多機車」的討論
: 所以婚姻生活的大吵，幾乎都是因為這種議題
: 現在才知道這是尋求情感支持
: 但我還是不能理解，老公也覺得很機車為什麼重要
: 同事又不會少塊肉

他希望你同理他的難受
但你在那邊事不關己的分析


: 3.追問
: 妻：我香港同事英文好好
: 夫：是跟臺灣人比，還是跟香港人比，還是英美殖民國家比？
: 妻：你一定要挑我毛病嗎？
: 天地良心，我沒有任何挑釁的意味

不是挑釁 是跑題了
你老婆只是隨口一句話 一個主觀感受
你就要寫個小論文 搞客觀分析
相信你常常這樣 所以他可能覺得很煩

: 真的只想知道這個「好」，是什麼等級的「好」
: 不然我不知道怎麼接話
你老婆應該回你：這個好是哪種等級 這很重要嗎？
就像你回你老婆：老公覺得同事機車為什麼重要？  一樣

: 4.我要
: 妻：剛剛去看的房子，要不要考慮買下來
: 夫：好
: 妻：就這樣，平常不是意見一堆？
: 夫：就好呀

他是在徵求你的意見 跟你討論 你只看“要不要” 就回答“好”
代表你解讀成是非問句 你以為只要給個單一答案就好
至於理由 不重要 但其實他這麼講 是想要跟你討論
因為一般人不會在這麼大的買賣上 只講“好” 或“不好”
多少 會把自己的感想說出來 權衡利弊得失

: 妻：這麼重要的事，你有沒有關心這個家

你老婆不懂你太用字面上來解讀句子這件事
所以過度衍伸 以爲你是故意找架吵
她要是知道你沒有惡意 或是本性如此 也許能學會包容
: 只要討論就是吵架收尾，幾乎沒有例外，所以家裡大事從不討論
: 我知道有些東西，老婆決定了就不會改
: 因為怕吵架，所以這種事我不會說多半個字
: 太太也是
: 但彼此內心是會怨懟及難受的，而且一直累積
: 直到現在只有「告知」，沒有「交談」
: 我想改，我願意改
: 但老婆太瞭解我
: 她要的不是表面配合或敷衍
: 她需要認同或同理，需要我是真的在同一個頻道
: 我不知道該怎麼辦
