身為一位男性我很能體會你的感受，然而整篇文章看下來
文內的潛台詞卻給我一種「因為你是我老婆所以妳聽我的」的感受
我不知道你自己有沒有發現自己有一些控制跟否定還有貶低你老婆的習慣
標題雖然講說你不知道該怎麼跟老婆溝通了
但文內所說的議題表現出來的卻是「我不知道該怎麼樣能讓老婆聽我的」
下面提供一些想法看看能不能幫助到你自己是如何思考你們的夫妻關係
: 是否同意記者抄文：NO
: 我的年紀比老婆大8歲 我37 老婆29  結婚兩年 我金牛座 老婆摩羯座
: 目前育有大寶 9個月  肚中還有二寶  婚前我有買房  頭期房貸都是我支付
: 生活費考量到我的收入比較高 她能拿多少就拿多少 剩下的我希望她可以幫小孩存起來
: 老婆個性算是比較強勢 加上我年紀比較大 所以我盡可能會多讓著他一點
舒服的伴侶關係多半是合作而非競爭關係，你很習慣讓自己多「讓」她一點
在這邊其實你想說的是你是對的，你老婆是錯的。因為你老婆錯了而你才需要讓
不知道你有沒有試著先不帶著自己的主觀意識去聆聽過你老婆的想法過
這往往良好對話的溝通前提
: 我的個性比較屬於目標導向 遇到問題 先找解決辦法
: 也會先設想在過去經驗上 可能遇到的問題 盡可能先做好準備
你可以問問你自己遇到了問題沒有辦法解決會怎麼樣嗎?
你有沒有辦法包容人生總有點小意外跟插曲的心態呢?還是一定要追求完美面面俱到?
: 但老婆比較像是遇到問題再說 大錯不犯  但是總是會有小意外
: 懷孕的過程 基本家事我盡可能全cover 但整理收納真的不擅長 還是交給老婆
: 老婆懷孕時也剛換工作  考量他可能要花比較多時間適應新工作
: 因此補給品也是我上網去查詢該吃甚麼 直接買  還有孕婦的內衣褲 牛仔褲 套裝
: 大樹 孕哺兒 媽媽餵 奇妮孕哺 亞妮活力 媽咪愛 我都有會員
: 還會帶他去換媽媽禮 跟他借媽媽手冊拍照 我去申請各家奶粉試喝
: 孕中期後每天一包滴雞精  後期再追加卵磷脂幫助哺乳
我知道你說這些是想表達你在這段關係中你很努力了
但是這些跟你想問的該如何溝通沒有很直接的關聯
而是下意識地想告訴我們說你夠好了，溝通不良不是你的問題
但是一段成熟的關係雙方都有責任，也並不是誰對誰錯的問題
: 因為我們都是新手爸媽  能預先做的都盡量準備好
: 一直到出月中後  我的心思的確放比較多在小孩身上
: 我買的是推車 安全座椅 澡盆 各式包屁衣 因為不確定尺寸從60-90都先買了
: 老婆買的是小孩的髮夾(小孩有剃胎毛) 可愛的棉質外套(當時是夏天)
: 當初買新家布置家具的時候 跟老婆討論意見 她比較在意她的花瓶要放哪
: 花錢請人家來大掃除的時候 問她重點要清潔哪裡  她說電視機後面的灰塵
這一段你是只想點出的是老婆跟你不一樣的地方，還是要來聽大家說都是你老婆的錯?

: 出月中在家顧小孩的時候 開始有比較多的爭執
: 例如幫小孩洗澡 我一開始堅持要兩個人一起幫小孩洗澡
: 她覺得我不信任她 覺得浪費時間  我說服她 因為小孩脖子很軟 最好有一個人扶著幫忙
: 她開始跟我爭吵 說她覺得被冷落 不受重視 我只先顧小孩 沒有照顧到她的情緒
: 也沒有時間去照顧貓 只給她飼料吃 沒有給她吃罐罐
: 好像我眼裡只有小孩一樣  說實話 我不否認 因為小孩還無法表達她的需求
: 加上新手爸媽的緊張  當時的心情真的是半夜都會起床看小孩還有沒有呼吸
: 剛開始我不知道怎麼反駁 所以就不講話 她就更生氣了 我的心情當時真的是谷底
: 甚至想乾脆甚麼都不要做好了  但想到小孩又不放心  後來講開了向她道歉
: 我也承諾我會多照顧到她的情緒一點 我也表明了我的壓力(工作 經濟 家庭)
: 因為小孩跟我們同房 而且嬰兒床是在我這側 晚上基本上都是我起來泡奶
: 餵完奶如果媽媽有醒 就給媽媽拍嗝 我去洗完奶瓶再接手拍嗝哄睡
: 後來長了帶狀泡疹 才跟小孩隔離一陣子
: 漸漸地小孩也能睡過夜了 精神狀況才好轉許多
: 我也不斷跟他說 我們的人生前半段都停留在產房了 後半段的責任與壓力不同
: 但女生的犧牲真的還是大多了  因此各方面我都盡可能滿足她
: 那天是我們結婚2周年
: 下午的時候我問她晚上想吃甚麼
: 她說她想吃 酸辣湯跟水餃  (我以為要叫外送 因為五天前她生日的時候也是叫外送)
: 直到下班的時候 她才丟了一家餐廳 說可不可以去這裡
: 我直覺就是小孩的奶粉都沒有準備 萬一歡起來 大家都不用吃了
: https://iiil.io/qg5m
: 她的想法就是吃餐廳的東西就好了  (四川吳抄手 很難想像有適合9M寶寶的食物)
: 我表達我的concern之後  她就說好 回家
: 但是下班去接她之後 (因為公司很近 懷孕之後就是我接送她上下班)
: 一上車就臭臉給我看  也不說話  我也不說話
: 我其實蠻不高興的  因為她只想到她想吃甚麼 但是小孩不是你能控制的 萬一歡起來
: 還不是丟給我  如果真的重視 想要有儀式感 可以提前準備小孩的東西
: 類似的事情發生不少次 很多時候她做決定的時候 是先想到她要的  再想其他人
: 我不知道這是不是因為年紀差距造成的溝通問題 或是個性差異  真的不知道怎麼辦了

而不是一昧地叫對方聽你的
