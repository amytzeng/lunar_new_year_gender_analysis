: 我昨晚提離婚了。
: 　
: 我說他是個盡責的家長，
: 但我們的愛情已經消逝殆盡了。
: 我舉出他不讓我牽手、
: 不送我禮物、
: 呼吸已經失去甜味等等證據。
: 　
: 問他要平日帶大的、假日帶小的，
: 還是相反。
: 　
: 我希望可以在八月初前解決。
: 想不到他先是叫我滾，
: 叫我滾出這個家，
: 冷笑說：「你想怎樣就怎樣？」
: 我說：「沒有，我讓你先選擇平日要帶哪一個。」
: 　
: 並且我很冷靜的跟他說：
: 「我現在走，事情都沒有處理怎麼辦？」
: 他說：「我不想聽你說這些。」
: 　
: 於是我跟他說了離婚的好處。
: 首先他的親子關係不會受到影響。
: 伴侶方面卻可以有無限可能，
: 以世界人口來講，
: 他可能可以跟另外一億人結婚。
: 人生只有一次，
: 不要未來五十年都沒有愛情，
: 活在一個名存實亡的婚姻裡。
: 所謂愛情會昇華成親情，
: 只是別人自欺欺人的說法。
: 　
: 且他就像是一個偷東西的慣竊，
: 卻死守著手上的袋子不讓我檢查。
: 而他的手機就像是薛丁格的貓，
: 我對他是否外遇只能半信半疑。
: 婚前他讓我看手機，
: 卻在我發現曖昧訊息後，
: 把手機鎖上密碼。
: 但我不想懷疑他，
: 只要離婚就不會有這個困擾。
: 　
: 然後他還是堅決不要，
: 他叫我去訴請離婚，
: 我說明明我們兩個可以解決，
: 為什麼要訴求離婚呢？
: 後來他又硬要抱我，
: 說如果我喜歡那些儀式感還有禮物，
: 他可以改變，
: 他也絕不會再跟別人搞曖昧。
: 他只是太把重心放在孩子身上，
: 對不起，忽略了我的感受。
: 　
: 後來我們算是暫時和好了，
: 謝謝熱心的大家建議。
: 希望未來可以一直維持恩愛的狀態。
: 但是他還是不讓我知道手機密碼，
: 這點大家怎麼看呢？
