: 狀態：半年內會結婚
: 很多人說結婚不是兩個人的事是兩家的事
: 以下的狀況實在不知道自己該怎麼想

結婚本來就是兩個家庭的結合
因為結婚後人不可能跟自己的原生家庭切斷連結
過度強調自己組成了一個新的小家庭
婚姻的經營都是小家庭為重，也期待對方這樣經營
那很多不平衡就會產生
未來結婚後
也不要凡事與對方原生家人遇到矛盾，都請你另一半去溝通、解決
與人的不愉快應該著重在事件本身而不是人
自己的不適要嘛自己調適、要嘛自己解決
因為你們彼此都是對方原生家庭的新成員
不要自己弱化自己的位階

: 男友目前在國外工作
: 幾年前在台灣買了一間房子
: 因為他長期在國外
: 房子就給已經結婚，並且有兩個小孩的妹妹住
: 妹妹後來離婚 就是變妹妹跟兩個小孩住
: 這點我也不知道該不該介意

不要問『該不該』介意，事實上你就是已經介意
沒有對錯的事物，你覺得介意合理就介意
不需要拿他人的意見來給自己背書
我說我不介意也沒有辦法改變你會介意的想法
所以你應該去思考你介意的理由
並且好好的去思考如何表達並說服你未婚夫

像我就覺得不應該介意啊
他要怎麼花錢那是他的自由
你們談好未來的小家庭
彼此要投入多少錢來經營
剩下的『自由處分金』，就不要過問與干涉
用什麼會『因此壓縮到小家庭的經濟』來當你不平衡的理由
就代表對方的自由處分金你沒有給予尊重
你沒有真心接受這份錢的名目與去向
若你不好好想想如何包裝與修飾
那介意的理由可能會如同我上述所說
你的介意就會師出無名

: 男友是說我們以後如果住台灣可以另外找地方住
: 或是請他妹妹搬離開家
: 我不太知道他照顧妹妹會到什麼程度
: 會不會影響我們之後的婚姻
: 另外因為男友說他小時候阿姨對他很好
: 他之後要帶爸媽出國玩
: 也找了兩個阿姨一起
: 應該也是全程招待

建議如上述，對於金錢的分配你要先調整
先討論未來你們在工作上能賺到多少錢
賺到的錢要如何使用(買房、買車、養育小孩…)
若彼此有能力且願意投入錢夠用來經營小家庭
剩下的錢就不要再去過問與干涉
但如果談不攏就分手吧
本來道不同就不相為謀

不要去思考為什麼他為什麼可以用這樣的程度去照顧他的大家庭
那就是用自己的成長經驗質疑別人的交往模式
這不應該是你覺得不平衡的理由
因為對家人友善、用自己的能力去幫助家人
這是你男友行為的本質，你男友在本質上沒有做錯事
你覺得不合理的點是什麼？

: 未來結婚對方是希望我能辭掉台灣的工作到國外生活
: 但我自己是希望能有工作
: 不然他的收入要照顧全家
: 或許還要照顧妹妹照顧他家人照顧到他阿姨
: 我真的不知道該怎麼拿捏？
: 用自己賺的錢生活養小孩比較實在

有自己的工作很重要，尤其在家事產值不被重視的大環境下
有自己的經濟能力，才能保有一定的權力與保障
工作還有勞基法保障，感情經營風險相對高
仿間不乏有什麼離婚後贍養費要給不給的
又或是質疑你生活費為什麼要開始不夠用的
婚後經濟最好是各自管理
你們都只是在這份關係中
在經濟上投入你們能夠承受的範圍
去期待你們應該得到的經營成果而已

: 我自己的父母都退休
: 父母都能打理好自己生活
: 男友爸媽收入都不高
: 他會想幫忙規劃他們的退休生活
: 我家這邊的經濟狀況比較好
: 他們家那邊感覺比較需要支持
: 不知道如果是只有一方有收入的狀況下
: 這樣的家庭支出到底要怎麼分配

你都感覺他們家需要支持了
那你為何又要對你男友的『支持』行為產生疑慮呢？
所以我前面已經強調
你不先討論如何表達這份疑慮的合理性
那就會師出無名
而你自己在出發點上都有點站不住腳了
你要麼去說服別人按照你的想法走？

不要想做什麼又怕別人說
你覺得是對的事情就去做

另外這世界很大
