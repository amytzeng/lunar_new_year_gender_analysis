看到這篇，以為是我認識的人發的 XD

我也遇到過原PO說的問題，不過是發生在我親姐姐的身上。我姐想創業的原因跟原PO說的
差不多，但她的狀況更加嚴重就是了，她討厭上班又厭世到極點，已經去身心科拿好幾年
的藥了。

我姐大概五六年前開始，就愛上手作甜點、咖啡這些東西。還會自己去材料行批貨，買一
堆原物料回來自己做甜點跟咖啡。家裡因此塞了兩個大烤箱甚至還花了20萬買一台咖啡機
。

然後她還買了超級超級超級多的原物料，什麼砂糖 / 黑糖 / 楓糖 / 海藻糖 / 蜂蜜 /
玉米糖漿、奶油乳酪 / 切達起司 / 帕瑪森起司、 鮮奶油 / 素食奶油 / 煉乳 / 優格 /
 酸奶、各式咖啡豆（印象中有的大概就是阿拉比卡咖啡豆，衣索比亞 / 哥倫比亞 / 哥
斯大黎加 / 蘇門答臘這些），一拖拉庫的食材塞得滿滿滿滿滿滿滿，整個廚房大概有一
半都被她的各項食材塞爆塞滿，要不是現在已經全部丟掉了，我真的很想拍照發上來讓妳
看看她究竟買了多少食材。

而她手作的東西，我認真說，絕對好吃，而且是超級好吃的那種。她做的檸檬塔、閃電泡
芙、可麗露都是一絕，我在外面吃過的甜點店少說也有一百多家，真心認為沒有任何店家
做的檸檬塔比她做的好吃。

之所以會回覆這篇文章，是因為我姐當初也有猶豫是否要自行創業，那時候我們身邊的人
都是大力支持，家人也給予很多幫助跟祝福，原因很簡單，因為我們都覺得她做的東西真
的很好吃XD

她一開始的創業模式，基本上就是從身邊認識的親朋好友開始。包括我跟爸媽在內的人，
都願意花錢買她做的甜點，而我同事吃過她做的甜點後，還會跟我下單團購。另外，她也
有自己花錢架設網站跟粉絲專頁，還花不少錢去推廣網頁可見度、去各大市集擺攤......
等等。

根據她自己的估計，她雖然沒有花錢搞店面出來，但光是食材、各項原物料、機台，乃至
於架設網站跟粉絲專頁，以及定期去各大市集擺攤的費用，零零總總的加起來，半年大概
燒了四十多萬跑不掉。

看了一下版友的回覆，大部分的版友都認為原PO低估成本費用，而原PO則是不解為什麼成
本要抓這麼高。

原因很簡單：不管妳要賣的東西是什麼，「備料」就是一個非常巨大的成本。

舉例來說，妳去一間餐廳吃飯，點了一碗牛肉麵跟酸辣湯好了。店家有可能在妳點餐完畢
、付錢完之後才打電話叫牛肉跟麵條到店裡嗎？店家有可能在妳付錢完後才打電話叫木耳
豆腐紅蘿蔔這些食材嗎？店家會在妳付錢完後才下單去買調味料跟餐具嗎？以上的問題，
想必答案都是否定的，對吧？！

那妳怎麼會沒計算到備料的錢呢？

假設今天妳真的開店了，也假設真的有客人上門，而且一上門就下單買了30杯的甜品，妳
有可能在客人下單後才去叫貨叫料嗎？如果沒有備貨備料，客人真的上門了，結果妳原物
料不夠做不出來，那憑什麼跟人家開店呢？

準備原物料、備料，都是不小的成本。因為吃的東西，很容易因為各種因素就過期腐敗不
能吃了。如果今天妳備了一百杯甜品的料，但過了一個禮拜都沒賣到一百杯的話，那妳這
剩下的原物料要怎麼處理？

直接丟掉就代表妳浪費很多原物料的錢，但如果留著繼續用，客人吃出問題怎麼辦？輕則
吃壞肚子腹瀉不止，重則住院都有可能，最近的食安風暴，一堆過期食材造成客人集體控
訴店家的新聞妳有注意到嗎？很多食材都有保存期限，而且大批大批叫貨叫料更容易耗損
，因為大批的原物料保存更加困難。

妳買的冰箱預算抓在一萬元以內，這邊隨便抓一個萬元冰箱給妳看：

https://i.imgur.com/btpoO3j.jpeg

妳覺得這種大小的冰箱能夠放多少食材？又能夠放多久？

如果這種大小的冰箱真的就夠用的話，代表妳的營業額非常非常低，連去飲料店打工都比
較划算就是了。


我姐當初創業時，是自己手作咖啡甜點已經好幾年了，身邊每個人吃過都讚不絕口，連我
同事都願意掏腰包出來團購。

但甜點這種東西不會每天吃，一般人一週吃一次已經很多了，所以她還大力搞網路行銷，
去市集擺攤，花錢推廣粉絲專頁跟網頁能見度。這些是很多小型創業的人會用的方式，妳
會使用這些方法推廣嗎？會的話，那妳的成本又會往上增加非常多，不會的話，妳的客源
要從哪裡來？

妳做的食品，客人可以多久回購一次？每天都吃有可能嗎？還是一週吃個一兩天？妳做的
東西現在有多少人願意花錢購買？這些問題妳有想過嗎？

妳有去看過家裡附近的飲料店，一天大概有多少客人嗎？

看的方式很簡單，不用站一整天，站在店外面一個小時就好了，在用餐時段去家裡附近的
飲料店外面站一個小時，有人走到櫃檯就按一下碼表，看看一個小時可以有多少客人吧。


如果妳連這點都不願意做，覺得去外面站一小時很累很奇怪的話，那我真心建議妳不要開
了，連這點心力都無法付出很難期望創業會多成功就是了。

如果願意花個幾天的時間，每天都站在飲料店外面一小時，大概推估他們一天有多少客源
，推估會出去外送幾次，可以讓妳大概知道相關產品能有多少機會。


我姐當初各種功課都做足、做出來的食物我們身邊的人都肯花錢購買，連我同事都團購好
幾次、網站也架了，市集也擺攤了、網路行銷都做了，但最後還是撐不到一年就收攤了。

如果妳做出來的成品，身邊的人已經願意花錢購買的話，代表妳的成品有一定的水準，但
那只是創業的第一步，連要回本都還很艱困。而其他各項雜支開銷，絕對不是妳想像中的
那麼簡單。


但我還是鼓勵妳創業啦，畢竟別人怎麼說妳都不斷反駁，看起來妳已經有了定見，對吧？
！

既然妳都決定要做了，就勇敢的去吧！搞不好老天眷顧妳，妳就是那萬中選一的人，誤打
誤撞什麼都沒準備就成功了呢！未來的事又有誰能夠預料到呢。

祝福妳了。
