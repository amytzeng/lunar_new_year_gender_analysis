這種就真的無聊， 說真的不都跟風網路，都是學的阿。

: 還有一個例子讓我真的非常不高興：
: 我跟我老公住在婆家買在外縣市的房子，我們有給房租，住五六年以上了，這縣市也是妯
: 娌的娘家，妯娌與她老公目前住婆家
: 住進來前婆婆說二樓兩個房間都可以給我們使用，上禮拜婆婆突然說二樓前面大間的清出
: 來讓給妯娌他們住

妯娌結婚了，婚後不想跟婆婆住。
婆家有一房本來租給妳， 現在要多分租一部分出去。
這沒有不公平阿..

有疑慮的是房租， 她們有沒有繳?  妳有沒有因為減少空間而降房租?
而且如不喜歡，妳也可以不要租婆家房。


: 我想說奇怪怎麼突然要房間，因為妯娌回這縣市也都是去住娘家

這妳管太寬了，
就算人家要了間房只是偶爾想去度假用、或甚至當倉庫又如何?

: 上週日妯娌突然私訊我說他們十分鐘後會過來，我想說老公兄弟過來也很正常
: 沒想到是妯娌帶著她的爸爸媽媽過來，我也是盡量招呼

招呼打關係是不錯。
但其實從婆婆跟妳說一間房給她們住那時起，
妳就不是這房子的唯一主人了。

所以這招呼不應該是 主人招呼客人
而是室友招呼室友的家人。

釐清這點至關重要
因為這牽涉到敏感的上下、主次關係的感受。

: 但在我中間去上廁所時聽到他們自己往樓上去了，我一上去他們正在我房間內參觀
: 我覺得很不舒服，裡面有很多我自己的隱私，完全不想給外人參觀
: 我公婆回來也都不曾上來二樓，更何況還是妯娌與她的爸媽

這就妯娌白目， 擅自帶自己爸媽參觀室友(妳) 的房間。

: 然後妯娌就指著大間房間說「以後這間是給我們的，他們睡後面那間小的」
: 我真的覺得非常不舒服，好像她想要在這房子也有一席之地，但我也覺得這是婆婆房子，
: 妯娌他們是也有權利住

因為自己房間被人任意參觀而不舒服，那很正常。

而妳都知道婆婆房子妯娌也有權利住， 為了這個部分不舒服的話
某方面來說妳自己也陷入了 "一席之地、或是說爭地盤" 的情境中。

: 但還是想到就覺得很不高興，好像她剛結婚，我有什麼他就什麼都該也有一份的意思

難道不應該嗎?
或是說妳有的她也有，不公平嗎?

: 該怎麼遠離她或是讓她不要再對我這麼有惡意？
: 我對她一直都很客氣不會講他的不是，真不知道哪裡惹到她了

是不是惡意很難說
妳自己都會因為糾結什麼  妳有的她也要有一份 而不爽。


: 也都已經盡量不跟她見面講話了，但不小心得見面時他又開始惡意滿滿，再說她只是講話
: 直
: 想到她就覺得非常憂鬱好不想回婆家

"我有什麼，她就該什麼都有一份的樣子"
不知道妳這想法有沒有 "不經意"的表現出來過

再次強調，這樣的想法就有主次的負面感受。
換個角度對妯娌來說， 她可能也覺得妳惡意滿滿
她不過是一樣的要在婆家房住， 就被妳這樣說。

講不好聽的
妳在這邊揣測她
搞不好對她來說就是不喜歡妳字裡行間透露出的那種 主次感受
從而對妳不滿阿..
