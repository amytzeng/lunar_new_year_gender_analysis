講到明星學校的8+9 我就有經驗
當年讀建中高二剛開學
新上任的衛生股長叫某個新同學掃地
起一點口角
結果放學後衛生股長就被7個新同學+烙來的
圍起來每人揍一拳 還要掏錢請七位大哥各一包菸才了事
結果過兩天那個"新同學"就轉走了
不講新同學 一些高一開始就同班的
我到高三開始留校讀書才知道
班上至少有5個同學有抽菸的習慣
還有的人沒有英文字典就跑去學弟教室抽屜"借"一本
翻完就直接丟垃圾桶
有個大一屆學長也講到他們班上有個人很會順手牽羊
同學需要什麼飲料文具都請他去福利社"拿一下"
還有同學是田僑小孩 女朋友一堆 高三就百人斬
像我這種普通單純家庭出身的
實在很難想像建中是這樣的學校

不要講建中
我現在工作一堆同事MIT畢業的
講到他們學校宿舍的怪人也是一籮筐
所以說其實真實社會就是這樣
與其把小孩從小放在溫室
不如把握時間多跟孩子相處 家長的身教還是影響最大的
我高中那些同學家裡很多也是8+9家庭
只是小孩比較聰明可以考上建中這樣

: 我的孩子現在已經在念研究所了。走完全套的
: 升學歷程，應該可以貢獻點意見。
: 我的看法是就近比明星重要。
: 理由一是家長畢竟是最重要的教育資源。所以
: 為了通勤而減少家長與孩子相處的時間，並不值得
: 家長與學校、老師、其他家長之間的互動也有影響。
: 跨區的家長跟學校之間的互動必然很少。
: 理由二是孩子與祖父母外祖父母間的情誼很珍貴。
: 我女兒今年23歲，這期間送走了四老中的二位，
: 另二位得了癌症，健康狀況也不佳。但是孩子跟
: 長輩互動一直親密，存歿都沒有遺憾了
: 理由三是現代家庭生育數少，孩子普遍缺乏同齡同儕。
: 跨區就讀跟同學之間互動增添許多障礙。

: 的孩子。如果在自己社區裡，家長還能出點力，跨區的話
: 就鞭長莫及了
