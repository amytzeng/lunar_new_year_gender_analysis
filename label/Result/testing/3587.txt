: 的熱情，也就消失了)

: 開，不開電視，她會翻臉)

: 太本來有約定不開電視，我就好意放開了一個小門，怎知她一看電視之後，她就忘了我們
: 之間有不開電視的約定，她現在否認了，當沒這約定，她還不想關掉，她連我們以前都戴
: 耳機，她也當不存在，她不願意再戴耳機了，就是一定要聽電視聲才行，但這會吵到我的
: )

: ，其他時間她在玩電腦的小遊戲(這小遊戲要過關的，要專注的，有時間性的，她在專心
: 玩小遊戲時，我不能跟她說話的，她會大聲噓我)，或者她在鑽研植物或者聽老師的上課
: 錄音，反正她就是要電視開著，她能一心兩用、一心三用，我真的沒辦法，我會被電視聲
: 干擾(我跟我太太的距離就一公尺，我一定會聽到電視聲)

: 於是這次經驗之後(就是，我好話說關掉電視，讓我專心寫作，她也不願意了，她也不尊
: 重我了)，我真的是醒了，我就對我太太下禁令，大家都不要開電視，我想回復到以前(其
: 實我有時候，也會想看一下電視新聞談話節目之類，可是我若把電視打開，這結果我想關

: 轟炸，所以乾脆我也不看了，其實我太太能用電腦看的，她變成就不願意戴耳機，而且要
: 18小時開滿電視)，我沒辦法了，我甚至嗆白，要這樣18小時看電視，我不要將來20年都
: 被電視吵，那就離婚吧，看她選電視還是選我，她可以嫁給電視？她不跟我說話了，直接
