你或許應該試著"跟他學怎麼煮"

不是像個行政主廚插著手站在後面看
是從切菜備料熱鍋下料調味, 從頭到尾跟著他一起做
"味道不合"不是一句話就知道差異在哪裡的
你不動手做就永遠不會知道為什麼/在哪裡會有差異

再來自己煮的自己比較會有吃的意願, 就算煮出來的是噴也是自己煮的噴,
接受度會提高超級多

然後兩個人一起煮你對調味這件事也就有插手的餘地
起鍋前試味道假如你也辦不到調出你要的味道那也沒話說了回到上一段

一陣子之後你就會發現你們兩個人對於食物調味就會趨近一致了

※ 引述《SCEW (↑X↓B L Y R A)》之銘言：
: 是否同意記者抄文：NOYES/NO
: 請問各位大大，如果老婆煮菜不好吃，或者是不合你的口味時，你們會怎麼做?
: 1.硬著頭皮吃，不管味道如何都說好吃or還可以
: 2.跟她說不太好吃，看有無改善方式
: 3.盡量不吃，自己買外面
: 以上三種方式其實就是我自己的經歷
: 我跟太太結婚9年，有兩個小孩8歲和6歲，太太婚前是完全不會煮菜
: 婚後她就開始嘗試煮菜，自己上網看教學或看食譜
: 一開始我很開心，覺得自己小家庭可以每天吃老婆煮的菜是一件很幸福的事
: 但事與願違，可能是我自己的問題
: 也不是完全，但大多時她煮的菜我都覺得不太好吃，甚至難以下嚥.....
: 開始的一、兩年我都用方法1
: 或暗示她這道菜好像少了什麼味道之類的 提示她能不能改善煮的好吃一點
: 但長期下來還是沒解決，我的提示就越來越直白
: 有時候會直接說這道菜不好吃，但還是會吃下去就是了
: 因為小孩的關係，太太覺得自己煮比較健康，這也是事實
: 所以週一到週五她也都會煮，只有六日放假會回娘家婆家，才沒煮
: 後面這幾年我就是用方法3了
: 能買外食就吃外食，太太就煮她跟兩個小孩的
: 但有時她會跟我小抱怨，說我總是不喜歡吃她煮的菜，但小孩都覺得好吃
: 其實我也有點難過，因為吃飯應該是一件開心的事
: 但吃到不好吃的食物時，心情一定會變差.....
: 真的很為難，請問我這樣做是不是錯了?
