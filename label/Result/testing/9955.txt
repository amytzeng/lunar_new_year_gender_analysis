※ 引述《SeedDgas (A)》之銘言：
: 大家好，關於板上這幾天的熱烈討論相信大家心裡都有些想法，
: 那系列文該怎麼處理，怎樣都會有人不滿意(當然也會有人滿意)，
: 而現行板規講真的也沒太大毛病，唯一的問題大概就是板主的裁量權過於曖昧。
: 所以比照之前對是否開放自刪的做法，
: 麻煩大家花一個禮拜的時間好好討論，
: 之後再花一個禮拜的時間投票來解決這個問題，把系列文的處理方式明確下來。
: 以前約定成俗的做法是超過20篇就公告禁止回應，
: 現在我希望把這東西寫進板規，看是乾脆就別禁，或者第Ｎ篇之後不能回，
: 總之就是這樣，麻煩各位板友各自發表看法了，感謝。
: 然後先來說說我的看法吧，
: 置底聊天區那邊我稍微提過了這邊講得仔細點。
: 我個人是傾向不鎖系列文的，因為本板是個聊天討論取向的看板，
: 只要有人想聊也沒偏離婚姻板的內容太多，那沒什麼理由不讓它們聊，
: 我不太喜歡靠著限制別人不能聊其他話題的方式來讓大家專注在某些話題，
: 怕系列文淹過自己想看的東西？那就回文啊，就這麼簡單。
: 當然這種作法可能會出現討論到後來論點重複或者廢文過多的情況，
: 但在我看來文章廢不廢，不是作為板主的我有資格判斷的，
: 純粹當個板友可以想噴廢文就噴廢文沒問題，這都是個人看法，
: 可是一但上升到板務層面就不太一樣了。
: 我不太認同一篇文章沒有違反板規的情況下，只因為我覺得它很廢所以就不該存在，
: 搞不好發文的人也是費盡心思啊，只是礙於各種因素所以才不盡人意，
: 作為一篇文章好壞應該是可受公評之事，反正這很主觀，
: 但只是因為我覺得寫的很爛就不該在板上出現嗎？我覺得我身為板主不能這麼做。
: 因此目前我傾向系列文都不禁，有違規的再用其他板規處理，
: 畢竟所謂的言論自由就是要忍受一些你不喜歡的言論，
: 不過這終歸只是個人看法啦，歡迎大家踴躍討論唷。

  就我的印象, S板主大大是傾向不禁系列文的, 過去也有討論過是否要開放,

  但我相信以過去那個時間點的風氣及狀況, 還真的無力去改革。


  大家也提出了, 其實板規的問題不太大, 但執行的人標準不一,

  導至過去不少因人的不同,而判決不同的狀況(非指現在板主)

  最大的問題在於板主的自由心證,

  而解決的方式也很簡單, 就是板主在板規上加上自己的定義

  什麼叫婚姻點? 什麼叫明確建議?

  講清楚說明白, 以後就靠這說明來認定啊~

  老實講啦, 在這一個系列文裏, 重點王的回文有符合現在板主們的認定?

  如果今天換了一個人, 那篇文老早就被刪掉了,

  那篇不刪,以後的人相繼模仿, 你刪不刪?

  過去也有同樣的情況發生啦, 老屁股的就不刪, 不熟的就全部違反板規!

  以前也只有marra板主在認真和板友們(包括我)在處理這些議題,

  可惜marra板主陣亡了, 雖然有爭吵, 但她是我唯一尊重的板主。


  回到系列文,說真的不用以20篇, 這種量化的數字去做標準,

  大家可以去回顧一下系列文啦,20篇內有多少有用的資訊? 有多少是來嘲諷的?

  有多少是來備份的?有多少是回文的人在那邊吵架?這些都佔掉份額。

  頂多在20篇後,開啟觀察機制,回文有太多無意義的爭吵(回文者之間,和原發文者無關)

  就開始公告在幾點後鎖文禁回。

  另外文章刪不刪, 我覺得就交給人去檢舉啦, 有檢舉在去處理,沒人檢舉就不要理


  以上,選二次板主的婚姻板觀察家  留
