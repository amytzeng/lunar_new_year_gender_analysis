關於夫妻財產制，給個連結妳自己看
https://law.medpartner.club/divorce-payments/
這個講得很清楚

至於妳們之間，我猜應該是沒特別約定的法定財產制
就用法定財產制看一下妳實際上的狀況會不會有問題
※ 引述《archer67 (羊入虎口了)》之銘言：
: 先說我目前沒有婚姻危機，
: 只是有點疑問，想請教ㄧ下大家。
: 我是沒在賺錢的家管，
: 從結婚後就沒有工作，
: 都是老公在賺錢。
: 婚後1，2年，我覺得這樣不太好，
: 想說多少賺一點，順便維持專業技能。
: 自己也存點錢，如果萬一婚姻破裂還有能力應付生活。
: 但我老公不讚成，
: 他說以後如果真的離婚了，
: 每個人可以分到的錢是：結婚後雙方賺的資產ㄧ人ㄧ半。
: 說這是法律規定的。
嗯！基本上算對
實際上是婚後的資產扣掉雙方的債務（也就是剩餘資產）之後平分
不過這邊的婚後資產不包括增與給個人的跟遺產
所以不管你或他在婚後有欠了一筆錢，這筆錢要用你們的財產還掉後再來分

: 我本來是相信的，
: 因爲房子車子都登記在我名下，
: 我老公說萬一真的離婚了，房子也是ㄧ人ㄧ半，
: 不是登記在我名下就都是我的。
他都登記在妳名下了，如果這樣妳還不相信，真的無計可施

登記在妳名下，代表妳要處份（賣）這些動產不動產時，不用經過他同意就可以處份
賣掉的所得是兩人共有，不是房子掛妳名下，妳賣掉就都是妳的錢
不過呢，換來的畢竟是"現金"嘛！現金這東西很快也很容易就可以"不見"
這就是很多人會說共同登記比較有保障的原因

妳先生很相信妳啊！他就是把妳們共有的財產交給妳保管

: 而且有一次匯ㄧ筆錢到我帳戶，他也說夫妻是視爲ㄧ個人，他匯給我等於匯給自己，所以
: 不用贈與稅。
是不用！不只錢不用，土地也不用
: 而且我用我的名字就可以報我們家的所得稅。
所以，他賺多少給妳看光光 XD

: 但是婚姻版看久了覺得怪怪的，
: 就是如果真的夫妻視爲ㄧ體，
: 那為什麼會有夫妻AA制？
: 或是家用誰出多一點誰出少ㄧ點的問題？
每個人個性不一樣吧
雙方能達成一致就好
: 就算一方存款比較多，到時離婚不是也要拿出來ㄧ人ㄧ半嗎？那現在誰出多一點家用不是
: 意義不大嗎？
不一定啊！可能人家結婚時就去辦分別財產制了
意義不是很重要，重點是在妳家自己目前的分法
兩個人是不是都開不開心，是不是都有安全感

大家都想要開開心心的，大家也都會想要安全感。這個不分性別

: 還是有什麼我不知道的陷井呢？
有！
首先呢，離婚協議書上可以另外議定，不受財產制的限制
不過議定當然要兩個人都同意，所以這也不算什麼陷阱
至於其它的陷阱，不合法的當然有，或者是用金融會計手段把財產洗不見的辦法也不是沒有
多半是在法律的灰色地帶，成本也很高
如果你們不是八九位數以上的有錢人，我想應該是不太需要擔心這個
而且如果需要擔心到這邊，那不如馬上立刻談離婚

看來你先生把妳的安全感當作一回事，而且盡可能主動給妳安全感
遇到這種人，妳沒必要多想吧！
