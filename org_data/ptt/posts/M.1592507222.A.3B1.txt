把無關緊要的全部刪掉

: 我花費在遊戲上的時間大概是
: 某個時間要組隊(約3分~5分鐘)
: 某個時間可能要打王(通常不參加，偶而參加約10~15分鐘)
: 公會戰時間我還是希望可以上線(約10~15分鐘)
: 每個月可能會有1~2次比較長的打王時間(約30分鐘，但是不一定有)
: 其他副本我通常利用通勤或是做家務瑣碎的按一按
: 老公通常8點多到家
: 那時候也是我差不多整理完廚房 想稍微休息的時候
: 我趁小孩在房間寫作業 我會一邊看作業 一邊玩遊戲
: 老公覺得我花遊戲影響到生活
從這段描述就能看出一天玩遊戲的時間絕對不只15分鐘

他覺得影響生活也是事實，因為除了本來的休閒跟零碎時間外

還開始持續侵蝕到其他活動的時間

: 我覺得我都忙了一整天了
: 為什麼連固定的15分鐘都要挑剔我
雖然你強調只有固定的15分鐘

但顯然你一天絕對不是只玩15分鐘

你先生挑剔的也不是因為你固定玩15分鐘

: 因為我花時間玩遊戲
: 老公開始挑剔我做的家事
: 可能是忘了收信、可能是地上的頭髮 、桌子擦得不夠乾淨
: 可能是等小孩睡了才慢慢收廚房
: 我回家像在面對另一個老闆
: 審查著我是不是哪裡少做了 做得不夠好
: 他覺得只要我這些事情做得不像以前那樣好
: 我就是玩遊戲影響了生活!!!
: 我就應該放棄
從旁人的角度來看，就是一個人開始整天就是玩手遊

然後開始很多事情都做不好，而這個人卻覺得別人不准她玩區區15分鐘

問題是你不是只玩區區的15分鐘啊

: 我覺得很難過
: 因為我覺得這不只是遊戲而已
: 它代表我婚姻生活裡面小小的自我空間和時間
: 我的遊戲裡還有我的兄弟姊妹
: 我覺得對於早早結婚之後不能常常跟他們相處的我
: 我們透過遊戲多了很多的相處
: 他期待的是一個完全燃燒努力的老婆和孩子的媽

你有沒有想過他可能是不希望自己老婆整天都在玩手機，玩到事情開始做不好？

: 我聽了覺得很心累
: 自己放棄理想的工作
: 回到家庭  (我還是有上班喔!)
: 每天日復一日做著沒有長進的家務 沒有成就感的工作
: 唯一的休閒興趣 也要被迫放棄

雖然你寫很多很感性的內容，放到這版就會一堆人覺得你好棒應該放自己自由

但在遊戲齡數十年觀察過數百上千人的眼中

你跟剛開始遊戲成癮的人有87%像，包括思考模式


所以我建議你先統計一下你目前一天到底花幾個小時在手遊上

然後想想你現在的抗爭是希望一天能花多少小時在手遊上

但別誤會，玩遊戲時間長本身不見得是問題

問題在於你是在怎樣的時間玩，怎樣的方式玩，是否影響到正事

先面對自己的正確遊玩時間，然後想想你要把手遊當作休息消遣去調整遊玩方式

還是當作你的新社交，新生活目標，侵蝕原本的生活也沒關係


如果是前者，那就去調整遊玩的時間，排到零碎時間或是娛樂時間

如果是後者就當我沒說，就放飛去追求你的新生活，然後學到經驗
