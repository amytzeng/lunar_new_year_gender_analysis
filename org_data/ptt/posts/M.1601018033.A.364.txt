: 講點現實的啦，
: 這篇整理下來大概就是要人常保好奇心，時刻提升自己，
: 前者我好像國中就聽過，類似後者的話更早，國小就在教活到老學到老了。
: 所以道理大家都知道，可是為什麼做不到？

好奇心、提昇自己是方向，要真正做到有兩個關鍵：

1. 願意踏上去嘗試的動力

可惜，多的是光看到道路，就不斷打槍自己的人，
這點不只是伴侶夫妻，我在單身者身上看到更多。

他們連「嘗試」都不想試，或是隨便走了兩步就說「我有試了啊，沒用嘛」
背後的一種心理機制是：「這代表並非我不好，也不是我這人有問題」

如果踏上改變的道路，卻還是關係不順、被打槍，那不就代表我人有問題？
於是乾脆一開始就找各種理由，說這條路行不通，或解釋為何自己不踏上去
他們會大聲呼喊：「不是我不想，是我沒辦法喔」


2. 找到方法

方向有了，但細節的方法是重點
一個人徒步走（還穿拖鞋），另一個人騎著腳踏車，效果自然不同

上面說到，隨便走兩步就喊「沒用沒用沒用沒用」（搭配Dio聲音）
通常就是用錯了方法，加上動力不強，正好有理由退回原點。


: 理論上來說這樣做的結果不是成功就失敗，
: 先講失敗吧，同樣用那三個例子，
: 1.觀察四周：安排美食之旅 => 沒錢沒時間
: 　　　　　　分享口罩趣聞 => 擔憂自己買不到口罩
: 　　　　　　討論買房 => 講身體健康的根本買不起
: 　　　　　　最後還是只能純粹當談資，一個話題有來有回頂多就十分鐘，

你的舉例，恰好是我見過不少男生的思維：「以理性解決事件」「著重現實層面」

我在上篇文章的例子中，明明也會跟對方聊哈士奇、寵物、貓狗
但你卻只聚焦在「沒錢沒時間怎麼去吃美食」

我完全認同「現實要顧」，但在我的工作經驗中
「只顧現實」卻無法提高雙方對婚姻或愛情的滿意度

以這串起頭文章來說，老婆解決育兒任務（現實問題）後，有著與先生互動的需求
那位原PO辛苦搞定工作（現實問題）後，也會擔心自己沒東西可聊，感情可能變淡。

當我們與伴侶聊天時，並非為了「解決某件事」例如吃美食、買房、買口罩
這些叫討論或溝通，是附帶在聊天之中的一小塊，而非主角
（除非你一開始就是為了討論某件事情，要雙方協調或做決定）

聊天的主軸是什麼？
分享生活故事、日常趣事、過去回憶、心情感受、心得想法、未來夢想

是的，都跟「解決問題」沒啥關連，因為聊天本來就不是要解決問題
而是「促進雙方交流」，更瞭解對方的生活與內在。

這有什麼「好處」？
很簡單，就是拉近關係、維繫感情，讓你們不會陷入平淡無趣的婚姻

如果與伴侶的互動，都只是在「解決事情」，
雙方的互動永遠在討論「某事該怎麼處理」、「某事如何解決」
你們其實不太像夫妻，而更像是同事。


: 　　　　　　講太多搞不好還被老婆嫌怎麼都在講些風花雪月的東西，
: 　　　　　　有那時間做夢不如多做點家事，幫忙帶一下小孩讓老娘去放風。

若出現這情況，表示你跟對方聊天時沒有引發「共鳴」
白話：自己講爽的，沒有製造互動

其實像本串原PO的老婆，也有點像這狀況，一直講自己整天的育兒故事
但沒有讓那位原PO能參與對話，單方向講不停自然讓他想放空。


: 2.與人互動：別鬧，長袖善舞是高階的技能，
: 　　　　　　現在多數人累得跟狗一樣，你不是特別會交際的話誰管你去死，
: 　　　　　　一個肥宅過來搭話別說異性啦，同性都敬而遠之好嗎。
: 　　　　　　就算真的聊起來也順利當成談資回家跟老婆分享，又回到上面的情況，

不是要大家成為social咖，整天在那邊哈啦裝熟
而是別讓自己「沒朋友」，或是成為辦公室的「邊緣人」

我上篇也說啦，我在工作時很不愛跟人講話
但休息時偶爾會跟同事搭個幾句，或分享最近有趣的事情

沒有人一開始就跟你很熟，別人也沒有義務跟你互動，
所有的關係都需要建立與經營。

一個肥宅沒人理他，有想過原因嗎？
表情很嚴肅？講話方式很尷尬？沒話題無法接話？

何者比較有可能改變現況呢：

「找到問題，逐漸調整，重複嘗試」

還是抱著「反正我就肥宅啦沒人會理我」心態乾脆不跟人互動？


: 　　　　　　安排旅遊？沒錢沒時間，有空不如幫忙帶小孩讓老娘去放風，
: 　　　　　　一起追劇？男女生喜好差那麼多，到底是想折磨誰。

男女生喜好差很多，無法追劇？
怎麼我看到滿多夫妻，雖然有各自喜好的興趣或看劇類型，但還是會有一兩個交集

但「共同喜好」、「興趣交集」不是莫名其妙繃出來
一個管道是透過聊天發掘，另一管道是雙方一起培養：

我邀請妳來接觸我的興趣、踏入我的生活，
有可能進而也變成她的興趣，那我們就有了可以一起做的事情
有可能她接觸後覺得無法，那也沒關係，我們再找別的試試。


反過來也是，讓對方邀請我接觸她的興趣，或是我看到她的興趣後主動嘗試
然後評估自己喜不喜歡，要不要再接觸。

不論如何，最重要的是「雙方對彼此的興趣，有沒有好奇心」
以及「雙方對世界是否有好奇心，願意接觸新的事物」


以追劇來說，我伴侶之前看愛的破降，我陪看一集後實在沒興趣
所以她在看愛的破降時，我就看自己的漫畫，互相尊重沒啥好折磨的
（結果她重複看了三次，我都開玩笑說是要寫論文嗎？）


後來我看訊號時，她跟著看一集覺得有意思，於是一起把第一季追完
又例如我們原本就都很愛看Friends，偶爾會隨便挑一集來看，然後一起笑到每丁每當


: 3.擴大生活圈：這個分類不是花錢就是花時間，很有可能又花錢又花時間，
: 　　　　　　　一個男人婚後有小孩，禮拜六補眠，禮拜天丟著老婆自己跑去學樂器，
: 　　　　　　　這在板上應該是唯一死刑吧。

方法是活的，學樂器只能「禮拜天丟著老婆出門」？在家自學也是個方法
畢竟剛開始先當興趣，不是一下就要練到專業級

更何況這可以與伴侶協調、討論的部分，
我週日上午出門（學東西、運動、跟朋友聚餐、接業外工作等）
下午回來帶小孩，換妳出門（看妳自己安排什麼事情），
或是妳去按摩放鬆一下，甚至在房間補眠也行，反正這時段我負責小孩。


我需要的時候妳來支援，你需要的時候換我cover，有小孩後更該如此
才不會讓其中一方（或雙方）沒有自己的生活，變成平淡無趣、容易burn out的婚姻


: 所以說回到一開始的問題，
: 這些道理大家都知道，講的也絕對沒錯，
: 但為什麼不是每個人都做得到呢？
: 我是覺得啦，能做到上面那樣的，
: 要嘛你本身就是個有趣的靈魂，要嘛你過著有餘裕的生活，
: 可是有這兩個條件的根本不會有原po那種煩惱，
: 有那種煩惱的硬要去學大概也是畫虎不成反類犬。
: 因為大多數人不是像你說的一樣「我就懶.jpg」

我並非走靈性路線的心理師（雖然我有些朋友是）
與其說有趣的靈魂，我更認為是「願意嘗試新事物的個性」

個性並非由先天百分百決定，而是後天習慣的養成
但我見過不少人，卻是根本不去接觸、不想嘗試，自己打槍自己。

「在入口就喊著不可能、沒有用」其實真的最輕鬆
大家在生活中可以看到太多例子，先甩鍋就贏了，因為不是我的問題


: 這讓我想到一句很傲慢的話，
: 「我成功是因為我努力，你們不成功是因為你們不努力。」

想聽實話嗎？
以我接了不少情侶夫妻的諮商晤談經驗來看：

一段關係的成功，是因為「願意努力」，而未必是「真的很努力」

有時「太努力」反而反效果，如同親職教養提的「good enough monther」
當你展現出「願意努力」的態度、行為時，對關係本身就有了正向幫助
更何況在婚姻裡，你有隊友，可以合作如何一起努力。


另外，想到一句以前老師說過的話：「成功者找方法，失敗者找藉口」
我倒認為應該反過來：「習慣找方法的人容易成功，習慣找藉口的人容易失敗」


: 光是維持它們目前生活可能就占去大部分精力，
: 當然你也可以說沒做到100%的努力就是懶，這樣我也不能說你錯，
: 只是又有多少人可以這樣長時間的燃燒生命呢？
: 我是聽過幾個啦，然後四十多歲就過勞去了，呵呵，唉。

我前文也說啦，要嘛改變「硬條件」現況，簡單說就是收入、工時
這點實在很難速成（除非中樂透、拿到內線、運氣好歐印美德醫之類）

但改變「生活習慣」相對簡單且可行許多，
更重要的是，「只有工作」的人生反而容易燃燒殆盡
「懂得經營生活」的人，則有較好的身心健康。


前文提過，要調整生活習慣、增加話題，就是：
觀察周遭或與人互動，接著記錄故事、轉為話題、分享聊天

另外擴展生活圈也是關鍵，但務必從簡單的、鄰近的、時間短的嘗試：

每天中午都吃一樣東西，今天特地走遠點吃好一點，是在擴展生活
每次休假都在家補眠，今天特地帶家人（或自己找朋友）去近郊走走，也是擴展生活
每次習慣用電商購物，今天特地去實體店面逛逛，同樣擴展生活


不是什麼「歐洲旅行」、「衝浪潛水」、「徒步環島」才叫擴展生活啊朋友


平常都在家簡單吃，偶爾想去外面吃餐廳，
未必要直上教父牛排，可以先吃個台灣牛排教父（孫O寶）

已經太久沒接觸大自然，突然想出去走走，
與其衝玉山被救下來消耗社會資源，不如就近象山先爬一下吧


先從容易達成的開始，小改變就有機會滾雪球形成大改變
但所有的前提都是「請先規劃你的時間，然後踏出第一步試試看」


有颱風怎麼辦？浪太大怎麼辦？船沈了怎麼辦？遇到海盜怎麼辦？撞到克蘇魯怎麼辦？
飄到荒島身邊沒有威爾森怎麼辦？船上食物吃不慣怎麼辦？被海霧艦隊攻擊怎麼辦？

我們可以無止盡找理由說服自己停在岸上，持續過著不滿意的日子
也可以做好初階準備後，播放快樂的出航日，
唱著卡摸美～卡摸美～卡摸美擱飛來，然後把船開出去，看情況修正路線

而且別忘了，你不是一個人在開船，記得向伴侶發出組隊邀請。
