原PO分段方式不好才會攪在一起.
應該這樣切

1.
: : 目前大小聘跟餅錢讓我苦惱
: : 大聘36萬 小聘16萬
: : 我哥結婚時 對方父母是不收大聘也不給嫁妝
: : 然後小聘用來支付餅錢 訂婚的一些支出
: : 大嫂婚後跟我哥在自家工廠上班  整天專心顧小孩

大嫂家不用大聘, 小聘也是花在餅錢和訂婚支出.
大嫂選擇專心顧小孩.

2.
: : 現在輪到我結婚
: : 大聘是要給我老婆 (有跟另一半討論過 由我拿出來)
: : 小聘是嫁女兒的錢要收
: : 另外跟我們要餅錢約10萬

未來老婆要收大聘. 岳家要收小聘, 還要另外10萬餅錢.

3.
: : 婚後想到外頭工作 比較自由獨立
: : 岳母整天在幫我找保母 打算孩子出生給人帶
: : 已經拒絕要自己顧

不知道賺的有沒有保母費多, 還要在外頭工作.
岳母還已經在找保母了.

4.
: : 以後可能回家做自家事業
: : 一定是我爸我哥拉拔我
: : 這樣下來我不知道怎辦
: : 就算我哥不計較 但我會愧疚

哥哥前幾年打拼很辛苦,
我直接回去接手享受成果, 覺得很愧疚.

大概是這樣啦~

-----

聘金要錢
餅錢要錢
住外面要錢
請保母要錢

如果岳家不要拿這個錢 (或等值嫁妝)
如果太太收入夠且願意負擔一半租屋和保母費
就公平拉.

以結果來看,
實際面女方是妥協先在家顧小孩個幾年, 住婆家省錢.
(一人賺錢三人花還是很花錢的, 不然就去工作賺錢拿錢出來也能解)

大概就差岳家想拿錢吧.
這也就是標題寫的  "聘金跟餅錢讓我為難"

--

建議
如果太太賺多, 就別堅持一定要親帶, 想去工作就去.
做自己想做的事, 生活品質會提高很多.
而且, 生下來後很高機率媽媽會主動想親帶一陣子, 也不用原PO要求.
由你講出來反而好像被你逼的一樣.

至於岳家要拿錢, 阿就真的比不上不拿的阿.
這年頭岳家想拿錢而爭吵的也不是只有你, 板上也不少文拉.

承接哥哥的努力,
講白了回家做要有賺頭才會有考慮的必要,
若賺的比自己在外多,又覺得承接哥哥努力而愧疚,
最實在的就是擬一個%數出來回饋.
