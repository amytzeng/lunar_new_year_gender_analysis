你一直說自己帶小孩的方式有理論研究背書

你傳遞給小孩的就是"爸爸是對的 其他人是錯的"
不讓小孩有自己思考的機會 我是覺得很不好

: 對於危險的事，會造成身體損傷的，從一歲多就講實話。
: 這個會手會斷掉、這個腳會斷掉、這樣會死。
: 小孩太矮車子看不見，在停車場不牽手會被撞，小孩被撞很可能會死。

有利用恐懼控制的嫌疑
比起好好解釋為什麼要牽手比較安全
你直接用不牽手會被撞死來講
有種"不聽話會死"之類的意思
像這樣用嚇的來教小孩 效果不一定好

: 馬路上亂跑會被車撞、汽車安全座椅不做好、總是在開車時遇到三寶時也會直接說，什麼
: 樣的交通行為會被車撞。
: 還有稱讚類的實話也是從兩歲三歲就老實說。
: 小班、中班開始就會說這些：
: 不是你很棒，是你做的很好。
: 不是你很聰明，是因為你花很多時間學習。
: 稱讚都會很明確。
: 長輩隨口的「好棒」「好聰明」 只是因為這樣講你會開心跟他們玩，但是你要記得你做
: 得很好是因為你總是花很多時間玩這些教具間學習跟練習，不是你坐在那沒事就好棒好聰
: 明。

這個更扯 爸爸稱讚的才是正確的方式
其他人的"好棒" "好聰明"都只是別有用心的想要小孩陪玩

事實上好棒好聰明只是很普通的稱讚方式 不用太多腦補
一句話直接剝奪小孩從別人那邊得到的成就感


: 保可夢只是一樣的塑膠殼，印不一樣的圖案放不同的數據，然後說幾顆星很厲害很稀有，
: 其實只是為了讓小朋友圖像崇拜，為了賺錢。
: 打來打去的卡通電視也只是打來打去，其實很笨，拍這些也是為了賣玩具賺小孩錢。
: 你同學誰誰誰會這樣是因為他的爸媽沒有花時間陪他看書陪他生活，讓他都在看打打殺殺
: 的電視所以講話和行為都很粗魯，那樣很笨，你不能學他，但是你也不可以這樣跟他說，
: 因為他會很難過。你可以選擇跟比較有受教育的小孩一起玩。

我是不知道你小孩喜不喜歡寶可夢
但是如果你小孩喜歡 你卻用這樣的方式告訴他
其實也只是在說 "你喜歡的東西在爸爸心理沒價值 所以你也不能喜歡"

同學的行徑背後也可能有很多原因
你沒有去花時間理解
反而直接跟你心中那些"沒價值的行為"硬作連結
甚至還拿來當作例子跟小孩說
"你看 他們就是不按照我的方式 所以才笨"
用來強化自己對小孩價值觀的操弄

: 這些其實都很重要，關於正確而有原因的稱讚小孩是有研究的。
: 隨意的稱讚很棒很聰明是有害的。
: 從小教小孩選擇適合的朋友也很重要，不要排擠人家，不要攻擊人家，可是可以跟更有受
: 教育的小孩一起玩。
: 因為再沒幾年，同儕的影響力就會比父母更顯著了，在那之前就要先培養他選擇朋友的能
: 力。
: 教小孩看懂一些行為跟話語背後的意義也很重要，但是可以跟他說不要跟對方說，因為對
: 方可能會很難過。
: 不是刻薄的那種實話，是很現實的實話。
: 讓他學習自己選擇自己要做什麼都有時間成本的實話。

: 我一直都沒對小孩講童言童語，跟孩子好好講話都說實話，而且尊重幼童的想法與感受。

: 小孩沒幾歲就很有想法了。
