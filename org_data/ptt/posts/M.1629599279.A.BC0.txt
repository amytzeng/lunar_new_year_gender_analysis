說實在話
當初我在決定孩子要在哪邊受教育的時候也猶豫很久
台北的娘家學區非常好
學區內的國小國中都是非常搶手的學校
而我的孩子都一出生就掛籍在台北

我們家的大長輩每次碰面都要一而再再而三的強調台北教育資源豐富
當年他也是捨棄工作上台北來就是為了給孩子好一點的教育環境云云
我自己是在台北長大的 很清楚台北真的資源非常集中

如果要讓孩子留在台北
我爸也是北部中部跑 我媽雖然可以幫忙
但她也是一個蠻愛趴趴走和擁有自己生活的人
一個小孩還可以幫忙
3個她應該會崩潰 所以我還是要自己想辦法

孩子的爸爸工作在中部
工作性質的關係 客戶廠商都集中在中部
搬到北部CP值很低

我自己是在家人的公司上班
時間雖然彈性 但剛創業收入不穩定

我的考量是我不希望小家庭再這樣分隔兩地
我的雙胞胎開始會跑會吵精力充沛
公婆體力上已經無法負荷
教養上也會跟阿公阿媽盧
身為媽媽我必須開始把孩子帶在身邊鎮壓

至於是否在北部就比較有競爭力
後來想想 應該說看父母要營造的是什麼樣的教育環境
我自己是台大畢業的
學校裡也看到很多從中部南部甚至東部來的優秀同學
能持續往前走的其實是讀書習慣
現在網路這麼發達
倒不一定要像以前一樣非得在台北才能得到資訊



: 背景
: 老公：居住南部較為鄉下，需輪班的公職人員，一天上班時數10-12小時，日夜顛倒，與
: 公婆同住，公婆年紀較大都已退休可以幫忙帶孩子
: 老婆：在生小孩之前居住偏北部 業務性質，工作時間彈性，主要客戶群在北部，娘家父
: 母都還在上班
: 育有兩子 分別2y&2個月
: 一直以來都是老婆是主照顧者，然後帶小孩兩邊跑，例如在娘家住兩個禮拜，再回來南部
: 住兩個禮拜

請問一下你老婆在台北就是在娘家上班了嗎？
不然如果也是上班族 我不覺得在岳父岳母沒有辦法有太多時間支援下
她有辦法一個人帶兩個孩子
還是是送保母？
可能你覺得這些沒有很重要所以都沒寫
可是我覺得這才是最重要要考量的點

你自己本身的工作性質就無法幫忙帶孩子
如果你要把你父母當成後援
你要考慮老婆的想法
不要一副我爸媽很nice一定可以幫忙帶的樣子
對你老婆來說 到南部等於是所有她熟悉的事物都要打掉重練
不光只有工作的問題
還有資源問題
比如說 在台北我知道哪邊可以找到臨時保母讓我喘口氣
在哪邊購物比較方便 花可以去哪裡買（現在都買不到切花好困擾阿嗚嗚嗚嗚）
這些都需要全部重新架構


: 娘家跟婆家都沒意見，覺得我們年輕人想住哪就住哪，婆家也不會給老婆壓力，覺得娘家
: 想住多久就住多久
: 之前都是兩邊跑，但現在面臨到現在小孩要讀幼幼班的問題
: 因為希望他去學校多一點團體互動所以打算送去唸書
: 現在一直評估小孩到底在哪邊會比較好
: 若在北部，競爭力較大，娘家可暫住，孩子大後還是要自己買房子，需老婆自己接送大寶
: 上下課兼顧二寶，因為娘家父母都要上班，但老婆的工作可以延續較沒問題！承接老婆媽
: 媽的工作
: 缺點：需帶著小孩跟老公分隔兩地，老公調動的話起碼3年後的事，再加上在北部需有自
: 己買房的壓力
: 若在南部，公婆打算買新房贊助頭期給老公，相對老公要繳貸款，公婆可以照顧小寶及接
: 送大寶上下課，跟老公一家團圓
: 缺點：老婆工作從頭來過，重新開發客戶，而且在很鄉下的地方對孩子未來的發展性不知
: 道會不會有影響

其實看到這裡
你有沒有發現
如果要到南部
所有的變動和調整都是老婆要承擔的
你不用改變工作 不用改變生活形態
甚至連育兒和買房子都是外包
除了繳貸款 你自己能給老婆的幫助是什麼？
金錢？時間？精神支持？工作援助？
請你務必思考一下
你必須要有調整自己的時間生活等的決心
這是你們的小家庭
你們必須站在對方的角度一起談一談
你能不能夠至少在精神上撐起一個家？
從原生家庭拿資源沒什麼 很多人都這樣
但除了清楚你自己拿的是原生家庭的資源外
你能不能承擔起一個家？


: 娘家一直希望孩子在他們那邊唸書，覺得發展性比較有空間
: 而且老婆的工作又能延續
: 但卡在老婆就必須一打二、二寶照顧問題、大寶上下課接送、跟老公分隔兩地對孩子的影
: 響
: 最近家裡老婆對這件事已經煩惱到很憂鬱了
: 因為娘家有給壓力希望老婆回去娘家上班，然後一直說北部競爭力對小孩的影響，小孩的
: 未來等等...

這些都是假議題
重點是你的態度



: 娘家很疼孫子希望能跟孫子一起生活，但在照顧上他們確實沒辦法因為都要上班
: 老公持尊重意見，但內心希望在南部能每天看到孩子，並且想參與小孩的幼兒園生活，加
: 上已經有房壓力相對較小，公婆能幫忙照顧接送孩子，覺得老婆若在南部找個簡單工作就
: 可以，家裡經濟還過得去。


你不能都要你老婆犧牲
犧牲她的職涯 犧牲她的生活 犧牲她熟悉的一切
你自己呢？
你有沒有覺悟為你的小家庭犧牲部分的你自己
你的時間 你的工作 甚至你的娛樂？
所謂的孩子競爭力，其實大部分是架構在父母的陪伴和用心上
孩子的快樂 往往也是被父母的感情影響

撇除掉雙方原生家庭能給予的支持
你有沒有辦法和你老婆兩個人維持一個家
這才是重點
如果你有這樣的決心 也表現出這樣的行動
我想你老婆不會那麼困擾

如果你老婆很愛工作 我想你不會上來po這篇
因為根本不需要選擇 她一定會留在台北
現在就是對她來說 至少小家庭是和她的工作同等重要
她才會猶豫不決
如果你希望她到南部跟你一起生活
請你拿出決心和行動


: 公婆則是持尊重意見，不論在哪對小孩好他們就好
: 請各位大大給點意見及支持......
: 若要噓可以噓小力一點嗎因為已經煩腦到不行了哈哈
: 不知道怎麼樣的選擇對孩子才是好的
