: 婆婆常找表妹來家族聚會，大概是想讓氣氛熱鬧一點。
: 有時她也會不請自來，大概因為是獨生女的關係，也喜歡熱鬧吧，
: 總之就是這些非預期的狀況，有時造成我的困擾。
: 例如已經為家族聚會訂好餐廳或想好菜單準備開伙，
: 突然被通知表妹也要來，我就需要重新聯繫或重新安排菜單與座位之類的，
: （我家位置偏僻，買菜不方便，家裡也不大），一次兩次可能還好，
: 但累積至今已有無數次（大節日的連假都會聚餐、公婆有時也會北上來訪），
: 和老公抗議無效，他覺得平常見面次數不算多，配合一下不要緊
: （平均至少1～1.5個月會有一次聚會）。
: 不過比這一點更心累的其實是，只要表妹一來，場子就變成她的，
: 焦點都在她身上，像是要一直看個人秀的感覺，但我對她的私事沒那麼好奇，
: 經常整場飯局都在聽她的工作生活或想認識的對象或感情觀人生觀之類的，
: 談不上有趣，就一般正常年輕OL的生活，
: 也可能是我們家有嫩嬰其實很累，我寧可回家好好補眠。
: 我了解長輩關心她，可能也喜歡聽，但我聽到後面真的只能放空，
: 畢竟也不熟不適合提供批評指教。
: 坦白說表妹不是壞人，而且我相信她是真心敬愛我婆婆，只是個性比較自我。
: 還有一點可能是，她終究不是我妹妹，我無法那麼順著她或縱容她吧，
: （包括配合她說來就來或配合她想吃的餐廳想去的地方突然改行程）
: 甚至如果我有女兒，絕不會這樣教她，一定要讓她懂得尊重別人，
: 大概就是這種觀念的不同，讓我的容忍度比較低，聽這樣的人說話會感到疲倦。
: 總之因為她或是婆婆的關係，已經有好幾次不想出席家族聚會了，
: 每次勉強出席心情都很無奈，反而只要她不在，就多少鬆一口氣，
: 覺得總算可以和家人之間有較多互動和交流。
: 眼看聚會在即，同樣的戲碼又再度重演
: （婆婆又臨時找她來，但我家餐桌擠不下，只好取消預訂好的食材，改找外面餐廳）。
: 腦海中不斷閃過各種可以不出席不看秀的理由，疲倦的媽媽很想圖個清靜，
: 又不希望老公和公婆擔心（老公不太接受我找理由「請假」），該怎麼做才好呢？

這種大鍋炒的聚會我也不喜歡。
我平常工作時間很長又緊湊，下班又要跟打仗一樣處理小孩和家務
(先生每天工時都超過12小時，無法指望)
所以周末的時間對我而言非常寶貴，只想好好休息，
不然就是把平常想處理但沒時間的雜事辦掉，
要是每個月都來一次這種家族聚會，我也會想發瘋。

這跟妳羨不羨慕對方無關，也不是嫉妒，
而是對我們這種不好此道的人來說，這就是一種"情緒勞動"。
從餐點準備開始就要顧慮到各方想法，
然後因為其他與會者妳不熟，說話要拿捏分寸，
又因為帶著無法自理的年幼孩子，妳要另外張羅他的吃喝拉撒，
其他人興起就把孩子抓來玩一玩，膩了就塞回來給妳，
可能聚會結束還要花力氣收拾殘局，
原本妳想作的事情，因為聚會把時間都用掉了，只能繼續往後擺。

那些喜歡聚會的人可能不社交會死，一天不講個三五萬字會死，
但我們就不是這種人啊，我寧可拿有限的空閒時間運動、閱讀、逛街，
都比跟一群半生不熟的人擠在一起想辦法找話題好。

我公公也是很愛呼朋引伴的人，又常常要我們帶小孩出席，
我每次跟他聚餐都覺得很痛苦，
聽他跟朋友在那邊幹古、講學生時代的事、當兵的事，
不然就是說些我笑不出來的玩笑話，或是只有兩三人懂的黑話，
我常常都不曉得我到底坐在那裏幹嘛，
然後小孩餵飯要我餵，上廁所要我幫忙，坐不住了我還得想辦法轉移注意力，
我先生從小習慣了這種社交場合，根本不覺得哪裡有問題。
幾次以後我就跟我先生抗議，
要他喜歡天倫樂就自己帶小孩，不要浪費我寶貴的時間，
兩人吵了好幾次大架才終於有一點點改善。

奉勸原po，妳說什麼怕老公擔心、他不會溝通，那都不是理由，
他的親戚請他自己負責，
不會準備餐點就從現在開始做，不會溝通就要練習，
妳做的愈盡善盡美，他們絕不會感激妳辦事周到，
只會覺得妳愛做都讓妳做，
婚姻裡吃力不討好的事情非常多，少一件是一件。
