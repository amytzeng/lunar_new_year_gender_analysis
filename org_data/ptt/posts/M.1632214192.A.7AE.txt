: 男友結婚。交往時另一半家境不佳，鮮花禮物沒收過，只有畫卡片給我。我比對方早出社
: 會，很早開始同居，是我一次先付清兩年份房租，他再逐月給我一半。2016年結婚時，他
: 剛工作不到一年，我放棄了婚禮婚紗，只有去戶政事務所登記。蜜月旅行他討厭坐遠程飛
: 機去歐洲，我答應他去東京就好，全程我出錢，還買了一隻表給他當結婚禮物（不是很貴
: 的）。新婚我們租了有公寓的電梯套房，前2年的房租我要簽約所以先出，他則買了一台
: 摩托車作為代步工具，並且支付我們兩個人的生活開支。那段日子過的很開心，我媽說我
: 廉價白白嫁給人家一定會後悔的言論被我嗤之以鼻，我甚至把當時幸福的感覺用word打下
: 來放在筆電的桌面，現在看起來極其諷刺。
:
: 好景不常，2017年我懷孕，產檢他就陪我去了2次，後面我只能自己去。動了胎氣也是自
: 己去住院，還被護士問先生是不是在很遠的外縣市。孩子出生以後，我在高雄的月子中心
: 坐月子，他只有來過3次。孩子出生以後，我回高雄工作，他為了追求自己的夢想把台南
: 的工作辭了，一個人跑去桃園從零開始，一個月回來一次，給我媽錢，讓我媽當我隊友。
: 我過上了偽單親的生活。我的育兒隊友只有我媽，而我媽是一個情緒勒索慣犯，每天言語
: 霸凌我，我只能跟另一半訴苦，卻被他說我不知足，負面，我過的已經比世界上大部分的
: 人好了。後來孩子上幼兒園，我揮別了我媽，一個人過起真單親媽媽的生活，每天請半小
: 時一小時的年假，奔跑去幼兒園接小孩，怕她留太晚會哭。
:
: 這種日子一直到了2021年的今天，三級警戒開始。配偶在桃園的工作量加重，開始拒絕跟
: 我視訊，也不大回line，三級警戒三個月他就三個月沒有回家。到了父親節，我見到自己
: 久違的配偶，女兒見到了久違的爸爸，但是他對我的態度很冷淡，也百般嫌棄我。在七夕
: 的隔天，他用line對我提出了離婚。
:
: 事情已經發生了，我哭了怨了，但還是要想想未來的日子怎麼過。我自己的背景，是被媽
: 媽帶著嫁給繼父的拖油瓶。這讓我想到另外一個問題，是萬一再婚對方的家族不會接納我
: 的小孩。
:
: 我自己是被我媽帶著跟繼父結婚的。在我的成長過程中，繼父的家族從來沒接納過我，每
: 年的過年我都是去外婆家過。一直到我17歲，繼父經歷了N度失業，並且也過了生殖能力
: ，阿嬤才讓我回家過年。到今天，我基本上不認識所有同輩的親戚小孩。繼父在事業很好
: 的時候很疼我，事業失敗後表面對我很好，仔細想想也很多把我當工具人的地方。譬如曾
: 經在高中時問我，長大就業能不能給他50%的薪水，不管賺多賺少。譬如在往生前的臨終
: 交代，沒有一句跟我有關。繼父這邊的親戚也跟吸血蟲一樣，到今天還住在我名下的房子
: 裡不肯搬走（明年春節後我要走法律途徑跟這些人永遠的劃清界線，這點倒不用擔心）
:
: 總之，我以一個“被媽媽帶著嫁去別人家族的女兒”視角回顧這過去的30年，覺得爛透了
: 。對方的家族完全跟我是陌生人，有需要才會利用。我還算運氣好的，至少繼父對我沒有
: 邪念。但是外公在我12歲那年慎重告誡過我，這個男人不是我的親生父親，因此成長的過
: 程，我一直對他很提防。自從12歲被告誡後，我跟繼父的相處，永遠都是衣著整齊的。他
: 不會進我的房間，也不會有什麼叫我起床的情況；也不敢跟繼父擁抱，我不敢跟他身體接
: 觸，怕出事。我一直記得大學參訪姐妹校後回國，每個人見到自己接機的親人都是衝上前
: 擁抱，只有我是推著車子飛奔到繼父一步前停下來說“爸，我回來了”
: 我很愛我的女兒，那是我身上掉下來的一塊肉。看著她天真的笑臉，一聲聲的說著愛我，
: 我真的不希望她和我一樣步步驚心的長大。我希望她無憂無慮，可以和一般女孩子一樣，
: 在家怎麼舒服怎麼穿。我希望她可以盡情擁抱家人，不用擔心自己的舉動會不會被對方解
: 讀為“釋出了親情以外的訊息”。如果要達到這個目標，我必需放棄再次認識別人的心，
: 做好孤老終生的打算。
:
: 另一半的經濟能力比我好上許多，女兒跟著他不會吃苦，能過上想學什麼才藝，想上什麼
: 學校都不用考慮的生活。但是他太自私了。他只顧自己，只愛自己，對女兒不會像我這樣
: 細緻的照顧。以後他的再婚對象，也不一定會善待我的孩子。孩子在家庭得不到關愛，可
: 能又會走上我的老路，有人對她畫卡片，就死心塌地的守著對方。
