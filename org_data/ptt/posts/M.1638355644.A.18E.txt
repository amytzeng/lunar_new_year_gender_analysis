對你來說是招待累人加上擁擠
雙重不舒服

不過下次要直說比較好
不要讓老公答應了又變卦

※ 引述《mowota06 (Glue)》之銘言：
: 我跟我老公還有小孩住在北部，
: 沒有跟公婆住在一起（他們住南部，小叔跟他們住在一起）
: 事情是這樣的
: 原本我計劃是
: 禮拜五拿蛋糕提前一天幫我兒子慶生
: 禮拜六早上我要去產檢，產檢完之後帶我兒子出去玩（甚至也可以兩天一夜）
: 結果昨天我老公接到電話說
: 公婆想來幫我兒子慶生
: 順便帶上小叔的大兒子一起慶生
: 然後在我們家住個一晚或兩晚（不確定）
: 但由於我們家是租的非常小
: 只有兩房一廳一衛
: 一間我們自己睡，一間放衣服以及雜物等等的
: 所以他們來真的非常不方便
: 而且我現在懷著第二胎我不想要有人打擾
: 我就跟我老公說要不要我去訂外面的旅館給給他們這住
: 我會這樣想是因為
: 1、家裡空間太小不方便
: 2、我不想讓他們睡地板我覺得很怪
: 3、家裡小到再多人進來真的沒有隱私
: 我老公也有跟我婆婆說明說家裡太小了
: 問要不要去外面訂旅館讓他們睡外面比較舒服？
: 他們說不用！
: 睡家裡打地舖就好啊！有沒有要住很久
: 主要是要讓「小孩一起玩」，看小孩
: 於是我跟我老公就吵架了
: 他的立場是堅持：他已經跟他爸媽說了
: 1、很小
: 2、要睡地板
: 他們都能接受就好
: 但是我的立場是：
: 1、很小之外我沒隱私
: 2、我不願意看他們睡地板很奇怪
: 今天換作我爸媽我一定會表明我不方便的地方
: 我會訂旅館給他們睡比較舒服
: 就算他們嘴裡說睡地板沒關係
: 我也不可能看他們睡地板
: 吵到後面我提出妥協的辦法：
: 1、訂外面旅館，要嘛他們去住，要嘛我們夫妻出去住
: 2、出去外面玩兩天一夜，全部都住外面
: 小朋友也能一起玩到
: 今天我已經把我不方便不舒服的地方講出來
: 就為了日後不要有一樣的事情再次發生
: （這件事已經發生過一次也是吵架，這次我攤開來講明白想要解決問題）
: 也提出了我覺得比較折衷的辦法了
: 但是我老公卻覺得我很無理取鬧針對他的家人
: 還問我說：如果今天是我爸媽來，他可不可以也不接受
: 我也跟他說明
: 我會跟他討論時間上我們能不能接受
: 確認時間上可以我才會讓他們來
: 我也會找旅館讓他們住外面（真的太小）
: 以不要打擾我們為主這才是最重要的
: 如果我老公覺得不妥我也不會讓我爸媽來
: 造成我們的困擾甚至讓我們吵架
: 我公婆其實是很好的
: 我也知道他們只是想來看孫子
: 只是今天這樣的事情竟然會讓我們夫妻吵架
: 我也覺得他們也不想看到
: 但是我也想要保有自己的隱私
: 而不是說什麼住一天兩天又沒關係等等的
: 然後讓自己不開心
: 如果每次發生事情都有一方妥協
: 沒有去解決
: 不滿一定會日積月累到後面無可收拾的地步
: 我不想要這樣
: 所以我把我的想法給說出來
: 所以我也不知現在該如何是好
: 我該做的都做了
: 我現在真的非常的難過…
