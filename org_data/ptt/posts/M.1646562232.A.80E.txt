: 分居幾天後，老公昨天回家收拾他的日常衣物。
: 我們重新談了一次那天的事件，這裡分享一下雙方在冷靜狀態下的說法:
我是一個非常討厭要跟姻親交際的人
同時我也是很討厭毛多的人
所以你們雙方的說法我都可以理解

: *老公表示:
: 他家人到社區大門時，小孩突然說想上大號，所以他媽媽才提議來我們家。
: 因為他手上提了很多東西，一時間忘記要先打電話問我。這點他知道自己不對。
: 他家人也剛好都沒人想到，但大家真的不是故意的。
: 另外他媽媽覺得是去自己兒子家裡，沒先說有這麼嚴重嗎。
: 雖然他媽媽也知道忘了先通知是不太好，但她覺得我有必要這樣嗎?
另一半這樣的解釋我可以接受
因為這算一連串的事件導致的
1:被家人開車送回
2:小孩想上廁所
不要說事後有沒有上,你小孩說他想上廁所
事後又沒上,難道是惡意的?
3:手上東西很滿+家人在聊天
有時候氣氛很嗨的確不會特別想到要打電話通知

我覺得這些事件的組合可能一年沒有一次
所以氣過之後就不會太放在心上
不過這個你的下面有提到你也可以接受
其實重點是在後續的溝通態度

: 他說他媽媽就是很不喜歡"家人之間不打招呼"，他們家教就是家人間應該打招呼，
: 接著開始舉例以前某某親人每次都不打招呼，所以他媽就很看不下去。
: 他也因為這樣覺得婆媳不要住一起比較好，所以才很體貼我，跟我一起買房子住外面。
: 接著說爸媽跟自己也很常直接跑去弟弟家，弟媳就從來沒因為這樣不開心。
: 既然我明白他很注重他家人，那為什麼不能當下先做個表面，事後再跟他私底下溝通。
: 當我表示，他在外面跟他家人談笑風生把我當空氣，我覺得這是不重視我的表現，
: 他就解釋說，其實他是在幫我接待大家，因為我不出去打招呼，大家都知道我在生氣，
: 其實大家有點尷尬，那為了避免氣氛更尷尬，所以他在外面聊天談笑緩和氣氛。
: 如果他進房間哄我，他怕媽媽會覺得，兒子怎麼這樣，只顧媳婦不顧外面的家人，
: 這樣他媽媽反而會對我印象更差。所以他是為我好、幫我做面子，我誤會他的用意了。
: 他送家人離開社區後，回來其實有想跟我好好溝通的。
: 我說他一開口的那句"你剛才為什麼都不出去打招呼"感覺不像是想好好溝通，
: 他說他覺得自己沒有態度不好，不知道我為什麼這樣覺得，其實他沒有想跟我吵。
: 他講到一半直接收東西回家，是因為當下雙方都很生氣，無法溝通，他想讓雙方冷靜。
: 接著，他感嘆我怎麼都不會想說要主動跟他爸媽打好關係，
: 並舉例說他爸媽跟他碎碎念的時候，他雖然會不耐煩，但不會頂回去跟爸媽起衝突。
: 如果我跟他媽感情很好，如果他有什麼事情讓我不開心了，我可以找他媽一起去唸他阿!
: 我問他，所以他的意思是，同件事情我跟他講，他會聽不進去然後跟我起爭執，
: 但是他覺得換他媽跟他講，他就不會生氣，也因此不會跟我吵架囉? 他說，對。
: 我問，我到底為什麼要用你家人當傳聲筒啊?
: 他嘆氣回，他不是那個意思，他只是希望我跟他家人感情好。
這部分我覺得在心平氣和下溝通是沒問題的
但是在已經吵開的情況下
就誰都不想退讓

: *我的說法:
: 1. 因為手忙腳亂，所以忘了通知我，這點我聽完可以理解。
: 我主要是因為他事後跟我溝通時的態度。
: 我感覺被單方面指責，以及他直接收拾東西回婆家的行為，是引爆我提離婚的原因。
: 另外不是因為此單一事件才提，是因為之前已經有過無數次雙方一直做無效溝通的情況。
: 我們兩個人在溝通時，都會抓著對方的錯不放，沒人先退一步軟化。
: 最後會和好，都只是因為事過境遷氣消了，覺得不想計較了，
: 但其實雙方都沒在改變自己的溝通方式，因此下次起衝突時還是一樣吵架。
所以其實重點根本不是尊不尊重的問題
是後續雙方溝通的態度導致更生氣
這我也很理解~
常常吵架爆發點已經不重要了
更氣的的是開吵以後的說法跟做法

: 2. 買房子的錢我們全部一人一半。這房子不只是"兒子的房子"，也是"媳婦的房子"。
: 會覺得"直接來兒子的家無所謂"，讓我感覺不受尊重。
: 他們把弟媳家當自己家，弟媳不生氣那很棒，但那是她的事，不能拿她的例子來壓我。
: 他們家族的家庭觀，跟我不合。
: 3. 他希望我 "先跟他爸媽打好關係，這樣我們夫妻的感情就會更融洽"
: 那我為什麼不能期望他
: "先好好對待老婆，非常疼老婆，讓老婆主動想愛屋及烏去親近他爸媽"?
: 我的要求有很不合理嗎?
: 他說，這是先有雞還是先有蛋的問題。
: 我的想法是: 好吧，沒人想先退，沒人想先反求諸己，那就雞跟蛋都不要了。
: 4. 我去他家時，跟他家人來時，我都很有禮貌的打招呼/寒暄，
: 就這次，因為被冒犯到，所以我沒打招呼。為什麼在他眼中就這麼嚴重?
: 我雖然不擅常跟他家人尬聊，但我已經有盡力想讓他家人對我印象不要太差。
: 去他家吃飯，我能幫忙洗碗都會幫忙洗碗。
: 有兩三次他媽媽說腳痠痛，我就幫她約spa館(我買單)，也跟她說想再去都可以跟我講。
: 母親節/婆婆生日，他說不用準備禮物，我都還是準備。
: 化妝水、精華液、護唇膏、口紅、餐券、按摩券。
: 一些小東西在特殊節日表達心意。過年禮盒也會準備。
: 他妹妹開刀但醫院不開放探病，我改成買一盒滴雞精給她。
: 我其實不是特別愛小孩的人，但他弟弟的女兒很喜歡我，
: 所以有見面時我都會陪她們玩，畫畫、黏土、撲克牌、扮家家酒。
: 說這些，不是想表達自己付出好多好委屈，
: 或是我為他家花這麼多時間心力，他為我家什麼都沒做，好意思在那邊一家人，云云。
: 說實話，我也沒真的付出多少，我承認自己沒有付出100%的心力去主動融入他們。
: 雖然我覺得自己盡力了，但也許因為我做的都比較像是 "表面上的禮數"，
: 所以他會都覺得 "我跟他家人感情不夠好"。
: 讓我一直有一種"自己不管怎麼做，對他來說都不夠好，都是我不對" 的無力感。
: 結論，他們的家庭觀，跟我不合。我們個性，也一直都不合。
: 離婚已經是我一定會做的了。

不然你也不會家裏亂糟糟被人看到就要爆氣~

就看怎麼幫助你另一半通過這些考驗

所以我會想辦法幫他通過這些家庭的考驗
