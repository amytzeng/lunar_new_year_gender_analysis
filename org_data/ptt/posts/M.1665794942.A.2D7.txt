: 雨天 我騎車接小孩(小孩1y2m)

: 這樣很過分嗎?
: 我有穿雨衣 然後用背帶 回到家的時候小孩也在雨衣中睡著
: 但我老婆卻質問我為何不開車
: 我的理由是
: 1.保母家有時候警衛不在  無法開進地下室
:   車子無地方停，就算停在外面 反而要撐傘抱著小孩上車
:   小孩更容易淋到雨
: 2.因為我們社區車位要用搶的 (公用的)
:   如果沒有車位 就是要停在路邊
:   又會遇到上面的撐傘抱小孩下車狀況
: 3.上班開車去保母家會塞車，小孩在車上會一直哭鬧
:   反而增加彼此痛苦，騎車可能快速3~5分鐘就送到保母家了
: 綜上所述
: 雨天騎車接小孩是很過分的事情嗎?




  其實有時候騎機車載小孩也是無可奈何啦~

  尤其是沒有後援 1個人顧幼嬰

  難免總是要出門買個便當 午晚餐 尿布 奶粉...


  除非嬰兒1歲以下 脖子還沒硬

  不然在安全的情況之下 騎機車載小孩也是不得不的選擇...


  要開車當然是最好

  但開車不是問題 停車才是問題

  說到停車 就要有停車場

  但除非是麥當勞得來速

  不然只要開車購物 通通都有停車的問題


  今天你要買個便當

  最近的停車場在便當店400公尺之外 走路大概10~15分鐘

  整個流程是這樣

    把小孩放到嬰兒座椅 -- 開車到停車場 -- 停車 -- 把小孩放到推車

    -- 推車步行10分鐘 -- 買便當 -- 推車步行10分鐘 -- 小孩放到汽座

    -- 推車放到車廂 -- 繳停車費 -- 開車回家


  我是覺得啦~

  為了買個午餐

  要推個嬰兒車

    夏天大太陽下走路20分鐘 / 下大雨中走路20分鐘

  能做到的人不多啦~


  不然 反對騎機車在嬰兒的人

  說看看 你們是怎麼開車出門買便當的??

   (不能路邊閃雙黃燈違停喔~ 這樣很沒品)




  其實阿~

  我覺得台灣少子化 國家應該要立法更加提升育嬰時的路權

  如果騎機車載6歲以下幼童

  應該要享有跟行人一樣的路權

  騎機車 可以走騎樓/人行道/斑馬線 ...

  這樣的動線 騎機車跟行人就會一樣是最安全的動線...

  時速10KM以下 + 行人路權

  安全性就很高了
