※ 引述《catloveallen (阿傻)》之銘言：
: 去年因至親離世帶給先生很大的遺憾，也因一直無法說服長輩與我們同住
: （年紀大了要離開既有的生活圈確實很辛苦）商討後決定於今年孩子上小
: 學前搬家一家團圓，也能好好照顧長輩兩全其美

: 爭吵主因是前陣子與主管提出辭呈後，主管立即表達支持並表示如果與另
: 一半取得共識後，能協助轉職至其他區域；工作職務與待遇不變，未來可
: 申請彈性上下班，且可待孩子安頓好後再行轉職，這些是目前的工作主管
: 能給予的最大協助

: 但至今與先生一直無法取得共識，我能理解先生一直期待一家團圓，我能
: 多花點時間陪伴先生與孩子，希望辭職後我能好好休息一段時間再重啟職
: 涯，這段時間無論我想做什麼他都不會過問，經濟上先生其實也給予了最
: 大的自由（他承諾未來我的工作所得，他不會過問，家用與貸款他皆能全
: 權負責）

: 我非常感激先生給予的全力支持，但自己私心的想法：目前的工作內容是
: 我真心喜歡且待遇佳，雖然工時相較長且壓力大，忙碌時期常回家小孩都
: 已睡著的狀態，但工作確實也讓自己從中獲得了莫大的成就感；也打聽過
: 目前的職缺，和其他公司相比之下，女性求職相較困難許多，且因家庭成
: 長環境，我更希望自己能保有一定的經濟能力，畢竟未來變化無法預知，
: 更不願意事事依賴另一半…

: 然而多日與老公溝通，始終無法找到彼此都能接受或是妥協的較好方案，
: 且身旁好友多數都是相同性質的工作內容，想法難免因我的單面說詞而支
: 持我，所以希望藉由板上求助，或許能找到讓彼此都能接受的最佳方案，
: 或是指出我的盲點能讓我清楚自己欠缺的部分，謝謝大家

除非是自己不想工作，不然繼續工作應該是最好方法。
至於如何照顧小孩兼顧家庭這就要和老公一起討論。
可以使用的方法就是雙方先將不可調整的部分寫下來。
然後兩邊先確認不可調整的部分是否是共識。
接著將需要的部分和不可調整的比對，找出中間需要妥協的方案。
這種事情夫妻雙方都需要努力，而不是單方面要負責的。
