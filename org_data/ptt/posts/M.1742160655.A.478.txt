不管是因為什麼理由住娘家
選擇住娘家是一個既定事實
那就必須要意識到
你們生活習慣相差甚遠
代表你先生也必須去調整
才能配合你們家的生活習慣
但你的言談中透露出你眼中彼此生活習慣的位階…
婆家在自己私領域的生活習慣那是他們自由
你反倒應該要入境隨俗
因為你先生住你娘家他也是在入境隨俗

: 至於為何沒有選婆家
: 因為生活習慣與娘家相差甚遠
: 婚前我就已經跟配偶明說我不會住
: 當初只是想在一個舒適跟自己熟悉的環境養育小孩
: 卻沒想到這會讓配偶不斷逃避育兒
: 變成一個豬隊友

每個人育兒方法不同
有人認為小孩不能在太乾淨的環境下成長
不然未來就會抵抗力低下
但也有人說要避免生活當中的過敏原
他不做我想不是他做得不好
而是他在心中根本就不認同你的育兒方式
若是他心中認同你的方式他自然會自我要求把它做好
如果育兒這件事情你過於強勢
什麼準則都是你說的算
他的價值觀一直被你否定
他自然會選擇逃避

: 孩子還沒睡過夜時
: 我每天只睡3小時
: 3小時是娘家家人下班後
: 幫忙接手照顧小孩
: 我才能睡覺
: 若不是住娘家 我真的快撐不下去

新生兒三小時的週期不是你小孩的個人特質
你可以媽寶板問問看
那些無後援的父母是如何走過來的
很多人會抓時間睡覺
但我看了很多父母
都是在小孩睡覺之後做自己想做的事
報復性的要將自己的獨處時光討回來
然後再說自己沒時間休息…
小孩出生後
很多家務要下放標準
有些人轉外食有些人降低清掃頻率
總是要有所取捨

: 我確實很懶得帶小孩外出
: 小孩極度容易炸屎
: 我每當想到要在外面處理這件事就很頭痛
: 出生至現在
: 只有打預防針跟去婆家才有出門
: 也許等到孩子開始吃副食品 狀況會好一點

人總是要出門去透透氣曬曬太陽的
你和你先生是你小孩更是
你可以先從有哺乳室尿布檯的區域開始
博物館、兒童樂園、百貨公司開始練習出門走走
等到小孩子的拉屎頻率慢慢降低
可以再開車出去遠一點的地方走走
懂得散心把自己的情緒顧好
在心理層面上對育兒環境也會有所提升

: 有時候覺得我沒去婆家
: 是否沒看到一些事
: 心裡就不會有疙瘩
: 跟配偶反應過一些其他的事情
: 他覺得都是小事
: 是我太敏感

這不就呼應我之前說的
你們之間的生活習慣、育兒習慣差很多
而這之間沒有誰高誰低
大家都只是在自己的私領域做自己
你先生入境隨你娘家的習慣
那你到婆家時就應該用相同的標準去習慣婆家
大家互相配合才能互相效力
而不是誰單方面去調整配合另一方

: 上週末還是有跟著配偶帶著小孩回婆家
: 請配偶準備孩子外出的東西
: 配偶也答應2-3小時就離開
: 還好都有按照計畫走

如果凡事都有配合你的計劃走
配偶也是心甘情願
那就請繼續保持…
祝福你們
