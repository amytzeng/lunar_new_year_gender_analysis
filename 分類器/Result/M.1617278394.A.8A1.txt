: 可是當職業婦女卻常需要為小孩犧牲

我覺得在於老公是否是神隊友，
像我們家是我老公臨時請假方便 就由他去做臨請假這件事，非得由性別來決定、而是工
作型態，如果真的是女生較容易臨請，兩夫妻也可以在別的地方讓男生多做一點，這樣由
妳臨時帶小孩看病妳也不會覺得不平衡或特累。

: 白天忙著工作，下班忙著張羅晚餐

那就不要煮，幹嘛累死自己，到底是滿足自己身為媽媽的存在感還是被要求？

: 孩子小時候若是學校有回家手工作業
: 要陪孩子想幫忙動手完成
: 學校辦活動也要親自參加

不太懂爸爸去哪呢？？？？

: 做家事 陪小孩 忙了一整天精疲力盡
: 平常累得要命 假日要帶小孩出門玩
: 會很懷疑什麼時後能有屬於自己的時間

 我老公跟我都互相理解每個人都需要獨處做自己事的時間，所以他有時半夜去看電影，
半早上去瘋排唐吉軻德，平日去跑步強身，我都豪爽答應，因為他也懂得讓我假日去做美
容，平日去健身房、上跳舞課等等

我老公常常會一個人帶小孩到處玩，我參與就是天倫之樂 我不參與就是我的free time，
所以說到底，要跟老公協調好生活型態，沒有其它解決之道。

: 為工作認真努力 又要為小孩用心良苦
: 至於現在的食品安全問題
: 總是希望自己的小孩吃的健康又營養
: 如果自己煮早餐就得早起
: 如果買外食就又希望能好吃又健康
: 但殊不知 食安問題搞得煩惱
: 不能吃太多糖 不能太鹹太油
:  市面上食品添加劑很多
: 買青菜要擔心農藥問題
: 肉類又怕有生長激素 瘦肉精
: 自己煮的要美味 水果要多汁
: 一邊煮還要一邊注意小孩動向
: 希望他們纖維質要吃夠 吃健康
: 少油少糖少鹽 但偏偏小孩又愛偏食
: 家裡要保持乾淨對好好休息很重要
: 若是雜亂的空間 對心情跟思緒有壓力
: 擔心塵蟎會導致過敏 氣喘
: 吸塵器 除溼機 空氣清淨機
: 市面化學合成清潔劑對身體可能有影響
: 垃圾要分類 時間到要拿去丟
: 東西要整理收納 有些時候還要佈置
: 營造家有溫馨感

看來真正原因是妳自己無法放過自己，說再多也沒用。其實我覺得身為媽媽最需要學習的
就是「做自己會感到快樂幸福的事、接受自己的不全能，然後要學習用包容的心看待一切
」
包含老公育兒方式很笨手笨腳、、小孩怎麼教就是跟妳唱反調、亦或是別人對妳的養育方
式指指點點 這些不如自己預期的雜事都要輕輕的看待並放下。

: 身材要保持苗條 最好看起來很年輕
: 保養得好 看起來有自信

不覺得做這件事很累。
