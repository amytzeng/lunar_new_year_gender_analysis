這個不好說什麼，做試管痛苦的也是女生，如果女生不太願意做，
男生出錢也是應該。

「可能都是我講話刺激到他」其實這是重點吧？
簡單來說幾乎每次吵架都是妳先點火的？那這樣可能要想一下為什麼？

: 有了小孩後，大部分都是我在生氣，因為他常態性得口氣差，他跟我講話的樣子，很像在
: 看仇人一樣兇狠，他的脾氣本來就差，只是婚前隱藏很好，婚後我才慢慢發現他的本性，

老實說，口氣差不差其實很主觀，後面妳舉的例子也證明了這點。

: 今天的爆點也不大，平常假日他會為了小孩早起，開電視跟小孩看，說這就是顧小孩，還
: 說他顧了很久，很辛苦很累。我心中只覺得這根本不負責任，但我沒說出口。接著我去煮
: 午餐，煮完後我跟2歲半的小孩說，你要練習自己吃，不能老是給人家餵，要自己練習，
: 快去坐在你的餐椅上準備吃飯。結果，小孩就聽著覺得委屈，就哭著跑去找爸爸，我先生
: 就（我覺得很兇的口吻但他不覺得）說，你幹嘛對她這麼兇！
我覺得不管用什麼方法顧小孩，至少人家時間就是花在小孩身上了，
跟小孩看電視也不可能看自己愛看的，單純看這件事情，
是妳該感謝他在這個時段幫妳照顧小孩，而不是嫌他顧得不好。

: 我說，我沒有兇他，我只是叫他要自己吃飯。
: 我先生就說，你看看你自己的嘴臉，就是一副很兇的樣子！
: 我說，我幫你們煮好飯，叫小孩要自己吃飯有什麼不對？你要這樣兇我！？
: 他說，我沒有兇你，是你自己在兇！
: 我大怒，摔下筷子在回房間冷靜。
兩邊都覺得自己的口氣沒問題，是對方的口氣有問題，就像我前面講的，
口氣差不差其實很主觀，妳覺得對方脾氣不好，其實可能對方覺得妳脾氣才差，
或者其實兩個人脾氣都不好

: 過半小時，我實在很餓，臭臉出來裝碗飯自己吃飯，暫時沒事。
: 吃完後，我先生準備要去好市多買本月的肉，我則要帶小孩去公園玩，我請他幫我買隱形
: 眼鏡，他說好。
: 過一會，我已幫小孩換好尿布，換好褲子準備要出門，我先生說，先把隱形眼鏡錢給我我

: 不能轉帳。
: 他說，我來顧啊，你先轉錢給我。

: 但我忍不住要問，你難道不能先墊付這800元嗎？你買好，連同菜錢我一起轉給你。
我也想忍不住問一句：妳錢包裡面沒有1000塊嗎？馬上從錢包拿1000就能解決的事情，
第一次看到請人幫忙買東西，沒有先給本金+車馬費還問別人難道不能先墊的。

: 他說，這兩筆是分開的，你的東西是你要付的，你要先轉給我，不然就不要買！

: 買個隱形眼鏡要逼我趕快給錢，你是有點這麼缺錢嗎？連個800都不能先墊付？
這個觀念不對，買菜吃飯是兩個人的事情，煮菜看妳有沒有堅持要煮，
我自己是覺得煮菜辛苦就直接外食沒差的，可是他幫妳買隱形眼鏡，
本質上是「純粹幫妳」，如果是我遇到這種態度的人，
我當下不會說什麼但後來會直接不買然後回來說身上剛好沒錢了，
妳要的話就自己再跑一趟買。因為憑什麼一定要幫妳做這件事？不是嗎？

: 他說，你到底要不要轉？不轉就算了。
: 我轉好了，但我心真的覺得很受傷，到底為何我的老公要跟我斤斤計較，什麼都要我出錢

: 可是為何當初大方的男友變成這種小氣鬼。
: 我們其實為了錢吵過很多次了，他說，如果他不用跟我一起買房子，一起養小孩，他可以
: 薪水的拿來對我好，也可以很大方。每個月的錢就這樣花掉就好，大家都開開心心。
: 我無奈，我相信未來房子蓋好後，小孩長大後還有更多吵不完事情。然後，夫妻間也沒有
: 改善解決方法，因為他說平常就很累，下班後他只想休息，沒有性生活，也沒有坐在同一
: 個空間聊天交流，他躲在自己的書房戴耳機看卡通，我一人在客廳看Netflix。
: 距離很遠，心很累。

1.你們到底房子買了沒？因為前面說有共識不買了，現在卻說「未來房子蓋好後？」
  看起來是已經買下去了(預購屋)，房貸頭期款誰出？房貸平常是誰在繳？
  還是說房子還沒買但先生已經在存錢？先生感覺上平常被錢逼很緊，為什麼？
2.說句可能不中聽的，妳的個性裡面有種「薄情寡義」的成分在，
  也就是說別人對妳好，妳可能會覺得那是應該的，沒有感謝還覺得別人做得不好，
  如果遇到這樣的人，個性再好的人也會變得斤斤計較，其實會計較還算好的，
  有的人可能直接冷處理了，就好像隱形眼鏡的例子，我身上剛好沒錢妳能說什麼？
3.妳當然對小孩也付出很多，錢跟心力都有，這是妳的優點，
  可以跟先生理性分析一下，不要口出惡言，相信他態度也會轉變的。
  如果實際上妳花的心血比較多，其他不看就算理性分析，也知道要和好，
  當然如果妳心意已決要離婚，就當我沒說。
