: 婚前我父母沒有喜歡她，尤其是我們第一次分手
: 她就把我的東西都寄回來，老人家覺得失面子，
: 所以等我們要結婚就用冷處理來刁難，最後我選的是她不是我父母
: 但放心吧，這不會被她記在心裡。
: =====婚後
: 婚後我們就出國了，做啥不重要，反正是需要專心，而且是她
: 也做過的工作，所以工作內容她知道，產出跟評比項目她也都知道。
: 生了兩個小孩，所以我也是請過五個月的育嬰假
: 婆婆（我媽）跟她娘親都有來顧過十天，然後都回台灣去
: 一個被氣跑，但婆婆就不知道了。
: 老婆後來診斷出重症一次，開刀五次，五次都不離不棄照顧
: 小孩上小學後，她盯的很緊，覺得小孩如果功課掉了就完了
: 但是她表面上就是為了小孩
: =====婚後十年的感想
: 我覺得遇到那種想全拿的人，真的是無解，而且不知道為什麼
: 想全拿的人會有玻璃心的狀態，被講一下下就會崩潰發飆，如果是外人
: 反而不發飆，因為面子，所以都回家發飆。
: 所以婚後我是越來越不說心理話，講過幾次，都是她發飆收場。
: 但如果是她講心理話，你就要感恩戴德感謝她。
: 像是她說你像自己的父母，就是要幫你改
: 如果你講她像她父母，你就是不懂她的情況亂下指導
: 所以其中一個被抱怨的就是我不會規勸她
: 另外就是，你的工作不是工作，但是她的工作就是工作。
: 所以當你有工作要趕時，你不能太趕，你要照顧她的心情，
: 如果吵架，你最好先顧吵架的事，不要先顧經濟，不然就是不愛她。
: 所以其實我的工時是不到每週四十小時，但是老闆都還蠻滿意我的產出
: (除了某個會覺得我不應該花時間給家庭的老闆，六個月後我就被fired了)
: 我想應該是因為我還算聰明，用最好的方法（程式化）跟感謝老天給我
: 還不錯的頭腦，不然我真可能會喝西北風吧。嗯，我也覺得我比她會做人。
: 其他的更不用說，房事等同單身（要她心情好時間對地方對才有），家事
: 用遠不敢少做，基本上廚房都我在顧（更不用說都是我在買菜接小孩），
: 她只顧自己的工作、興趣，跟小孩（小孩我當然也要幫著用她的方法教）
: =====某個結論
: 基本上我老婆是挺會辯的人，自詡理性人士，但十年後才發現
: 包裝在理性與科學下的，是挺玻璃心、不會做人（惹怒所有聘過她的老闆）
: 然後惹怒兩家家人（只剩她父母還對她不錯）
: 最可怕的是，什麼事都要辯到贏的結果，還真的會覺得大家都要讓她、幫她，
: 她說她曾是演譯邏輯社社長，還常勝軍，科學、文學、音樂都有發展到
: 某些專業，所以我覺得也挺目中無人，講科學，她就用她學到的科學電你
: 講文學她也會電你，講音樂也會用她的專業電你，所以最後我很多地方都
: 不得不放棄，最慘的是我的興趣，音樂、運動都被收到我趁小孩上運動課時
: 在旁邊的GYM做一下，音樂無法離開耳機，更不用說動漫偷看，電動只能玩她
: 想玩的。烹飪也只能煮快的，做太多反而被她認為我太閒。
: 教育就不提了，全力顧小孩功課，工作自己想辦法，而且她要睡時要陪她說，洗
: 澡陪她洗，煮飯陪她煮，更不用說如果上她的兼職班我還要負責接送她。又要人
: 陪，又不給一個普通上班族時間工作，我其實是不知道怎麼辦。
: 另外心態只能顧她，最近她在冷戰我，因為我上次跟小孩吵架，我看到她時沒講話

: 個事情講她，就被她說我亂講一通，除了哈哈哈我還能說啥？
