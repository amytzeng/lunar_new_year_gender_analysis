要去揣測對方的心意真的不容易，
可以有很多的角度， 最終也可能困死自己。
這篇用比較黑暗直接的角度，不一定就是真相。
但有時候困在難以溝通的狀況，不如就按對方字面的意思解讀。

: 我懂內人的辛苦，這路真的很辛苦，非常辛苦，不是隨便任意發脾氣的
: 她心理壓力也很大，才會這樣敏感

你真的很替她想，但有沒有想過為什麼你善意地替她解讀卻仍被她嫌棄呢?

: 且她一位也是高齡朋友最近順利生子真的打擊她很大

你要想的是，這對她打擊很大，
她會因為這個打擊而做什麼?

: 因此通電話引爆衝突點，內人先不高興我關心家裡。一番爭執後導向了離婚分開
: 我想她認為我行為讓她覺得太過於顧南部的家人
: 以往也有爭吵過類似問題比如回南部車票錢太貴，後來我認為她的想法也是有道理的
: 我也願意在她認同的情況內回南部(目前協調一年三次)
: 或是說我比較為我家人，我也覺得我應該以妻子的優先較高
: 總之我有調整我的作為，想讓她可以盡量有安全感
: 只是我可能做不到位，我這點我到現在還沒完全懂。

怎麼想都想不通為什麼要因為一通關心家裡的電話而鬧到吵架對吧?
有沒有可能因為她早就想離婚，因此才連這種事都能拿出來炒?
而你還在自責著自己是不是真的太過照顧家人。

不但沒打到重點，還可能因為對方也知站不住腳而淪文無底洞。
你想以改變自己來要求她改變，
但這根本只是一個藉口理由，吵下去你贏了也只會讓她更不高興。

: 我們談到最後，她覺得離婚她自己過，不用煩惱這些事情她會快樂
: 我很難過，但，如果真的她會快樂，我希望她快樂。
: 而且她也不願再談了，走進死胡同，似乎沒路可走了

就當她說的話是事實，對她來說離婚她覺得比較快樂。

: 我想很多，真的覺得表面的理由實在太莫名，太牽強
: 但很多點很難我們倆自己談出來，我想走諮商，她不肯

表面理由當然牽強，你只是不願承認真實的理由。

: 受限於我們各自的生活背景，內人個性又難說服。
: 比如我有勸過也有無子的夫妻生活，我也願意照顧她這輩子
: 但她就會說哪有結婚沒有小孩的，那幹嘛結婚(她身邊的確都有)
: 有些想法她有定見後，我多講最後都是又吵起來

因為她也決定放棄為了拚小孩努力了。
而她認為結婚就是為了要有小孩，
既然沒有要繼續拚小孩了，離婚她覺得比較快樂。
按這個邏輯一切都說得通。

你在那邊想東想西，
想她是不是因為壓力大，或是為什麼結婚一定要有小孩? 是不是你太顧家裏?

倒不如去想對她來說她覺得沒有要拚小孩，
那她離婚過單身生活更可以享受人生。


再看你們爭執的點以及她要求的補充
基本上是錢錢錢。

你買房子沒登記她的名字，也不知道婚姻中你們財物怎麼分配。
她娘家拿出60補貼試管(儘管基本上會做試管很大因素是她的身體問題)
你太顧家，回家很花錢 - 她從你身上得到的不夠。

這些都說明她對於一個已經確定沒有小孩而不願繼續的婚姻
她看中的是自己能得到多少， 而沒有經營這個婚姻該付出什麼。

對她來說離婚過單身生活，
經濟上不用再考慮小孩子，生活上也不用考慮你無憂無慮。
只要自己過得開心就好。
甚至可能可以和另一個更優的對象過無子的下半輩子。

這可能才是她的考量
這個考量有多少是出自於愛你，愛你們這個家?
這考量又如何能跟你坦白，甚至再花時間去走諮商?

接受這個邏輯，你想不通的就能說通了。
當然你不希望真相是這樣，但你需要為了這個可能替自己想一想。
你不能確定她為什麼不願意如你所想的好好溝通及諮商
但就算你因此誤解她也是她不願溝通解釋，且對你說出這些血淋淋的話的結果。
更何況，走到最後(離了婚)這些話也都會變成事實。

我相信為了維繫這段婚姻，
你能做的，你想做的都已經很多了。
但對於一個不願溝通表達立場的人，去看她說過的話然後接受也是一種方法。

你從善意、彼此有感情、想繼續經營感情而努力的角度出發是很好的。
但也該適度的接受對方話中你難以接受的部分而調整自己。

既然你太太也會看這篇
希望她也能坦白，別為了想做好人而讓原PO繼續跌跌撞撞。
希望這篇從負面角度的解讀最終不是事實。
