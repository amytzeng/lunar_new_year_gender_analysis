先說結論，
這可能是太太針對你父母親的社交恐懼症，
聽我講述完，你會明白她的一切行為都合乎邏輯。




※ 引述《gakkism5566 (新鮮事)》之銘言：
: 和太太新婚，最近第一次一起回我家鄉過節（會住幾天），過程中似乎沒什麼問題
: ，但太太事後卻認為已有婆媳問題，不知道是不是我太遲鈍。

: 案例一、
: 這次回家前，我媽就請我問太太，希望見面時能給太太一套結婚金飾，但款式傳統，看太
: 太是想拿金飾，還是拿等值於金飾的現金，太太則明確表示希望拿現金。
: 當天我和太太回到家，我媽就把裝有等價現金的紅包拿給太太，寒暄幾句歡迎並請太太拿
: 去買自己喜歡的東西，就下樓讓我們倆自己休息了。
: 事後太太覺得媽媽很像拿錢砸他，只是顯示他有做完應履行的義務。我整個過程都在旁
: 邊，其實沒感受到不舒服的感覺。


社交恐懼症的人會預期自己在社交時出醜、被評價，
因此互動的當下都會表現很順從，
畢竟動輒得咎，
可是事後會越想越不對勁。

因為太太已自我預言互動是惡劣的，
所以說話的時候心中充滿了焦慮跟恐懼，
她感到不舒服，不是因為你媽拿錢怎樣，
而是來自她自己的焦慮，
但是她錯把這個焦慮歸咎在你媽身上。



: 案例二、
: 我和太太吃飯都是一方先全付（有記帳），一段時間後再結算，可說是事後AA制。
: 太太家吃飯都是他長輩付錢，而我家不一定。由於我長大後離鄉工作，約一、二個月才回
: 家一趟，所以偶爾有帶現金在身上或吃可刷卡的店時我就會付，並不覺得出社會後請父母
: 吃飯有什麼，況且也不是豪華大餐。
: 這趟回家有幾餐外食，太太幫長輩們點完餐後就立馬買單了，第一餐結束後有私下問太太
: ，先記著之後再還錢他可以嗎，得到太太正面的回應，就沒想太多按慣例處理，自己也沒
: 去領現金，仍維持由太太先付帳的方式。
: 但事後太太跟朋友抱怨（這幾天太太跟我說我才知道），認為我爸媽不出錢不ok、我不馬
: 上去領現金換成我來先付也不ok，我表現不及格。我知道她這樣想後，雖然不理解這件事
: 為何值得抱怨，但也就立刻把餐費還給太太。


你們吃飯付錢沒問題，
但是跟你們長輩這些人就會有問題，
焦慮跟恐懼讓她心有餘悸，
當下順從，
事後抱怨。




: 案例三、
: 我和太太（及彼此雙親）有共識不宴客，登記就好。連假回家的第二天，我媽一早
: 拿了他收藏已久的金戒指給我和太太一人一個，說雖不宴客但希望有東西可以給我們，
: 希望我們可以收下祖父母輩傳下來、他收藏的戒指。
: 我和太太都欣然收下，但事後太太表示那沒多少錢，我媽不用講成好像沒宴客多可憐。我
: 覺得雖然戒指真的沒多重，但是感覺媽媽是想要示好，不過不知為何太太並不領情。


跟案例一一樣，
你媽給戒指是善意，說宴客是為這件事找一個理由，
但是充滿焦慮的太太只能解讀出負面訊息，
貶低戒指的價值，好緩解自己的焦慮。



: 案例四、
: 我和太太婚前就說好要自己存錢買房，盡量不造成長輩負擔，畢竟是他們的退休錢。而
: 最近兩人存款加起來正好夠買間中古屋（也買到了，過程略）。
: 有關頭期款，太太的爸爸在買之前聽到女兒想買就有給他一些。而我沒有請長輩幫忙，是
: 簽約後我才告知他們買到了。
: 由於中古屋需要裝修，我爸媽得知後表示願意贊助裝潢費。而我和太太付完頭期僅存的錢
: 裝潢，的確會有點緊。考量我家兩老也不是很有錢，因此我在打聽爸媽可以贊助的區間金
: 額後，就想跟太太討論要跟爸媽拿多少。
: 太太首先表示不解我幹嘛問他，接著說爸媽贊助還講區間，好像在釣胃口，給得不夠爽
: 快，並且這些錢別算在他頭上，他沒拿我家半毛錢，要給就給他兒子（指我），而我自己
: 答應太太要照顧他，所以在生活過的輕鬆的前提下，理應要把錢拿到，並用在跟他一
: 起的房子裝潢上，有剩的錢也要拿來提前償還貸款。
: 雖然有道理，但聽來蠻不是滋味。


雖然沒有直接讓太太跟你爸媽對話，
但是要加上她的意見，這也等同於在社交，
要是最後說的價格太多讓他們生氣怎麼辦？
要是說的太少讓自己父母不開心怎麼辦？
好恐懼阿！
所以這問題丟回去給你，
沒有太太的事情，
就免於恐懼了。




: 案例五、
: 我家是透天，住我家的期間，太太希望我多待在房間陪她，並要我轉達給長輩說，他不會
: 住滿整個連假，他會提前離開，總之盡量少讓他和我爸媽對到話。
: 除表示一直待在房間不下樓不太禮貌之外，其餘的我都盡量居中減少相處的機會。而太太
: 要提前離開的事情，我爸媽並不在意，因為放假有自己的行程安排沒什麼。至於每一餐想
: 要吃什麼，也都是我先問太太他想吃什麼，再由我轉達給父母，最後都有如太太的期望吃
: 到他想吃的。
: 但是總不會全程都沒有對話機會，例如長輩問太太想不想吃A、想不想去B、想不想
: 用C等，這些若已知太太的喜好，我就會先幫他回應，但幾次後，長輩會覺得我幹嘛總幫
: 太太決定，不讓吃A、不讓去B。所以我就請太太直接回應，要不要都沒關係，
: 不會有人強迫你。太太事後對此表示他很不開心，認為我造成了婆媳問題，

太太的恐懼根源就是要與你爸媽社交，
你卻讓他們直接對話，
這不是犯了大忌嗎？
還不切腹謝罪嗎？


簡單來說，
婆媳問題是一隻先射好的箭，
後面每一次的互動都只是將靶畫上去而已。
