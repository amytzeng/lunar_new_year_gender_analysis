聊天內容可以明顯看到先生對太太的不耐煩。
事件三更是表露無遺。

不耐煩的原因是什麼?
如果是先天個性使然，那也沒什麼好問的了。
反倒是這樣還能娶到老婆很神奇。

人家女生說自己發生了一些事心情不好失眠，
直酸回去自己思考股票也失眠， 這根本是沒想聊天吧?

本篇是基於先生對太太不耐煩的角度回看聊天的內容。

: - - - - - - - - - - - - - - - - - - - - - - - - - - - -
: 事件一
: 看電視世界新聞某國經濟狀況不好導致房價大漲
: 租屋價格大漲40%（數字忘了）
: 妻：「好可憐喔連租屋都漲成這樣生活一定很辛苦。」
: 夫：「你不知道他們都住國宅嗎很便宜，而且租遠一點就好了哪會辛苦。」
: 妻：「如果是我們房租漲那麼多倍被迫搬家的確很辛苦啊，通勤很累，他們很可憐耶怎麼
: 沒有同理心。」
: 夫：「那不然你要捐錢給他們嗎？」
: 妻：「我單純看到新聞的有感而發說他們房租暴漲很可憐，為什麼要扯到捐錢？」
: 夫：「你覺得他們可憐就捐款啊我有說錯嗎？」

聊天 和 談心是不同層級的事情。
女方聊天著重於共感、同理的回饋。

事件一中女方覺得好可憐， 但男方根本無感。
耍廢看電視還要陪太太苦他人所苦? 還要假設性思考如果自己被迫搬家的情境。

男方後面其實是在挖苦女方， 夫回話的意思直白就是下面這樣
"假設那麼多， 阿妳是有要捐錢嗎?"

女方也感受到了，所以她不爽 問對方為什麼要扯捐錢。


: 事件二
: 最近颱風的關係風切聲很大
: 妻：「有點擔心颱風已經走了風切聲還那麼大，冬天東北季風感覺也會很大聲。」
: 夫：「不會，颱風的風是四面八方吹過來才那麼吵，東北季風只有一個方向不會那麼大聲
: 。」
: 妻：「東北方正對主臥衛浴的窗戶，我就是擔心衛浴那個方向的聲音啊。」
: 夫：「不會，東北季風是有方向的。」
: 妻：「對啊我指的就是衛浴的方向啊。」
: 夫：「好好好你說得對。」

颱風天宅在家耍廢
女方說 擔心颱風走了還風切聲大

男方根本不care, 搞不好那也不是新居。
心理翻白眼，太太又來了

接下來開始風向的討論， 一入既往的陷入不爽的循環
先生意識到這件事，所以說 "好好好你說得對"
意思是 : 又來了，妳要怎樣就怎樣吧。 就不知還有沒有搭配個白眼。

: 事件三
: 妻：「昨天因為發生了一些事情心情不太好睡不著失眠了。」
: 夫：「我昨天也因為思考股票不知道要買哪支所以也失眠了。
: 妻：「為什麼我跟你說我心情不好，你不是應該要先問怎麼了嗎，而是直接忽略講你自己
: 的事。」
: 夫：「沒有啊因為你跟我說你失眠，我也跟你說我也失眠啊，難道只有你可以講你失眠我
: 不可以講我也失眠嗎？」
: 妻：「你可以主動跟我講啊，但因為是我先講的，先後順序不是應該你先關心我再換我關
: 心你嗎？」

夫妻在一起三年， 一些習慣都是累積已久。

女方說 因為發生一些事情心情不太好睡不著失眠了。
這其實是想討拍， 想要對方問、對方關心。
但用了一種給人感覺毛多，要講不講還要別人回問的方式。

先生直覺翻白眼， 又來了。
"我昨天也因為思考股票而失眠了。"

這諷刺、不耐煩、真的是表露無遺。
女方是真的沒感受到嗎???

是真的沒感受到才會天真探究順序，
還是意會到了但拉不下臉，所以用責問對方做回應?

-----------

男方的問題很明顯， 就是對待親人不夠好。
對老婆很不耐煩。
即使太太很常用事件三的說話方式，也不是每個人都會酸回去。
不耐煩就會失去幽默、聊天喇賽、傾聽關心的能力。

找出不耐煩的原因不是要合理化不耐煩這件事，
而是要去解決問題。

而女方也要了解 自己提出的話題不是閒聊而已。
是否太在意對方的回應有沒有共感到自己的心情?
是否講話太扭捏， 長此累積造成對方不耐煩?


: 事件四
: 夫：「以前這條商店街很多店家現在都倒光了。」
: 妻：「對啊，可能疫情期間成本太高了撐不住了。」
: 夫：「不是，是現在人力資源基本時薪一直調高，但來店客量又不足，店家會精算開店根
: 本賺不到錢，不如關起來不做了。」
: 妻：「我們講的原因不是一樣嗎？」
: 夫：「沒有，不太一樣。」


對比先生所開的話題
就真的是閒到不行的閒聊， 走在路上有眼睛的人都看的到店倒光了的閒聊。
女方若是回 "喔對阿，都倒光了" 大概就句點了的話題。
先生不會說 "很多店倒光了好可憐。"
然後期待聽到同理，共感的回應。

而 疫情間成本高 跟 人力資源時薪上調 確實是再講不一樣的事。
事件四該關注的部分在於
既然雙方聊天會變成對於事情的爭辯
那麼到底是怎麼收尾的，有沒有好好的收尾?

女方認為男方故意唱反調
但反過來說，如果男方說的沒錯，他就會覺得女方笨又不承認。
錯了又不能講，講了肯定會生氣， 超難聊、死鴨子嘴硬。

所以事件四如果就收在男方說 "沒有，不太一樣"
然後彼此心裡忿忿不平不講話。
女方覺得男方唱反調， 男方覺得女方經不起人講。
這收尾就是很差的。
(而當然不可能總是先生對，角色互換一樣是不好的結果)

而死鴨子嘴硬的人，往往只會覺得對方在找碴
但其實自己就承認錯誤或是稱讚對方一下就可以了。
而一旦心裡對對方有了這種"死鴨子嘴硬"，"愛唱反調"的印象
爾後就會先入為主的覺得對方又來了， 使得聊天變得更難聊。


要解決這個問題首先就不能將問題歸因於單一或某個人。
男方有不耐煩的問題， 女方也確實有煩人的地方。
當兩個人的問題程度死亡交叉時就爆炸了。

又如到底是唱反調還是死鴨子嘴硬，
是自己不能接受反對意見，還是對方硬要強調他是對的?
到底是對方講話不好聽，還是自己有時也太敏感像刺蝟?
當兩個人的問題程度死亡交叉時就爆炸了。

回頭看事件一
當男方"挖苦" 不然妳要捐錢嗎的時候。
如果女方視為一個聊天的話題，
樸實的回答: 不要， 就像我覺得乞丐可憐也不等於我想掏錢。
說真的也沒什麼好不開心的， 男方挖苦就變成偽議題。

當女方感受到的是挖苦， 而她需要的又是共感， 偏偏男方又不耐煩。
吵架只是剛好拉。
