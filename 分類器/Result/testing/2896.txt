似乎因為原文前幾樓推文，討論導向「怎麼了嗎？」到底會不會給人不好的語感，不過我
認為重點不是那句，所以發文佔個版面><

※ 引述《bellaj (貝拉J)》之銘言：
: 是否同意記者抄文：/NO
: 昨天我本來晚上要帶小孩去逛夜市，
: 但是昨天的天氣不好，小雨下下停停又變冷，
: 中午我就跟孩子們說晚上的夜市取消了。
: 午餐時間老公過來吃飯，
: 盛好飯走過來餐桌一邊問：「你們晚上有要去夜市嗎？」
: 我說：「下雨我們就不去，有跟孩子說好了。」

看這句覺得原po表達不清，果然老公誤會成「(如果)下雨我們就不去」。

: 妹妹問「什麼時候還會帶我們去？」
: 我說：「等過年吧！」

妹妹用「還會」而不是「才會」，所以老公沒發現誤會了。

: 老公也接著話題跟孩子閒聊幾句（我不記得了）
: 下午四點半老公突然傳賴問我還要不要去夜市？
: （我整個黑人問號？？不是中午講過了嗎？）
: 但還是回他「沒有。怎麼了嗎？」
: 想說他是不是有其他事情才會又問我ㄧ次
: 結果他就爆氣回我一堆話
: 我只覺得莫名其妙，但還是耐著性子解釋。
: 然後他就說晚餐不吃了。他只要生氣就會不吃飯。
: （但是又不會承認自己在生氣喔！）
: 晚上回家的時候都沉著臉不發一語
: 我還主動關心他「你後來有吃飯嗎？」怕他餓肚子
: 他說「我下班在外面吃了。」
: 之後我們就沒再對話。
: https://i.imgur.com/9wMN8Va.jpg

老公以為看下雨與否決定要不要去，然而外面陰雨綿綿——這種天氣對部分人來說並不會
取消逛夜市，雨不大撐傘就好——所以詢問確認。

原po認為中午已經確定不去並告知，所以回答「沒有」後反問「怎麼了嗎」。

這邊開始注意老公的回應時間。

一開始只回「問一下」三句，隔了幾分鐘「要不要去也該跟我說一下」——我認為這句才
是重點。

試想一群人約好下午出遊，但上午天氣不好，討論後決定看下午有沒有好轉，結果時間快
到了對話框依舊靜悄悄......
「所以有要去嗎」
「沒有，怎麼了嗎XD」

——要不要去都不用確認通知的喔QQ
還問我怎麼了，我沒有被通知到啦QQ
就算你笑著說也無法改變我被遺忘的事實QQ

老公很傷心，想了幾分鐘決定不要悶在心裡好好說出來，才有訊息的間隔。他如果因為怎
麼了嗎就生氣，我認為不會隔幾分鐘，所以這邊還沒真正生氣。

: https://i.imgur.com/u5Mamxy.jpg

原po不懂老公幹嘛糾結「怎麼了嗎」，覺得反應大。

老公的視角如前述，所以「我沒有反應很大」是真的，不是喝醉的人說自己沒醉。

接下來原po解釋完，老公就發現是中午有誤會，於是說明「我是聽到若下雨就不去」，並
下結論「沒事了」。

本來話題到這結束就一切完美，沒想到原po來個爆擊

「我是很明確的說下雨不去 並且指著雙寶說已經跟他們說要取消了」

*翻譯：我說得超清楚der，還搭配指著雙寶的動作，怎麼會聽錯！？你耳包？

這句才讓老公氣到不吃飯。

注意對話時間，老公如果早就生氣，沒事了後面應該立刻接不吃晚餐，但他到原po回完過
了5分鐘才說，可能他本來都覺得話題結束放下手機了......

: 是他太難搞還是我哪裡惹到他了呢？（嘆氣

一切開端是原po中午話講不清楚，引爆點則是重複強調自己講很清楚——自以為表達清楚
引發的慘劇QQ

不過跟熟悉的人說話，有時就是會省略，而且自己不會有感覺。除非你們說話很常誤會，
不然不用改變說話方式，只要記得 溝通是雙向，所以有誤會時，兩邊都可能有問題 。對
方表達他的理解時，不要踩得很硬自己一定沒說錯，笑笑感嘆中文博大精深就好，說不定
還可以發個笨板咧XD

至於另一邊，老公因誤會而有情緒，被原po重複強調而惹怒，雖然能理解，但生悶氣是不
好，這是老公要改善的部分。

不過，如果是因為老公不擅吵架、吵不贏原po，而只能用生悶氣的方法表達不滿，那原po
也要思考一起改善，畢竟你也不喜歡他生悶氣呀。
