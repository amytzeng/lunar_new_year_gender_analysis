: 我也是被吵的那一個，但是先生完全沒有感覺
: 想分享一下理解他人痛苦的重要。
: 沒感覺的人只會覺得我很煩 很亂 情緒很差
: 但是我這一方面是覺得真的很吵，聽到那個聲音就會很煩躁 很想砍人
: 尤其是如果心情已經很差，
: 先生還一副無所謂的樣子說，本來就會有這種聲音，叫我自己忍耐
: 這種時候除了原本被噪音吵到快瘋掉以外，
: 還會覺得先生一點都不在乎我的痛苦，心情差上加差，也不會有什麼好臉色
: 我可能也是好一點，我不會製造噪音回去
: 但是我聽到某些頻率的噪音就會一直非常非常焦慮
: 焦慮到我先生覺得我很煩 焦慮個屁
: 綜合以上 噪音造成的焦慮  先生又針對我攻擊
: 當然就會吵架了
: 那該怎麼做?
: 當然第一，是減少噪音來源。
: 第二，就是配偶理解的重要性了，
: 就算不能減少噪音來源，也至少不要說一些
: "為什麼只有你這樣"  "大家都這樣被吵也沒事"  "妳忍一下就好"的風涼話
: 吵架吵幾次之後   我先生已經沒有講過這些話了
: 當然更好的做法是像little0817的先生一樣，
: 除了不講風涼話以外，進一步幫忙把噪音消除，或者研究耳塞
: 一方面減少噪音，一方面也表達支持，一舉兩得。
: 雖然減少噪音那些耳塞通常不太有用啦...但是就算治不了本，
: 至少不會因為先生不支持自己的痛苦，而增加額外的負面情緒。
: 所以才會說，
: 作為一個不敏感的人，原則上去理解聲音敏感人的痛苦會有幫助。
: 舉個小例子，我聽得到人家抖腳的衣服摩擦聲。那是一種非常煩躁的頻率。
: 咖啡廳、辦公室，只要有人抖腳我都聽得到。
: 在家裡，我先生也會抖，一開始跟他說可以不要抖腳嗎？他也覺得我在找碴。
: 說為什麼我沒事要限制他的自由，當然又是吵架........
: 真的溝通很多次之後，他還是不相信我真的聽得到。
: 我花很多時間證明我真的聽得到，很麻煩，但是不證明他不相信。
: 做法就是每次一起出去我都可以跟他說哪邊有人在抖腳，
: 他一抖我也可以立刻告訴他你在抖腳我有聽到。
: 他理解之後，請他不要抖腳，他會默默的不抖。不再跟我吵了
: 現在至少在家不會為了抖腳聲這件事情吵架了。
: 先理解是確實有幫助的。
