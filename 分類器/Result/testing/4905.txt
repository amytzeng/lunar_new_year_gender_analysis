: 以一個結婚八年的男性觀點來看
: 台灣還存在著跟公婆同住的觀念真的是相當落後且不健康的風氣
: 整理一下各種奇奇怪怪的理由:
: 1.為了孝順
: 百善孝為先這是美德沒錯
: 但你把一個生活習慣完全不同的老婆塞進原生家庭內
: 讓老爸老媽整天嫌東嫌西  還要"花時間精力教育" 更不用說遇到不合心意的逆媳
: 氣到高血壓就飽了  是想提早繼承吧?
: 然後住爸媽家裡美其名共享資源
: 實際上就是繼續吸爸媽的血  然後還多帶一個人進來共享資源  這樣叫孝順?
: 更不用說小孩生了之後還要爸媽賣老命幫忙照顧
: 然後當橋樑解決婆媳問題的方法看來看去就是叫某一方吞下去
: 叫媽媽吞下去叫不孝  叫老婆吞下去叫不義  到底要不孝還是不義呢?
: 真的想孝順  住附近常回家隨時有個照應就好
: 2.為了省錢
: 的確台灣高到不行的房價  讓很多普通上班族想買個房都沒辦法
: 但有個選項叫租屋  台灣房租與房價的比例倒是算很低的
: 房子買不起很正常  但還沒聽過租不起的  真心想找絕對找得到可負擔的租屋
: 省那一點點租金  換來一堆痛苦跟麻煩  一點都不值得
: 跟公婆住  你能隨時想跟老婆大戰就大戰嗎?
: 想在客廳/廚房大戰直接放棄  就連在房間都要注意音量
: 夫妻情趣直接降到谷底  更不用說孩子生了公婆一天到晚衝進房間看孫 還玩個屁
: 3.為了照顧小孩
: 這是相當多女性妥協的地方  小孩出生後需要人手多照顧沒錯
: 但為了方便就住一起簡單的說就是利用公婆而已
: 照顧小孩非常花精力  所以如果要讓公婆一起顧小孩
: 最好的方法就是夫妻平常上班時間帶去給公婆  晚上下班接回來自己照顧
: 一方面學著自己承擔這些責任也讓自己跟小孩更親
: 二方面也要讓老人家晚上可以好好睡覺喘口氣
: 一樣不需要住在一起  在附近租/買個房子不是簡單解決嗎?
: 4.想讓公婆認同自己是個好媳婦
: 這種屬性的我就比較沒辦法了...天選之人....
: 其實每次聽到婆媳住一起的問題我就覺得到底是為什麼那麼難搬出去住
: 台灣應該要立法婚後跟原生家庭住的人多課點稅才對
: 都已經2021年了  世界上已開發國家中婚後跟原生家庭住的比例真的很低很低
: 就連日本那種女生婚後不上班的老舊思想都沒在跟公婆同住了
: 為什麼台灣還有這麼落後的思想呢?
: 爸媽那輩就算了  那是上一代的問題  但讓他延續到下一代就是我們的問題了
: 以男性觀點來看  就是不要委屈你老婆  但也不要利用你父母
: 大概就是這樣

這跟台灣的物理性質和傳統文化有關。
台灣地狹人稠，交通方便，開車一天內就可以環島一圈，
這些條件綜合起來，造成的結果就是：
"新家庭和原生家庭的生活圈很容易重疊"。
跟原生家庭同住、或住得很近，都不難達成。

另外台灣盛行集體主義和孝道文化，子女是父母生命的延續，
子女成立了新家庭，在意義上相當於原生家庭的延伸，而非自成一個獨立單位。
既然是原生家庭版圖擴大，住在一起自是順理成章，
除此之外，家中規則的制定以能滿足集體利益為前提，
並不以尊重個人的獨立性與差異為優先，
加上長幼尊卑的儒家觀念，最終就形成今天看到的現象：
先生基於家族利益最大化要求太太加入自己的原生家庭，
並設法"融入"其中(也就是抹消掉自我)。


回到原po所處的困境，看公婆的反應和談話就知道，
他們是那種在省錢的過程中得到快樂的人，
與原po透過花錢換取高品質生活的價值觀就像天平的兩端，
公婆看媳婦花錢如流水很痛苦，
但是媳婦看公婆錙銖必較還管到自己頭上，更是開心不起來，
所以奉勸原po的先生也不用找什麼平衡點了，根本就找不到的。
你硬要把兩種背道而馳的價值觀擺在同一個屋簷下，
只會為自己帶來更多的災難。
