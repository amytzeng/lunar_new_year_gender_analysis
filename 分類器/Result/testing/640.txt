: 就算有新對象，本來也打定主意不再結婚，
: 因為前一段婚姻給我的印象就是：婚姻真的太可怕了。
: 沒想到竟然在前年遇到一位完全顛覆我認知的男人。
: 他讓我相信原來這世上真的有像修杰楷這樣的人，
: 可以完全不在意對方的過去，也可以對我的孩子視如己出般的疼愛。
: 原來這世上真的有像臉書上那位口罩男那樣懂得站在女人角度為女人想的男人，
: 韓劇《愛的迫降》暖男利正赫對女主角的種種溫柔體貼，完全就是他每天對我的日常。
: 他一直非常努力用行動化解我對於感情與婚姻的恐懼。
: 無論我再怎麼憂鬱悲觀負面，他總是堅定地牽著我的手，告訴我他會一直陪我，
: 並且不厭其煩地一直告訴我：我很好，要多想自己好的地方，我們要一起正面。
: 每次下班陪我回到家，一定是他包辦所有的家事，煮飯洗碗洗衣晾衣摺衣掃地修理東西....
: 甚至陪孩子看功課下棋看卡通玩遊戲、教導孩子所有該有的規矩，
: 並且常常告訴我的孩子「媽咪很辛苦，要多體貼媽咪，讓媽咪多休息。」
: 然後把我趕去洗澡、趕進房間睡覺，再幫我關燈蓋被，他才放心回他自己的家。
: 完全就是把我像女兒一樣放在手心疼愛且細心地照顧著。
: 經過一年多的交往（或許觀察還不夠久？）我終於拗不過他的苦苦央求，
: 在去年和他去戶政事務所登記結婚了。（因為是再婚，我不想高調張揚，所以沒宴客）
: 結婚這將近一年以來，雖然他後來工作被無預警地調去大陸，所以變成遠距離，
: 但每當他定期回台時，他總是一如往常一樣細心溫暖地照顧著我和孩子。
: 近半年來，他開始常常提起「想生我們的孩子」的事。
: 事實上，他已經45歲了，我也40歲了，我們其實早就不年輕。
: 在這個年紀想要生小孩，真的很難。
: 每次提起這話題，我總是無盡地沉默，
: 他總是難掩失落地說：「好啦，沒關係，如果老婆不想要，我還是會尊重老婆。」
: 對我來說，我已經有了一個孩子，已經體會過當媽媽是怎麼一回事了。
: 現在孩子已上了小學，好不容易脫離了夜夜不成眠、為了孩子整天焦頭爛額的日子，
: 要我重頭再來一遍，老實說我真的會怕。
: （有小孩和豬隊友的媽媽們應該都懂吧）
: 但對於一直渴望擁有幸福婚姻家庭老婆孩子的他，
: 好不容易等到現在終於娶到他想要的老婆了（他是這麼說啦），
: 我可以理解，他會想要生下自己血緣的孩子也是自然。
: 而且他的父母年紀也很大了（父親75歲，母親69歲），他的哥哥則是不婚不生主義者。
: 雖然他父母會跟我說，沒關係啦，兩人生活得幸福最重要，不用勉強。
: 他們也會跟鄰居們介紹我的孩子說「是他們的孫子」。
: 但是偶爾還是會感受到他們對於那些有孫抱的親友們的欣羨，和不經意透露的遺憾..
: 再加上，我們年紀都這麼大了，之前曾被醫師建議如果要生最好要趕快，
: 他又偏偏在大陸工作，無法常在台灣，這樣的情況下，
: 醫師強烈建議我們，先生做精蟲冷凍，太太則趕快進行試管。

要不要生小孩 是你們夫妻該決定的事情
也不是上來問個網友幾句就可以決定的
現在看來就是他想要 你不想要

其實讓你煩惱的事情都是很實際的
你再生就是高齡產婦 也不再有體力顧小孩
小孩子應該是在雙方都想要的情況下出生
也不該覺得"虧欠"丈夫就硬著頭皮生

既然你考慮了這麼多事情
為什麼每次對丈夫的請求都是"沉默以對"呢?
為什麼不和丈夫坐下來好好溝通協調?
還是你覺得丈夫知道你不想要小孩 就會離你而去?

如果真的是這樣 建議你們還是早點分開比較好
雙方要的東西不一樣的時候
趁早放手也是讓對方解放
不然你一輩子都覺得虧欠他 他也一輩子遺憾
這樣怎麼可能會有幸福的婚姻?

說難聽點 快不能當媽媽的是你
人家還是可以當爸爸的
真的無法就提早放手吧
雙方都沒錯 只是想要的不一樣
