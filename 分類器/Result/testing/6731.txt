: 感謝大家的熱烈回應，不管是怎樣的意見，我都看到了，也想了很多。
: 以下想把一些事情講清楚，也分享一些我在國外觀察到的現象。
: 首先，我太太、她的姊妹和我岳母已經苦勸岳父施打疫苗快一年了。
: 因為岳母本身是醫療人員，每天在醫院看診，其實帶著病毒回家的機率很高。
: 同時，我岳父的家醫科醫師也建議他接種，但無法保證絕對沒有後遺症。
: 可是，我岳父不但不施打，還常以「不要帶病毒回家」為由，試圖限縮岳母的行動。
: 岳母身體健康，交遊廣闊，和親友、同事甚至病患的關係很好，會結伴出遊。
: 岳父自己不施打疫苗，卻以此為由限制岳母的行動，甚至干涉她外食、交友，
: 我們既擔心岳父的健康，也對他不理性干涉配偶的行動感到不解。
: 再者，我太太雖然隨我移民海外，還是很關心原生家庭，我們兩邊也會給孝親費。
: 逢年過節的禮物、禮金都不會少，並沒有隨疫情擴散後停止。
: 因此有些版友懷疑我們的孝心，duck不必。甚至我岳父失業多年，我們還有幫他繳房貸
。
: 當然，這不構成我太太一定可以住家裡的理由，但是台灣也是夫妻財產共有制，
: 我岳母是非常歡迎女兒和外孫回家，但是我岳父不准許，還對她大小聲。
: 他除了隔海傳訊，還在上班時間頻頻傳訊、打電話給我岳母、我太太的姊妹、堂姊妹，
: 要他們遊說我太太不得回國，說「病毒都是海歸帶回來的」、「疫苗不安全」。
: 另外，疫情爆發這兩年，岳父的偏執與恐慌更加高漲，除了自己幾乎足不出戶，
: 連家裡的電器壞掉、牆壁需要裝修，都不准岳母報修，怕外人帶病毒來，
: 每次和岳母不愉快，兩人就隔海打電話給我太太相互埋怨，我太太聽多了當然會煩，
: 可是她很孝順，總是試圖排解，但之後的負面情緒難免會朝我發洩。
: 而且，我在原文寫得很清楚，不是為了住岳父家才要他去打疫苗，現在也已訂好房。
: 我岳父理應尊重我們的決定才是，可是他施壓親友，想藉此阻止我太太回台，
: 其中的理由包括「岳母等人接觸我太太和小孩，會把病毒帶回家」，我們怎麼忍耐？
: 以下，我想花時間談點防疫的事情。
: 這次事件的癥結之一，在於「到底是不是海歸造成台灣疫情爆發」？
: 當然，海歸一定會帶來一些確診案例，但是台灣長期不比照國際實施普篩，
: 到底有多少黑數？沒人知道。我瞭解有人會談偽陽性的問題，
: 可是多數發展程度與台灣接近的國家都實施普篩或至少廣篩，人家也有他們的理由。
: 把台灣的確診數全數歸咎到海歸，我認為是不公道的。
: 其次，海歸和台灣的旅宿業者已經為了替台灣爭取提高防疫能量的時間，做出很多犧牲
。
: 我先講旅宿業者的損失。根據觀光局的統計，這波疫情造成的「國際觀光」產值損失，
: 每年可能在新台幣2-4000億元(這點的估計會隨年份有所不同)：
: https://reurl.cc/e61eym
: https://reurl.cc/NpVklq
: 台灣政府能夠拿各位的納稅錢貼補這些業者到什麼時候？還能夠撐多久？
: 所以觀光業已經在上個月呼籲政府開始解封開放：
: https://ec.ltn.com.tw/article/breakingnews/3848507
: 在海歸的部分，多數海歸都還保有我國國籍，政府透過實施長期隔離，達到實質阻止，
: 是侵害了憲法保障的公民居住遷徙自由。當然，憲法有避免緊急危難的例外條款，
: 但是疫情已經進入第三年，疫苗、治療藥物早已問世，還可不可以算是緊急危難？
: 有人會說，「海歸自己可以請假」，但是動輒十天以上的隔離期，海歸也是要上班的。
: 換個角度來說，如果台灣有充足的疫苗與藥物，疫苗施打率很高，老人和小孩都可打，
: 跟隨國際趨勢走向共存本來就是應該的。可是事實如何呢？
: 以下是台灣75歲以上長者施打率停滯的新聞：
: https://reurl.cc/ak2oMl
: https://reurl.cc/5GY6bM
: 至於國外5-11歲小孩早就可以施打的BNT/ Pfizer，在台灣又是如何呢？
: https://udn.com/news/story/122190/6189945
: 現在美國50歲以上已經開放施打第四劑了：
: https://reurl.cc/pWOzXb
: 至於賴以救命的藥物，在美國已經是走到藥房就可以拿到的方便程度：
: https://aspr.hhs.gov/TestToTreat/Pages/default.aspx
: 只要在藥房檢測為陽性，條件適合，立刻就可以拿到Lagevrio和Paxlovid等藥物。
: 那麼台灣的情形呢？
: 每千人只有一份藥物：https://www.worldjournal.com/wj/story/121222/6221299
: 快篩劑超級貴又不好拿：https://www.storm.mg/lifestyle/4277252
: 台灣的藥物每千人只有一份，而且是分三季分批到貨，可是疫情已經開始上升了。
: 國際的存貨數據是總人口的2%-7%的數據，台灣是0.1%。
: 我寫那麼多是為什麼？是因為台灣不做普篩或廣篩，以防疫為名把旅客擋在國境外，
: 可是自身疫苗施打率在長者的部分一直拉不起來，又不開放學齡兒童施打。
: 在藥物的準備上又是遠遠落後國際標準，就會造成不敢共存和開放邊境的惡性循環。
: 這種作法不但侵害海外國人的公民權益，又對台灣的經濟造成重大傷害。
: 台灣是先進國家中少數不開放海外不在籍投票的國家，我們海歸想表達心聲也沒辦法。
: 台灣內部又充斥著一切都歸罪於海歸的聲音，不斷施壓政府嚴管邊境，
: 卻沒有用同樣的力度去監督政府趕快進口疫苗、藥物，提供便宜可靠充足的篩劑。
: 只要檢測陽性，不論有無症狀和輕重，全部送去負壓隔離病房，醫療體系當然受不了。
: 其實，只要完整施打三劑疫苗或有陰性檢測證明，已經有多國開放免隔離入境。
: 縱使確診，只要無症狀或輕症，就是給予治療藥物，要病人在家休息。
: 另外室內空間多數實行「疫苗護照」查驗制度，只有完整接種者可以入內。
: 美國現在多州對此鬆綁，不是說這個制度錯誤，而是疫苗覆蓋率和痊癒者夠多了，
: 有了群體免疫，自然可以減少商家和客人的不便，朝向正常生活邁進。
: 可是當陳時中部長日前說考慮實施疫苗護照制度，要施打三劑才可以去某些場合，
: 很多網民反應很激烈，說都是縱放海歸造成的，憑什麼限制他們的自由。
: 我看了留言真的一堆黑人問號。
: 做個結論，海歸和各位在台灣的國人一樣都是公民，法定權利都是一樣的。
: 如果要說避免緊急危難，海歸也為故鄉忍耐快三年了。
: 不能因為部分在台國人的恐懼，或者政府若干防疫措施還要再加強，
: 就轉化成「都是海歸的錯」，繼續嚴管邊境，這對海歸和對台灣自己都不好。
