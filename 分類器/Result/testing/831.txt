: 近一年成功轉領域且進了業界不錯的公司(就職尚未滿半年)
: 我目前的困擾是有非常多人會來找老公諮詢，
: 包含轉領域前的準備、上過的課、寫履歷、面試技巧、工作內容等等
: 他每天會花非常多時間在講這些電話
: (他會晚上下班後一邊陪孩子一邊帶藍芽耳機講電話、或是明明說工作做不完的半夜在講)
: 這些人其實也都不是非常熟，可能學弟的女朋友的妹妹、前同事的學長之類的
: 他給我的理由是:以後都在同領域，說不定之後反而是需要他們幫助
: 但我感覺他在"被諮詢"的過程中得到了成就感?
: 做公司分析、幫別人模擬面試甚至改履歷 都讓他獲得滿滿成就感
: 所以有時候講到一半別人要忙先掛電話
: 他還會跟別人說隨時可以找他討論...
: 我覺得他陪孩子的品質變得很差
: 每天也越睡越少
: 可能他對被諮詢的熱情，越來越多人"介紹"來問他各種問題
: 我其實也滿怕一些其實他根本沒遇過的問題 用想像給出了建議
: 如果最後結果不好 對那個人也不好吧
: 不知道大家有沒有什麼建議或該怎麼勸說他比較好呢?
: 感謝!

看了很有感
因為我以前也是好為人師的人
唉。。。

說真的，建立人脈是一回事情
但常常都是，連謝謝都不一定會收到
免費幫別人，別人當然開心啊，
但不是所有人都知道感恩的，偶爾還會內傷甚至被誤解
妳看版主做得再好，總是偶爾會被嚴厲批評。。。

另外，真正的人脈與口碑，絕對不是靠大量24小時免費服務去換來的
成功的業務人才，也要by case收費，一樣結交客戶建立人脈。
更別說專業領域的諮詢顧問。。。而且輔導的是同領域的對象?
感覺起來好怪。。。
很想知道是甚麼領域是這麼大公無私，不怕栽培出潛在對手

而且看起來他是不分親疏全部努力服務
就算是打口碑，他也該考慮一下成本
特別是陪同家人的時間。。。

只能跟妳分享，我老婆常常提醒我
家人比外人重要多了
請他自己思考，陪老婆小孩，可否更有品質
或是設定自己擔任免費顧問的工時
要弄清楚下班時間，加人才是最需要他花時間陪伴的。。。

否則人脈還沒建穩固
身體就承受不了，免疫力降低，那很危險啊。。。

然後，小孩就是妳不推他，他也很快就不需要妳的一種生物
萬一將來他外面朋友滿滿，小孩卻都不跟他講心裡的話
會很心酸的，更要花數倍的心力，然後不一定挽回
也許妳可以這樣勸他。。。

祝福妳們!
