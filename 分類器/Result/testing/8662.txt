看完第一段，我想你跟我媽是同一類型的人

「覺得」他人無法達到一定的標準，所以選擇不要求，凡事都是自己來

也是因為看我媽用這樣的態度導致一路走來都很累

我後來有點半強迫自己心態和作法上都要做調整，就用你提出的事件當範例

: 事件一
: 因為二寶臨時早產兩週，原本打算要把舊的奶瓶、新的未拆封奶嘴頭在週末一次洗好，結
: 果提早生，只好在月子中心遙控交代老公處理
: 回家前也再次確認有完成，出月中回家看到新的奶嘴頭依然未拆封，因為他以為的奶嘴頭
: 不是我以為的奶嘴頭
: 月中回家再一小時馬上要喝奶，只好趕快拆封清洗用熱水燙過消毒
: 請他抱著寶寶，我處理完成後順利喝到奶，沒有跟他抱怨（處理當下可能有臭臉），事件
: 結束

看起來先生應該是一直陪在你旁邊，等你處理

那我的作法會是：自己抱著寶寶， 請先生用「熱水消毒」的方法儘快處理

分析如下：
1. 他不知道哪些東西是你說要消毒的奶嘴頭，這樣可以讓他知道下次你說到

   「奶嘴頭」的時候，他知道你在說什麼。

    題外話，我目前懷孕5個月（第一胎）；說到奶嘴頭我也是會想一下那是什麼，

    有時候真的會和奶嘴搞混...

2. 我相信一般來說的清洗/消毒流程不是這樣，應該是你當下靈機一動的作法；

   讓他操作過一次，之後真的遇到像這樣很緊急的狀況時，他會知道該怎麼處理

   你之後只要跟他說「趕快幫我先用熱水消毒一下xxx」他就會知道有哪些該注意的地方

: 事件二
: 昨晚老公半夜12點下班，我在11點問老公有沒有打算跟同事吃宵夜，老公說沒有，打算在
: 家吃
: 我跟老公說請他回家先回房間顧一下二寶我要洗奶瓶，因為我11點時要洗奶瓶二寶一直哭
: 要人抱無法放下，跟公婆住小孩一直哭會有無形的壓力
: 老公回家12:30我跟二寶已昏睡，老公叫我我有回應但沒起床，他就自己把奶瓶洗一洗放
: 消毒鍋了
: 我早上起床發現有洗是有洗，但有些擠奶器小配件漏掉沒放進消毒鍋，所以漏掉的只能再
: 次重新消毒
: 另外有些應該倒放的東西沒有倒放，裡面還有積水，只能重新烘乾
: 老公還沒起床，沒有吵架，沒有唸他，只有發現的當下自己碎碎唸給自己聽，目前也已經
: 消化完情緒了，事件結束

一樣，我不清楚這些配件和奶瓶清洗/消毒的SOP

所以我分成兩個部分

急著使用：像你一樣先處理，情緒消化完，他也清醒的時候跟他說

          1. 感謝他昨天在你昏睡的時候還「記得要洗奶瓶」這件事

          2. 不過你前一晚可能太昏頭，沒說到其他配件也有使用，要一併清洗；

             還有奶瓶正放比較容易積水，可以的話，下次洗完用倒放的，也不會

             有落灰塵進去。

          3. 下次再請他洗一次，你要在旁邊（操作如下）

不急著用：情緒一樣先消化完，等下一次要洗這些東西的時候，請他一併再洗一次

          不過要站在他旁邊陪他聊天，有意無意的提醒他哪些是該注意的

          （配件要洗，奶瓶要倒置）

分析：
    他其實不知道哪些東西是有用過的，如果你只是說奶瓶要洗，除非東西都是一起

    放在水槽，不然回家晚了，任誰都很難有精神再去檢查有哪些也是要一起洗的。

    主要是你要讓他知道哪些東西會是「同捆包」的內含物，還有這些東西的處理方式

: 我跟老公有談過，嫩嬰時期比較累容易有情緒，比較容易吵架，大家互相體諒一下撐過去
: 就好
: 有時我發脾氣也能感受到老公在忍耐，事後也會講開感謝他
: 想請問大家，遇到這種好心幫忙分攤，明明再細心一點就能做到80分100分了，卻只做到6
: 0分，以至於我還得再處理一次的狀況，我要怎麼調適心情呢？
: 我知道要感謝他幫忙，不然奶瓶其實他也可以放著，或是認真叫我起床洗
: 遇到今天這種狀況，我該跟他說下次哪裡還可以再更好一點（但就會像在抱怨），還是就
: 不要說呢？
: 請各位前輩指點，謝謝

如同有些版友提到的，孩子也是他的；真的不要用「他人很好還幫忙」的角度思考

我之前要跟我妹說過年時要幫忙媽媽，前幾次也是好好跟他說道理（家是大家的之類的）

然後他不知道該怎麼做，我就示範+看一小段時間

如果可以的話，我會找其他可以陪在他旁邊的工作也一起做，邊聊天邊完成。

然後有機會就在大家面前稱讚他，有出錯也是用開玩笑的方式帶過


我自己的心得（雖然才3x年）

無心去做的人是少數，真的該生氣也不用客氣（看起來你先生不是這種人）

但是大部分的人都是不知道怎麼做，很多時候覺得自己達不到標準就放棄努力

是很可惜的。

我自己也是慢慢調整，或許一開始會花比較多力氣，也會覺得麻煩

不過就當作偶而換條路走，會看到不同風景
